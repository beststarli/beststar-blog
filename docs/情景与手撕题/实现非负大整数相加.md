JavaScript对数值有范围限制：
```js
Number.MAX_VALUE // 1.7976931348623157e+308
Number.MIN_VALUE // 5e-324
Number.MAX_SAFE_INTEGER // 9007199254740991
Number.MIN_SAFE_INTEGER // -9007199254740991
```
如果想要对一个超大的整数（大于9007199254740991）进行加法运算，但又想输出一般形式，那么运算符是无法达到的，一旦数字超过MAX_SAFE_INTEGER，JavaScript就会自动将其转换为科学计数法表示，导致精度丢失。

## 代码实现
```js
function sumBigNumber(a, b) {
  let res = ''
  let temp = 0
  a = a.split('')
  b = b.split('')
  while (a.length || b.length || temp) {
    temp += ~~a.pop() + ~~b.pop()
    res = (temp % 10) + res
    temp = temp > 9
  }
  return res.replace(/^0+/, '')
}
```

主要思路是：
- 首先用字符串的方法来保存大数
- 初始化res，temp来保存中间的计算结果，并将两个字符串转为数组，以便进行每一位的加法运算
- 将两个数组对应位相加，对10取余，得到当前位的结果
- 判断当前位是否大于9，如果大于9，则temp为1，表示下一位需要进位
- 重复上述步骤直到计算结束

其中：`~~`是取整的意思，`~~a.pop()`表示取出数组a的最后一位，并将其转为数字类型，如果a为空，则返回0。正则表达式`/^0+/`表示匹配字符串开头的所有0，`replace(/^0+/, '')`表示将开头的所有0替换为空字符串，从而去掉前导零。

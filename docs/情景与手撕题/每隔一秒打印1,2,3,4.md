## 代码实现
### 闭包
```js
for (let i = 1; i < 5; i++) {
  (function(i) {
    setTimeout(function() {
      console.log(i)
    }, i * 1000)
  })(i)
}
```

### let块级作用域
```js
for (let i = 0; i < 5; i++) {
  setTimeout(function() {
    console.log(i)
  }, i * 1000)
}
```

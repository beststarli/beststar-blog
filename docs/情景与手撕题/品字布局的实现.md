品字布局是前端开发中常见的布局需求。

## 浮动实现
该方法是在三个盒子的宽高都知道的情况下才能实现，对于上面的1使用margin属性让他水平居中，下面的2和3使用浮动实现：
```html
<div class="div1">1</div>
<div class="div2">2</div>
<div class="div3">3</div>
```
```css
.div {
  width: 100px;
  height: 100px;
  font-size: 40px;
  line-height: 100px;
  text-align: center;
  background-color: #f0f0f0;
}

.div1 {
  background-color: #ff9999;
  margin: 0 auto; /* 水平居中 */
}

.div2 {
  background-color: #99ff99;
  float: left; /* 左浮动 */
  margin-left: 50%; /* 向右移动50% */
}

.div3 {
  background-color: #9999ff;
  float: left; /* 右浮动 */
  margin-left: -200px; /* 向左移动50% */
}
```

## inline-block实现
将div设置为inline-block实际上和上面的float作用一样，就是让下面的两个块不换行。
```html
<div class="div1">1</div>
<div class="div2">2</div>
<div class="div3">3</div>
```
```css
.div {
  width: 100px;
  height: 100px;
  font-size: 40px;
  line-height: 100px;
  text-align: center;
  color: #f0f0f0;
}

.div1 {
  background-color: #ff9999;
  margin: 0 auto; /* 水平居中 */
}

.div2 {
  background-color: #99ff99;
  display: inline-block; /* 设置为inline-block */
  margin-left: 50%; /* 向右移动50% */
}

.div3 {
  background-color: #9999ff;
  display: inline-block; /* 设置为inline-block */
  margin-right: -200px; /* 向左移动50% */
}
```

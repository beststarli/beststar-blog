绘制梯形主要用到border属性。
```html
<div class="trapezoid"></div>
```

## 直角梯形
```css
.trapezoid {
  height: 0;
  width: 100px;
  border-bottom: 100px solid red; /* 设置底部边框的颜色和宽度 */
  border-right: 40px solid transparent; /* 设置右边框为透明，形成梯形的斜边 */
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{height:0, width:'100px', borderBottom:'100px solid orange', borderRight:'40px solid transparent'}}></div>
</div>

## 等腰梯形
```css
.trapezoid {
  height: 0;
  width: 100px;
  border-width: 0 40px 100px 40px; /* 设置边框宽度，形成等腰梯形 */
  border-style: none solid solid; /* 设置边框样式为实线 */
  border-color: transparent transparent red; /* 设置边框颜色，形成梯形的底部颜色 */
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{height:0, width:'100px', borderWidth:'0 40px 100px 40px', borderStyle:'none solid solid', borderColor:'transparent transparent orange'}}></div>
</div>

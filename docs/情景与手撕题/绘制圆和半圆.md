CSS绘制圆和半圆主要用到border-radius属性，通过设置不同的值可以实现不同的效果。

## 绘制圆
```css
div {
  background-color: red; /* 设置背景颜色 */
  height: 100px; /* 设置高度 */
  width: 100px; /* 设置宽度 */
  border-radius: 50%; /* 设置圆角半径为50% */
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{backgroundColor:'orange', height:'100px', width:'100px', borderRadius:'50%'}}></div>
</div>

要注意的是，在使用border-radius时，使用50%和100%都可以得到一个圆。border-radius的值是百分比的话就相当于盒子的宽度和高度的百分比，如50px 150px的矩形，如果border-radius设置为100%，则等价于：
```css
border-radius: 50px 150px; 1
```
而border-radius又是由border-top-radius、border-right-radius、border-bottom-radius和border-left-radius四个属性组成的，上面的border-radius: 100%等价于：
```css
border-top-left-radius: 100%;
border-top-right-radius: 100%;
border-bottom-right-radius: 100%;
border-bottom-left-radius: 100%;
```
或
```css
border-top-left-radius: 50px 150px;
border-top-right-radius: 50px 150px;
border-bottom-right-radius: 50px 150px;
border-bottom-left-radius: 50px 150px;
```
在W3C标准中对重合曲线做了规定：如果两个相邻的半径和超过了对应的盒子的边的长度，那么浏览区要重新计算以保证他们不重合。也就是说如果相邻圆角的半径都设置成大于50%，那么浏览器会根据圆形的实际情况做计算。所以为了避免不必要的计算，建议在绘制圆时直接设置border-radius为50%。

## 绘制半圆
```css
div {
  background-color: red; /* 设置背景颜色 */
  height: 50px; /* 设置高度 */
  width: 100px; /* 设置宽度 */
  border-radius: 0px 0px 100px 100px; /* 设置圆角半径，形成半圆效果 */
}
```

  <div style={{backgroundColor:'orange', height:'50px', width:'100px', borderRadius:'0 0 100px 100px'}}></div>
</div>

一般两栏布局指的是左栏宽度固定，右栏宽度自适应的布局方式。

## 利用浮动
### 写法一
利用浮动，将左边元素宽度设置为200px，并且设置左边浮动。将右边元素的margin-left设置为200px，宽度设置为auto（默认auto，撑满整个父元素）
```css
.outer {
  height: 100px
}

.left {
  float: left;
  width: 200px;
  background-color: red;
}

.right {
  margin-left: 200px;
  width: auto;
  background-color: blue;
}
```

### 写法二
利用浮动，左侧元素设置固定大小，并左浮动，右侧元素设置overflow:hidden，这样右边就出发了BFC，BFC的区域不会与浮动元素发生重叠，所以两侧就不会发生重叠。
```css
.left {
  width: 100px;
  height: 200px;
  background-color: red;
  float: left;
}

.right {
  height: 300px;
  background-color: blue;
  overflow: hidden;
}
```

## 利用flex布局
利用flex布局，将左边元素设置为固定宽度200px，将右边的元素设置为flex:1。
```css
.outer {
  display: flex;
  height: 100px;
}

.left {
  width: 200px;
  background-color: red;
}

.right {
  flex: 1;
  background-color: blue;
}
```

## 利用绝对定位
### 写法一
利用绝对定位，将父级元素设置为相对定位。左边元素设置为absolute定位，并且宽度设置为200px，将右边元素的margin-left的值设置为200px。
```css
.outer {
  position: relative;
  height: 100px;
}

.left {
  position: absolute;
  height: 100%;
  width: 200px;
  background-color: red;
}

.right {
  margin-left: 200px;
  height: 100%;
  background-color: blue;
}
```

### 写法二
利用绝对定位，将父级元素设置为相对定位。左边元素宽度设置为200px，右边元素设置为绝对定位，左边定位为200px，其余方向定位为0。
```css
.outer {
  position: relative;
  height: 100px;
}

.left {
  width: 200px;
  height: 100%;
  background-color: red;
}

.right {
  position: absolute;
  top: 0;
  left: 200px;
  right: 0;
  bottom: 0;
  height: 100%;
  background-color: blue;
}
```

## 示例一：一维数组
`arr = [1, 2, 3, 4, 5]`，求数组元素的和。
```js
let arr = [1, 2, 3, 4, 5]
let sum = arr.reduce((total, i) => total += i, 0)
console.log(sum) // 15
```

## 示例二：多维数组
`arr=[1,2,3,[[4,5],6],7,8,9]`，求数组元素的和。
### 转为字符串处理
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
const ans = arr.toString().split(',').reduce((total, i) => total += Number(i), 0)
console.log(ans) // 45
```

### 递归处理
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function add(arr) {
  if (arr.length === 1) {
    return arr[0]
  }
  return arr[0] + add(arr.slice(1))
}
console.log(add(arr)) // 45
```

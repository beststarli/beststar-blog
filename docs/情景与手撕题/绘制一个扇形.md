CSS绘制扇形的思路与三角形基本一致，只是多了圆角的样式。我们可以通过设置border-radius属性来实现圆角效果。
```css
div {
  border: 100px solid transparent; /* 设置边框为透明 */
  width: 0;
  height: 0;
  border-radius: 100px; /* 设置圆角半径 */
  border-top-color: red; /* 设置扇形的颜色 */
}
```

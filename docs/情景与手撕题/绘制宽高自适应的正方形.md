可以利用vw和vh单位以及margin和padding来实现宽高自适应的正方形

## 利用vw来实习
```css
.square {
  width: 10%;
  height: 10vw;
  background-color: red;
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{width:'20%', border:'1px dashed #999', padding:'8px'}}>
    <div style={{width:'100%', height:'10vw', backgroundColor:'orange'}}></div>
  </div>
</div>

## 利用margin和padding
```css
.square {
  width: 20%;
  height: 0;
  padding-top: 20%;
  background-color: red;
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{width:'50%', border:'1px dashed #999', padding:'8px'}}>
    <div style={{width:'100%', height:0, paddingTop:'100%', backgroundColor:'orange'}}></div>
  </div>
</div>

## 利用margin-top
```css
.square {
  width: 30%;
  overflow: hidden;
  background-color: red;
}

.square::after {
  content: '';
  display: block;
  margin-top: 100%;
}
```

<div style={{display:'flex', justifyContent:'center', padding:'24px 0', marginBottom:'24px', background:'#f5f5f5', borderRadius:'8px'}}>
  <div style={{width:'40%', border:'1px dashed #999', padding:'8px'}}>
    <div style={{width:'100%', overflow:'hidden', backgroundColor:'orange'}}>
      <div style={{content:"''", display:'block', marginTop:'100%'}}></div>
    </div>
  </div>
</div>

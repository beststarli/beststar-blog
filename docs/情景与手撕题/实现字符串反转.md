在字符串的原型链上添加一个方法，实现字符串翻转：
## 代码实现
```js
String.prototype._reverse = function(str) {
  return str.split('').reverse().join('')
}

const str = 'abc'
console.log(str._reverse(str)) // cba
```
需要注意的是，必须通过实例化对象之后再调用定义的方法，否则找不到该方法。

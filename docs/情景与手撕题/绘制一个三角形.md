CSS绘制三角形主要用到的是border属性，平时在给盒子设置边框时，往往都设置的很窄，误以为边框是由矩形组成的，事实上border属性是由四个三角形组成的，我们可以通过设置border的宽度和颜色来绘制出一个三角形。
```css
div {
  width: 0;
  height: 0;
  border: 100px;
  border-color: orange blue red green;
}
```
![三角形](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/job/三角形.png)

## 上三角
```css
div {
  width: 0;
  height: 0;
  border-top: 100px solid red; /* 上边框颜色 */
  border-left: 100px solid transparent; /* 左边框透明 */
  border-right: 100px solid transparent; /* 右边框透明 */
}
```

类数组转换为数组的方法有这样几种：
## call调用数组slice方法
```js
Array.prototype.slice.call(arguments)
```

## call调用数组splice方法
```js
Array.prototype.splice.call(arguments)
```

## apply调用数组concat方法
```js
Array.prototype.concat.apply([], arguments)
```
## Array.from方法
```js
Array.from(arguments)
```

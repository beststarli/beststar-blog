## 递归实现
普通的递归思路很好理解，通过循环递归的方式一项一项地遍历，如果每一项还是一个数组，那么就继续往下遍历，利用递归程序的方法实现数组的每一项的连接：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  let result = []
  for (let i = 0; i < arr.length; i++) {
    if (Array.isArray(arr[i])) {
      result = result.concat(flatten(arr[i]))
    } else {
      result.push(arr[i])
    }
  }
  return result
}

flatten(arr)  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

## reduce方法实现
从递归可以看出就是对数组的每一项进行处理，那么其实可以用到reduce来实现数组的拼接，从而简化递归方法的代码：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  return arr.reduce((prev, next) => {
    return prev.concat(Array.isArray(next) ? flatten(next) : next)
  }, [])
}
console.log(flatten(arr))  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```


## 扩展运算符实现
这个方法采用了扩展运算符和some方法，两者共同使用，达到数组扁平化的目的：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  while (arr.some(item => Array.isArray(item))) {
    arr = [].concat(...arr)
  }
  return arr
}
console.log(flatten(arr))  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

## 字符串处理
可以通过split和toString方法来共同实现数组扁平化，由于数组会默认带一个toString方法，所以可以直接把数组转换成逗号分隔的字符串，然后再用split方法把字符串重新转换为数组：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  return arr.toString().split(',')
}
console.log(flatten(arr))  // ['1', '2', '3', '4', '5', '6', '7', '8', '9']
```

## ES6 flat方法实现
可以直接调用ES6的flat方法来实现数组扁平化，flat方法的语法是：
```js
arr.flat([depth])
```
其中depth是flat的参数，depth是可以传递数组的展开深度（默认不填写，数值为1），即展开一层数组，如果层数不确定，参数可以传入Infinity，代表无论多少层都展开：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  return arr.flat(Infinity)
}
console.log(flatten(arr))  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

## 正则表达式和JSON方法
先采用JSON.stringify将数组转换为字符串，然后通过正则表达式过滤掉字符串中的数组括号，再利用JSON.parse将字符串转换为数组：
```js
const arr = [1, 2, 3, [[4, 5], 6], 7, 8, 9]
function flatten(arr) {
  let str = JSON.stringify(arr)
  str = str.replace(/\[|\]/g, '')
  str = '[' + str + ']'
  return JSON.parse(str)
}
console.log(flatten(arr))  // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

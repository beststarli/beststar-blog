通过两次循环相邻元素两两交换，完成冒泡。
## 代码实现
```js
function bubbleSort(nums) {
  const a = nums.slice();
  for (let i = 0; i < a.length - 1; i++) {
    for (let j = 0; j < a.length - 1 - i; j++) {
      if (a[j] > a[j + 1]) [a[j], a[j + 1]] = [a[j + 1], a[j]];
    }
  }
  return a;
}
```

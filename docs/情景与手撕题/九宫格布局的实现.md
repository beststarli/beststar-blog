九宫格布局是前端开发中常见的布局需求。

通用的HTML结构：
```html
<div class="box">
  <ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
    <li>6</li>
    <li>7</li>
    <li>8</li>
    <li>9</li>
  </ul>
</div>
```
公共样式：
```css
ul {
  padding: 0;
}

li {
  list-style: none;
  text-align: center;
  border-radius: 5px;
  background-color: skyblue;
}
```

## flex实现
对于九宫格布局，flex实现很简单，需要设置一个flex-wrap: wrap属性让子元素换行。

由于我们给每个元素设置了下边距和右边距，所以最后同一列的右边距和最后一行的下边距撑大了ul，这里使用类型选择器来消除他们的影响。
```css
ul {
  display: flex;
  flex-wrap: wrap; /* 允许换行 */
  width: 100%; /* 设置ul的宽度 */
  height: 100%; /* 设置ul的高度 */
}

li {
  width: 30%; /* 每个li占30%的宽度 */
  height: 30%; /* 每个li占30%的高度 */
  margin-right: 5px; /* 设置右边距 */
  margin-bottom: 5px; /* 设置下边距 */
}

li:nth-of-type(3n) {
  margin-right: 0; /* 每行最后一个元素没有右边距 */
}

li:nth-last-child(n + 7) {
  margin-bottom: 0; /* 最后一行的元素没有下边距 */
}
```

## grid实现
grid布局相对flex布局就更容易了，只需要设置grid-template-columns和grid-template-rows属性就可以了。
```css
ul {
  width: 100%; /* 设置ul的宽度 */
  height: 100%; /* 设置ul的高度 */
  display: grid; /* 设置为grid布局 */
  grid-template-columns: 30% 30% 30%; /* 设置3列，每列占30% */
  grid-template-rows: 30% 30% 30%; /* 设置3行，每行占30% */
  grid-gap: 5px; /* 设置网格间距 */
}
```
其中grid-template-columns和grid-template-rows属性分别定义了网格的列和行，grid-gap属性定义了网格之间的间距。

## float实现
首先需要给父元素的div设置一个宽度，宽度值为子元素宽度的3倍加上间距的总和，然后给每个子元素设置固定的宽和高，为了让他换行可以使用float来实现，由于子元素的浮动形成了BFC，所以父元素ul使用overflow:hidden来消除浮动带来的影响。
```css
ul {
  width: 100%;
  height: 100%;
  overflow: hidden; /* 清除浮动 */
}

li {
  float: left; /* 左浮动 */
  width: 30%; /* 每个li占30%的宽度 */
  height: 30%; /* 每个li占30%的高度 */
  margin-right: 5%; /* 设置右边距 */
  margin-bottom: 5%; /* 设置下边距 */
}

li:nth-of-type(3n) {
  margin-right: 0; /* 每行最后一个元素没有右边距 */
}

li:nth-last-child(n + 7) {
  margin-bottom: 0; /* 最后一行的元素没有下边距 */
}
```

## inline-block实现
inline-block实现和float实现类似，都是用来让元素换行的
```css
ul {
  width: 100%;
  height: 100%;
  letter-spacing: -10px; /* 消除inline-block元素之间的空白 */
}

li {
  width: 30%; /* 每个li占30%的宽度 */
  height: 30%; /* 每个li占30%的高度 */
  display: inline-block; /* 设置为inline-block */
  margin-right: 5%; /* 设置右边距 */
  margin-bottom: 5%; /* 设置下边距 */
}

li:nth-of-type(3n) {
  margin-right: 0; /* 每行最后一个元素没有右边距 */
}

li:nth-last-child(n + 7) {
  margin-bottom: 0; /* 最后一行的元素没有下边距 */
}
```
需要注意的是设置为inline-block的元素之间可能会出现间隙，所以使用letter-spacing来消除这个间隙。该属性可以用来增加或减少字符间的空白（字符间距）。也可以给ul设置font-size: 0来消除间隙，但这种方法会导致子元素的字体大小也被设置为0，所以不太推荐。

## table实现
HTML结构：
```html
<ul class="table">
  <li>
    <div>1</div>
    <div>2</div>
    <div>3</div>
  </li>
  <li>
    <div>4</div> 
    <div>5</div>
    <div>6</div>
  </li>
  <li>
    <div>7</div>
    <div>8</div>
    <div>9</div>
  </li>
</ul>
```
首先给父元素设置为table布局，然后使用border-spacing设置单元格之间的间距，最后将li设置为表格行，将div设置为表格单元格，CSS样式如下：
```css
.table {
  width: 100%;
  height: 100%;
  display: table; /* 设置为table布局 */
  border-spacing: 10px; /* 设置单元格之间的间距 */
}

li {
  display: table-row; /* 设置为表格行 */
}

div {
  width: 30%; /* 每个div占30%的宽度 */
  height: 30%; /* 每个div占30%的高度 */
  display: table-cell; /* 设置为表格单元格 */
  text-align: center; /* 设置文本居中 */
  border-radius: 5px; /* 设置圆角 */
  background-color: skyblue; /* 设置背景颜色 */
}
```

# React 介绍

## 介绍

记录 React 框架的学习笔记、组件开发和状态管理等内容。

学习React的动机是因为要做[交互式水动力建模与模拟平台](/project)这个项目，由于港府方面对技术栈没有要求，于是在师兄的建议下开始边做边学各种前端主流技术栈，翻看了一下自己笔记本上的时间线，最开始了解到有这个项目是在2024年9月30日，不过这段时间也在为应对[江苏省长江崩岸预警应用系统](/project)项目验收做维护，实际真正开始学习React的时间是2024年12月4日，算起来到现在接触React整整11个月。

前几天整理项目，看到今年一月份闭关时仅浅浅学了一个月React做的[演示系统Demo](/project)源码里漫天乱飞的React钩子，不禁感慨从当初只知道堆屎山到现在可以熟练管理状态这一路走的实属不易，尽管现在用React做项目已经比较熟练，但对于React的底层原理我还是了解不多，因此我想把React再仔细重学一遍，故开此专栏做详细笔记。

主要学习参考自[React官方文档](https://react.docschina.org/)，我学习的版本为React v19。

# 为什么选择Expo

## Expo是什么

Expo是React Native的一个开源平台和工具链，主要用于简化移动应用的开发、构建和部署。它允许开发者使用JavaScript或TypeScript编写一次代码，即可同时生成iOS、Android和 Web应用，而无需直接处理复杂的原生代码配置。简单来说，Expo就像React Native的“上层加速器”，提供了开箱即用的原生模块（如相机、推送通知等 100+ 个），让初学者和团队快速上手。

## 选择原因
Expo的核心特点是：
- 跨平台支持：从单一代码库构建 iOS、Android 和 Web 应用，支持 Expo Go App 实时预览（无需模拟器）。
- 简化开发流程：内置 CLI 工具、热重载、调试器（如 React Native Debugger），以及云构建服务（EAS Build），无需本地安装 Xcode 或 Android Studio 就能测试 iOS。
- 模块化扩展：提供大量预置 API（如位置服务、文件系统），复杂需求可“弹出”（eject）到 bare workflow 集成自定义原生代码。
- 免费起步：核心功能免费，云构建有额度限制（超出需订阅）。

相比React Native CLI，Expo 减少了环境配置的痛点（如在 Windows 上开发 iOS），适合快速原型和 MVP 项目；CLI 更灵活但门槛高。

Expo的应用场景为社交 App、工具类应用、原型验证等，尤其适合跨平台需求。

# 129.求根节点到叶子节点数字之和
给你一个二叉树的根节点 `root` ，树中每个节点都存放有一个 `0` 到 `9` 之间的数字。

每条从根节点到叶节点的路径都代表一个数字：

例如，从根节点到叶节点的路径 `1 -> 2 -> 3` 表示数字 `123` 。
计算从根节点到叶节点生成的 **所有数字之和** 。

**叶节点** 是指没有子节点的节点。

示例1：
![129-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/129-1.jpg)
- **输入**：root = [1,2,3]
- **输出**：25
- **解释**：
```txt
从根到叶子节点路径 1->2 代表数字 12
从根到叶子节点路径 1->3 代表数字 13
因此，数字总和 = 12 + 13 = 25
```

示例2：
![129-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/129-2.jpg)
- **输入**：root = [4,9,0,5,1]
- **输出**：1026
- **解释**：
```txt
从根到叶子节点路径 4->9->5 代表数字 495
从根到叶子节点路径 4->9->1 代表数字 491
从根到叶子节点路径 4->0 代表数字 40
因此，数字总和 = 495 + 491 + 40 = 1026
```

## 解题思路
### 1.递归与回溯
既然原题意是要求每条路径的数字总和，且每向下一层前一层的结果就要乘10进一位来添加新的节点的结果，所以想到了DFS递归，当节点为空返回0即可，用一个数组来存储当前路径上的节点值，当到达新一层的节点时将当前节点值添加到数组中，当到达叶子节点时就将数组中的值进行遍历计算得到当前路径的数字并累加到结果中，关于进位注意到可以每遍历下一个新的数就将之前的求和结果乘10再加当前遍历的数字即可实现进位，即`sum = sum * 10 + num`。当前节点不是叶子节点时继续递归遍历左右子树，注意每次递归最后都应将当前节点值从数组中弹出以恢复现场。

- 时间复杂度：O(n)，其中n是二叉树中节点的个数。每个节点被访问一次。
- 空间复杂度：O(h)，其中h是二叉树的高度。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function sumNumbers(root: TreeNode | null): number {
    let ans = 0
    const result = []
    function dfs(node) {
        if (node === null) {
            return 0
        }
        result.push(node.val)
        if (node.left === null && node.right === null) {
            let sum = 0
            for (const num of result) {
                sum = sum * 10 + num
            }
            ans += sum
        } else {
            dfs(node.left)
            dfs(node.right)
        }
        result.pop()
    }
    dfs(root)
    return ans
};
```

### 2.递归优化
事实上用一个变量就可以维护当前路径的数字了，不需要用数组来存储当前路径上的节点值。每向下一层前一层的结果就要乘10进一位来添加新的节点的结果，即`sum = sum * 10 + num`，当到达叶子节点时就将当前路径的数字累加到结果中即可。

- 时间复杂度：O(n)，其中n是二叉树中节点的个数。每个节点被访问一次。
- 空间复杂度：O(h)，其中h是二叉树的高度。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function sumNumbers(root: TreeNode | null): number {
    let ans = 0
    function dfs(node, carry) {
        if (node === null) {
            return
        }
        carry = carry * 10 + node.val
        if (node.left === null && node.right === null) {
            ans += carry
            return
        }
        dfs(node.left, carry)
        dfs(node.right, carry)
    }
    dfs(root, 0)
    return ans
};
```

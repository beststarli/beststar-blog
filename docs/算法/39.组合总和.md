# 39.组合总和
给你一个 **无重复元素** 的整数数组 `candidates` 和一个目标整数 `target` ，找出 `candidates` 中可以使数字和为目标数 `target` 的 所有 **不同组合** ，并以列表形式返回。你可以按 **任意顺序** 返回这些组合。

`candidates` 中的 **同一个** 数字可以 **无限制重复被选取** 。如果至少一个数字的被选数量不同，则两种组合是不同的。 

对于给定的输入，保证和为 `target` 的不同组合数少于 `150` 个。

示例1：
- **输入**：candidates = [2,3,6,7], target = 7
- **输出**：[[2,2,3],[7]]
- **解释**：
```txt
2 和 3 可以形成一组候选，2 + 2 + 3 = 7 。注意 2 可以使用多次。
7 也是一个候选， 7 = 7 。
仅有这两种组合。
```

示例2：
- **输入**：candidates = [2,3,5], target = 8
- **输出**：[[2,2,2,2],[2,3,3],[3,5]]

示例3：
- **输入**：candidates = [2], target = 1
- **输出**：[]

## 解题思路
### 1.排序后回溯
我们可以先对输入数组进行排序，这样在回溯过程中就可以提前剪枝。当我们在回溯过程中发现当前组合的和已经超过了目标值 `target`，我们就可以停止继续探索这个组合，因为后续的数字只会更大，不可能再满足条件了。每一轮回溯结束后，需要恢复现场，回退可选项。

时间复杂度：O(2^n)，其中n是输入数组的长度。回溯算法的时间复杂度通常是指数级的，因为在最坏情况下，算法需要探索所有可能的组合。

```ts
function combinationSum(candidates: number[], target: number): number[][] {
    const state = []  // 状态子集合
    candidates.sort((a, b) => a - b)  // 对可选数字排序
    const start = 0  // 遍历起始点
    const res = []  // 结果数据
    backTrack(state, target, candidates, start, res)
    return res
};

function backTrack(state: number[], target: number, choices: number[], start: number, res: number[][]): void {
    // 子集的和为target时，记录解
    if (target === 0) {
        res.push([...state])
        return
    }

    // 遍历点从start开始，避免生成重复子集
    for (let i = start; i < choices.length; i++) {
        // 数组已排序，当与当前选项的和超过target就不可选
        if (target - choices[i] < 0) {
            break
        }
        state.push(choices[i])  // 尝试做出选择
        backTrack(state, target - choices[i], choices, i, res)  // 进行下一轮选择
        state.pop()  // 恢复现场，撤销选择
    }
}
```

## 二刷
### 1.排序后回溯原地递归
在构建递归函数时需要传入当前的索引 `index`，表示从 `candidates` 数组的哪个位置开始选择数字。这样可以避免重复选择已经使用过的数字，同时也保证了组合的顺序性。每次选择一个数字后，我们将目标值 `target` 减去该数字，并继续递归调用函数，递归的起点是当前循环的索引因为同一个数字可以无限制重复被选取，直到目标值为零或小于零。当`target`减去当前选择的数字后小于零时，说明当前组合已经不可能满足条件，我们可以直接返回，避免不必要的计算。

- 时间复杂度：O(2^n)，其中n是输入数组的长度。回溯算法的时间复杂度通常是指数级的，因为在最坏情况下，算法需要探索所有可能的组合。
- 空间复杂度：O(n)，其中n是输入数组的长度。递归调用栈的深度最多为n，因此空间复杂度为O(n)。

```ts
function combinationSum(candidates: number[], target: number): number[][] {
    candidates.sort((a, b) => a - b)
    const ans = []
    const path = []
    function dfs(index, sum) {
        if (sum === 0) {
            ans.push([...path])
            return 
        }

        for (let i = index; i < candidates.length; i++) {
            if (sum - candidates[i] < 0) {
                break
            }
            path.push(candidates[i])
            dfs(i, sum - candidates[i])
            path.pop()
        }
    }
    dfs(0, target)
    return ans
};
```

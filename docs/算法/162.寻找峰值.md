# [162.寻找峰值](https://leetcode.cn/problems/find-peak-element/description/)
峰值元素是指其值严格大于左右相邻值的元素。

给你一个整数数组 `nums`，找到峰值元素并返回其索引。数组可能包含多个峰值，在这种情况下，返回 任何一个峰值 所在位置即可。

你可以假设 `nums[-1] = nums[n] = -∞` 。

你必须实现时间复杂度为 `O(log n)` 的算法来解决此问题。

示例1：
- **输入**：nums = [1,2,3,1]
- **输出**：2
- **解释**：3 是峰值元素，你的函数应该返回其索引 2。

示例2：
- **输入**：nums = [1,2,1,3,5,6,4]
- **输出**：1 或 5 
- **解释**：你的函数可以返回索引 1，其峰值元素为 2；或者返回索引 5， 其峰值元素为 6。

## 解题思路
### 1.二分查找
题目说明需要使用O(log n)的时间复杂度，因此可以使用二分查找来解决。我们可以通过比较中间元素与其右邻居的大小关系来决定搜索的方向：
- 如果 `nums[mid] > nums[mid + 1]`，说明峰值在左侧（包括 mid），因此我们将右边界移动到 mid。
- 如果 `nums[mid] < nums[mid + 1]`，说明峰值在右侧，因此我们将左边界移动到 mid + 1。

我们不断缩小搜索范围，直到左边界等于右边界，此时左边界就是一个峰值的索引。所以二分查找循环的条件是 `left < right`，当循环结束时，`left` 就是峰值的索引。

- 时间复杂度：O(log n)，其中 n 是数组的长度。每次迭代将搜索空间减半。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间。

```ts
function findPeakElement(nums: number[]): number {
    let left = 0
    let right = nums.length - 1
    while (left < right) {
        const mid = Math.floor((left + right) / 2)
        if (nums[mid] > nums[mid + 1]) {
            right = mid
        } else {
            left = mid + 1
        }
    }
    return left
};
```

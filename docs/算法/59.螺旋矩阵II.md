# [59.螺旋矩阵II](https://leetcode.cn/problems/spiral-matrix-ii/description/)
给你一个正整数 `n` ，生成一个包含 `1` 到 `n²` 所有元素，且元素按顺时针顺序螺旋排列的 `n x n` 正方形矩阵 `matrix` 。

示例1：
![59](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/59.jpg)
- **输入**：n = 3
- **输出**：[[1,2,3],[8,9,4],[7,6,5]]

示例2：
- **输入**：n = 1
- **输出**：[[1]]

## 解题思路
### 1.方向计算
这个题的思路和[54.螺旋矩阵](/docs/算法/54.螺旋矩阵.md)类似，都是通过方向数组来控制遍历的方向。我们可以定义一个方向数组`DIRS`，表示四个方向：右、下、左、上。然后使用两个指针`i`和`j`来表示当前的位置，每次将当前位置的值赋为当前的计数值，然后根据当前的方向更新位置。如果下一个位置越界或者已经被访问过，就改变方向。

- 时间复杂度：O(n²)，其中 n 是矩阵的大小。我们需要遍历整个矩阵，每个元素只访问一次。
- 空间复杂度：O(1)，不需要额外的空间来存储矩阵，只需要常数级别的变量来存储当前的位置和方向。

```ts
const DIRS = [[0, 1], [1, 0], [0, -1], [-1, 0]]

function generateMatrix(n: number): number[][] {
    const ans = Array.from({ length: n }, () => Array(n).fill(0))
    let i = 0
    let j = 0
    let dir = 0
    for (let val = 1; val <= n * n; val++) {
        ans[i][j] = val
        const x = i + DIRS[dir][0]
        const y = j + DIRS[dir][1]
        if (x < 0 || y < 0 || x >= n || y >= n || ans[x][y] !== 0) {
            dir = (dir + 1) % 4
        }
        i += DIRS[dir][0]
        j += DIRS[dir][1]
    }
    return ans
};
```

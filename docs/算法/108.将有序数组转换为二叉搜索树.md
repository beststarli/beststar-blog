# 108.将有序数组转换为二叉搜索树
给你一个整数数组`nums`，其中元素已经按 **升序** 排列，请你将其转换为一棵 **平衡** 二叉搜索树。
:::info **平衡**
**平衡二叉树** 是指该树所有节点的左右子树的高度相差不超过 1。
:::

示例1：
![108-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/108-1.jpg)
- **输入**：nums = [-10,-3,0,5,9]
- **输出**：[0,-3,9,-10,null,5]
- **解释**：[0,-10,5,null,-3,null,9] 也将被视为正确答案：
![108-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/108-2.jpg)

示例2：
![108-3](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/108-3.jpg)
- **输入**：nums = [1,3]
- **输出**：[3,1]
- **解释**：[1,null,3] 和 [3,1] 都是高度平衡二叉搜索树。

## 解题思路
### 1.分治递归
注意到建立的平衡树的根节点就是数组的中间元素，左子树就是数组左半部分的平衡树，右子树就是数组右半部分的平衡树。我们可以使用递归的方式来构建这棵树。每次递归调用时，我们都找到当前数组的中间元素，将其作为当前子树的根节点，然后递归地构建左子树和右子树。这样在节点数量上就能保持平衡。

时间复杂度：O(n)，其中n是数组的长度。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function sortedArrayToBST(nums: number[]): TreeNode | null {
    function buildSubTree(left, right) {
        if (left > right) {
            return null
        }

        const mid = Math.floor((left + right) / 2)
        const root = new TreeNode(nums[mid])

        root.left = buildSubTree(left, mid - 1)
        root.right = buildSubTree(mid + 1, right)

        return root
    }
    return buildSubTree(0, nums.length - 1)
};
```

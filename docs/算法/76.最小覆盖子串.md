# 76.最小覆盖子串
给定两个字符串`s`和`t`，长度分别是`m`和`n`，返回 s 中的**最短窗口子串**，使得该子串包含`t`中的每一个字符（**包括重复字符**）。如果没有这样的子串，返回空字符串`""`。

:::info 子串
**子字符串**是字符串中连续的**非空**字符序列。
:::

测试用例保证答案唯一。

示例1：
- **输入**：s = "ADOBECODEBANC", t = "ABC"
- **输出**："BANC"
- **解释**：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。

示例2：
- **输入**：s = "a", t = "a"
- **输出**："a"
- **解释**：整个字符串 s 是最小覆盖子串。

示例3：
- **输入**：s = "a", t = "aa"
- **输出**：""
- **解释**：t 中两个字符 'a' 均应包含在 s 的子串中，因此没有符合条件的子字符串，返回空字符串。

**进阶**：
- 你能设计一个在`O(m + n)`时间内解决此问题的算法吗？

## 解题思路

## 进阶：在O(m + n)时间内解决此问题
### 滑动窗口解法
解法参考自[宫水三叶](https://leetcode.cn/problems/minimum-window-substring/solutions/2384313/gong-shui-san-xie-hua-dong-chuang-kou-ji-14q9/?envType=study-plan-v2&envId=top-100-liked)。

定义两个长度为 60（足够存下所有字母种类）的数组`c1`和`c2`，用于存储字符频率。其中`c1`用于记录字符串`t`中字符的频率，`c2`用于记录当前滑动窗口内字符的频率。

设定好字母与频率数组下标的映射关系：小写字母`a-z`对应下标`0-25`，大写字母`A-Z`对应下标`26-51`。

使用变量`tot`来记录还需要匹配的字符种类数，当`tot = 0`代表当前滑动窗口对应的子串能够实现对`t`的覆盖，即任意字符满足`c2[i]≥c1[i]`。

使用双指针`j`和`i`表示滑动窗口的左右边界。从前往后遍历字符串`s`，在每个位置上更新字符频率数组`c2`。若`c2`中字符的频率达到了`c1`中的字符频率，则将`tot`减 1，表示一个字符已经匹配完成。

每当右边界往后移动一步之后，滑动窗口会增加一个字符。此时我们检查左边界能否右移，同时不会使得 tot 变大。即每次右边界右移后，我们检查左边界`c2[j]>c1[j]`是否满足：
- 若满足：说明当前左边界指向字符并非必须，当前子串`s[j...i]`必然不是最短子串。我们让左边界`j`进行右移，并重复进行左边界`c2[j]>c1[j]`的检查，直到窗口不能再收缩
- 若不满足：说明当前窗口没有任何一个后缀字符串能够实现对`t`的覆盖，我们并不能对窗口实现收缩

每次对窗口移动完成后，我们检查当前`tot`是否为 0（对字符串`t`的覆盖是否完成），若为 0 则尝试用当前窗口对应的字符串`s[j...i]`更新`result`。

时间复杂度：O(m+n)，其中 m 和 n 分别是字符串 s 和 t 的长度。

```ts
function minWindow(s: string, t: string): string {
    let length = s.length
    let tot = 0
    const c1 = Array(60).fill(0)
    const c2 = Array(60).fill(0)

    for (const x of t) {
        if (++c1[getIndex(x)] === 1) {
            tot++
        }
    }

    let result = ""
    for (let i = 0, j = 0; i < length; i++) {
        const index1 = getIndex(s[i])
        if (++c2[index1] == c1[index1]) {
            tot--
        }

        while (j < i) {
            const index2 = getIndex(s[j])
            if (c2[index2] > c1[index2] && --c2[index2] >= 0) {
                j++
            } else {
                break
            }
        }
        if (tot == 0 && (!result || result.length > i - j + 1)) {
            result = s.substring(j, i + 1)
        }
    }
    return result
};

function getIndex(x: string): number {
    return x >= 'A' && x <= 'Z' ? x.charCodeAt(0) - 'A'.charCodeAt(0) + 26 : x.charCodeAt(0) - 'a'.charCodeAt(0)
}
```

## 二刷
### 滑动窗口
所谓“涵盖”就是`s`的子串中每个字母的出现次数都大于等于t中每个字母的出现次数。我们枚举`s`子串的右端点`right`，如果子串涵盖`t`，就不断右移左端点`left`直到不涵盖为止，在移动过程中更新最短子串的左右端点。

具体做法是：
1. 初始化`ansLeft = -1`，`ansRight = s.length`，用来记录最短子串的左右端点。
2. 用一个哈希表或数组`countT`统计`t`中每个字母的出现次数。
3. 初始化`left = 0`，以及一个空哈希表或数组`countS`来统计`s`子串中每个字母的出现次数。
4. 遍历`s`，设当前枚举的子串右端点为`right`，将`s[right]`的出现次数加一。
5. 遍历`countS`中的每个字母及其出现次数，如果出现次数都大于等于`countT`中的字母出现次数：
    - 如果`right - left < ansRight - ansLeft`，说明找到了更短的子串，更新`ansLeft = left`，`ansRight = right`。
    - 将`s[left]`的出现次数减一，右移`left`，`left++`。
    - 重复上述步骤，直到`countS`中有字母的出现次数小于`countT`中对应字母的出现次数。
6. 最后，如果`ansLeft < 0`，说明没有找到满足条件的子串，返回`""`；否则返回`s.substring(ansLeft, ansRight + 1)`。

由于本题大小写字母都有，为了方便代码实现可以直接创建大小为128的数组，将ASCII值座位数组的下标。

- 时间复杂度：O(m+n)，其中m和n分别是字符串s和t的长度。
- 空间复杂度：O(1)，因为countS和countT的大小是固定的128。

```ts
function isCovered(countS, countT) {
    for (let i = 'A'.charCodeAt(0); i <= 'Z'.charCodeAt(0); i++) {
        if (countS[i] < countT[i]) {
            return false
        }
    }
    for (let i = 'a'.charCodeAt(0); i <= 'z'.charCodeAt(0); i++) {
        if (countS[i] < countT[i]) {
            return false
        }
    }
    return true
}

function minWindow(s: string, t: string): string {
    const countS = Array(128).fill(0)   // s子串字母的出现次数
    const countT = Array(128).fill(0)   // t中字母的出现次数
    for (const char of t) {
        countT[char.charCodeAt(0)]++
    }

    let ansLeft = -1
    let ansRight = s.length
    let left = 0
    // 移动子串右端点
    for (let right = 0; right < s.length; right++) {
        countS[s[right].charCodeAt(0)]++    // 右端点字母移入子串
        // 涵盖
        while (isCovered(countS, countT)) {
            // 找到更短的子串
            if (right - left < ansRight - ansLeft) {
                // 记录此时的左右端点
                ansLeft = left
                ansRight = right
            }
            countS[s[left].charCodeAt(0)]-- // 左端点字母移出子串
            left++
        }
    }
    return ansLeft < 0 ? "" : s.substring(ansLeft, ansRight + 1)
};
```

### 优化涵盖判断
用一个变量`geCount`维护目前子串中有`geCount`种字母的出现次数大于等于t中对应字母的出现次数。设`t`有`kinds`个不同字母，那么子串每种字母的出现次数都大于等于t中对应字母的出现次数，等价于`geCount === kinds`。

如何维护`geCount`？为了方便实现把`countS`和`countT`合并为一个`diff`，定义`diff[lchar] = countS[lchar] - countT[lchar]`，当`diff[lchar]`从小于0变为0时，说明`lchar`的出现次数从比t的少变为一样多。
- 如果字母`lchar`进入窗口后，`diff[lchar] = 0`，意味着lchar在子串和t中的出现次数从小于变成大于等于，那么把`geCount`加1。
- 如果字母`lchar`移出窗口前，`diff[lchar] = 0`，意味着lchar在子串和t中的出现次数从大于等于变成小于，那么把`geCount`减1。

不能在`diff[lchar]`大于等于0的时候就把`geCount`加一，这样的话对于同一个字母`lchar`，`diff[lchar]`从0变为1、2、3...时都会把`geCount`加一，导致重复统计。

- 时间复杂度：O(m+n)，其中m和n分别是字符串s和t的长度。
- 空间复杂度：O(1)，因为diff的大小是固定的128。

```ts
function minWindow(s: string, t: string): string {
    const diff = Array(128).fill(0)     // 窗口每种字母个数 - t 每种字母个数
    let kinds = 0
    for (const char of t) {
        const code = char.codePointAt(0)
        if (diff[code] === 0) {
            kinds++    // 统计t有多少个不同的字母
        } 
        diff[code]--
    }

    let ansLeft = -1
    let ansRight = s.length
    let geCount = 0     // 窗口内有geCount种字母的出现次数大于等于t中相应字母的出现次数
    let left = 0
    // 移动子串右端点
    for (let right = 0; right < s.length; right++) {
        const char = s[right].codePointAt(0)    // 右端点字母
        diff[char]++    // 右端点字母移入子串
        // 原来窗口内c的出现次数比t的少，现在一样多
        if (diff[char] === 0) {
            geCount++   // 从小于变为大于等于
        }
        // 涵盖：所有字母的出现次数都是大于等于
        while (geCount === kinds) {
            // 找到更短的子串
            if (right - left < ansRight - ansLeft) {
                // 记录此时的左右端点
                ansLeft = left
                ansRight = right
            }
            const lchar = s[left].codePointAt(0) // 左端点字母
            if (diff[lchar] === 0) {
                // lchar移出窗口之前检查出现次数
                // 如果窗口内lchar的出现次数和t一样
                // 那么lchar移出窗口后，窗口内lchar的出现次数比t的少
                geCount--
            }
            diff[lchar]--   // 左端点字母移出子串
            left++
        }
    }
    return ansLeft < 0 ? "" : s.substring(ansLeft, ansRight + 1)
};
```

# [220.存在重复元素III](https://leetcode.cn/problems/contains-duplicate-iii/description/)   
给你一个整数数组 `nums` 和两个整数 `indexDiff` 和 `valueDiff` 。

找出满足下述条件的下标对 `(i, j)`：
- `i != j`,
- `abs(i - j) <= indexDiff`,
- `abs(nums[i] - nums[j]) <= valueDiff`

如果存在，返回 `true` ；否则，返回 `false` 。

示例1：
- **输入**：nums = [1,2,3,1], indexDiff = 3, valueDiff = 0
- **输出**：true
- **解释**：可以找出 (i, j) = (0, 3) 。
```txt
满足下述 3 个条件：
i != j --> 0 != 3
abs(i - j) <= indexDiff --> abs(0 - 3) <= 3
abs(nums[i] - nums[j]) <= valueDiff --> abs(1 - 1) <= 0
```

示例2：
- **输入**：nums = [1,5,9,1,5,9], indexDiff = 2, valueDiff = 3
- **输出**：false
- **解释**：尝试所有可能的下标对 (i, j) ，均无法满足这 3 个条件，因此返回 false 。

## 解题思路
### 1.定长滑动窗口遍历（超时）
用一个长为 `indexDiff` 的滑动窗口遍历数组，遍历时用一个双重循环，内层循环遍历滑动窗口内的元素，判断是否存在满足条件的下标对 `(i, j)`，如果存在则返回 `true`，否则返回 `false`。

:::danger **超时！**
这个方法无法通过全部测试用例。
:::

- 时间复杂度：O(n * indexDiff)，其中 n 是数组 nums 的长度。外层循环遍历数组，内层循环遍历滑动窗口，最坏情况下需要进行 n * indexDiff 次操作。
- 空间复杂度：O(1)，不需要额外的空间来存储数据结构。

```ts
function containsNearbyAlmostDuplicate(nums: number[], indexDiff: number, valueDiff: number): boolean {
    for (let i = 0; i < nums.length; i++) {
        let left = i
        let right = left + indexDiff
        for (let j = i + 1; j <= Math.min(i + indexDiff, nums.length - 1); j++) {
            if (Math.abs(nums[i] - nums[j]) <= valueDiff) {
                return true
            }
        }
    }
    return false
};
```

### 2.桶排序
将数组中的元素按照一定的规则分到不同的桶中，然后在桶内或者相邻的桶之间查找满足条件的元素。对于超出桶范围的元素，可以通过滑动窗口的方式来维护桶的大小，确保只保留最近的 `indexDiff` 个元素。这里的划分规则是将每个元素 `nums[i]` 映射到一个桶中，桶的大小为 `valueDiff + 1`，这样可以确保同一个桶内的元素之间的差值不会超过 `valueDiff`。如果两个元素在同一个桶中，则它们的差值一定小于等于 `valueDiff`。如果两个元素在相邻的桶中，则需要检查它们的差值是否小于等于 `valueDiff`。

- 时间复杂度：O(n)，其中 n 是数组 nums 的长度。遍历数组时，每个元素只会被放入桶中一次，并且每个桶最多只会存储一个元素，因此总的时间复杂度为 O(n)。
- 空间复杂度：O(n)，其中 n 是数组 nums 的长度。需要额外的空间来存储桶，最坏情况下每个元素都可能被放入不同的桶中，因此空间复杂度为 O(n)。

```ts
function containsNearbyAlmostDuplicate(nums: number[], indexDiff: number, valueDiff: number): boolean {
    const bucket = new Map()
    for (let i = 0; i < nums.length; i++) {
        const num = nums[i]
        const id = Math.floor(num / (valueDiff + 1))
        if (bucket.has(id)) {
            return true
        }
        if (bucket.has(id - 1) && Math.abs(num - bucket.get(id - 1)) <= valueDiff) {
            return true
        }
        if (bucket.has(id + 1) && Math.abs(num - bucket.get(id + 1)) <= valueDiff) {
            return true
        }
        bucket.set(id, num)
        if (i >= indexDiff) {
            bucket.delete(Math.floor(nums[i - indexDiff] / (valueDiff + 1)))
        }
    }
    return false
};
```

# 124.二叉树中的最大路径和
二叉树中的 **路径** 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 至多出现一次 。该路径 **至少包含一个** 节点，且不一定经过根节点。

**路径和** 是路径中各节点值的总和。

给你一个二叉树的根节点`root`，返回其 **最大路径和** 。

示例1：
![124-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/124-1.jpg)
- **输入**：root = [1,2,3]
- **输出**：6
- **解释**：最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6

示例2：
![124-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/124-2.jpg)
- **输入**：root = [-10,9,20,null,null,15,7]
- **输出**：42
- **解释**：最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42

## 解题思路
### 1.递归
这道题的思路与[543.二叉树的直径](https://leetcode.cn/problems/diameter-of-binary-tree/)类似，都是树形DP的解法。不同的是，这道题需要计算路径和，而不是路径长度。

我们还是对节点进行递归，分别递归计算左右子树的最大路径和，然后根据左右子树的结果计算当前节点的最大路径和，并更新全局最大路径和。对于左右子树路径和的计算有两种情况：
- 子树路径和为正数，那么父节点连接这条子树路径可以使路径和更大。
- 子树路径和为负数，那么父节点连接这条子树路径会使路径和更小，因此父节点不连接这条子树路径，即这个子树的路径和设为0，这与[53. 最大子数组和](https://leetcode.cn/problems/maximum-subarray/)的思路类似。

递归返回值的含义是：从当前节点出发，往下最多能得到的最大路径和，只能选一条路，不能同时选左右两条。

时间复杂度：O(n)，其中n是树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function maxPathSum(root: TreeNode | null): number {
    let maxSum = -Infinity

    function dfs(node) {
        if (node === null) {
            return 0
        }

        const leftSum = Math.max(dfs(node.left), 0)
        const rightSum = Math.max(dfs(node.right), 0)

        const currentPathSum = node.val + leftSum + rightSum
        maxSum = Math.max(maxSum, currentPathSum)

        return node.val + Math.max(leftSum, rightSum)
    }

    dfs(root)
    return maxSum
};
```

## 总结
本题考查的是树形DP，需要对树进行递归，计算每个节点的最大路径和，并更新全局最大路径和。递归返回值的含义是从当前节点出发，往下最多能得到的最大路径和，只能选一条路，不能同时选左右两条。对于子树路径和为负数的情况，需要将其设为0，表示不连接这条子树路径。

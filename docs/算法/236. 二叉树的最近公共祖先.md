# [236.二叉树的最近公共祖先](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/)
给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

[百度百科](https://baike.baidu.com/item/%E6%9C%80%E8%BF%91%E5%85%AC%E5%85%B1%E7%A5%96%E5%85%88/8918834?fr=aladdin)中最近公共祖先的定义为：“对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

示例1：
![236-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/236-1.png)
- **输入**：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
- **输出**：3
- **解释**：节点 5 和节点 1 的最近公共祖先是节点 3 。

示例2：
![236-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/236-2.png)
- **输入**：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 4
- **输出**：5
- **解释**：节点 5 和节点 4 的最近公共祖先是节点 5 。因为根据定义最近公共祖先节点可以为节点本身。

示例3：
- **输入**：root = [1,2], p = 1, q = 2
- **输出**：1

## 解题思路
### 1.递归
根据公共祖先的定义，我们对每个节点进行递归，那么当前节点作为子树的根节点递归时，返回值有以下几种情况：
- 返回当前节点本身：
    - 当前节点是空节点
    - 当前节点是p或q
    - 节点的左右子树都找到了
- 返回左子树的递归结果：节点只有左子树
- 返回右子树的递归结果：节点只有右子树
- 返回null：节点的左右子树都没有找到

返回的节点的准确含义是指「最近公共祖先的候选项」。对于最外层的递归调用者来说，返回值是最近公共祖先的意思。但是，**在递归过程中**，返回值可能是最近公共祖先，也可能是空节点（表示子树内没找到任何有用信息）、节点`p`或者节点`q`（可能成为最近公共祖先，或者用来**辅助判断**上面的某个节点是否为最近公共祖先）。

时间复杂度：O(n)，其中n是树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function lowestCommonAncestor(root: TreeNode | null, p: TreeNode | null, q: TreeNode | null): TreeNode | null {
    if (root === null || root === p || root === q) {
        return root
    }

    const left = lowestCommonAncestor(root.left, p, q)
    const right = lowestCommonAncestor(root.right, p, q)
       
    if (left !== null && right !== null) {
        return root
    }

    return left ?? right
};
```

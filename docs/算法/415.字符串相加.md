# 415.字符串相加
给定两个字符串形式的非负整数 `num1` 和 `num2` ，计算它们的和并同样以字符串形式返回。

你不能使用任何內建的用于处理大整数的库（比如 `BigInteger`）， 也不能直接将输入的字符串转换为整数形式。

示例1：
- **输入**：num1 = "11", num2 = "123"
- **输出**："134"

示例2：
- **输入**：num1 = "456", num2 = "77"
- **输出**："533"

示例3：
- **输入**：num1 = "0", num2 = "0"
- **输出**："0"

## 解题思路
### 1.反转相加进位
将字符串 `num1` 和 `num2` 反转，方便从最低位开始相加。使用一个变量 `carry` 来记录进位，初始化为 0。遍历较长的字符串的每一位，分别取出 `num1` 和 `num2` 当前位的数字（如果当前位不存在则视为 0），计算它们的和 `sum = digit1 + digit2 + carry`。将 `sum` 的个位数添加到结果数组中，更新 `carry = Math.floor(sum / 10)`。遍历结束后，如果 `carry` 大于 0，则将 `carry` 添加到结果数组中。最后将结果数组反转并连接成字符串返回。

- 时间复杂度：O(max(m, n))，其中 m 和 n 分别是字符串 num1 和 num2 的长度。我们需要遍历较长的字符串一次。
- 空间复杂度：O(max(m, n))，结果字符串的长度最多为 max(m, n) + 1（当两数相加产生进位时）。结果数组的长度也为 O(max(m, n))。

```ts
function addStrings(num1: string, num2: string): string {
    const arr1 = num1.split('')
    const arr2 = num2.split('')
    arr1.reverse()
    arr2.reverse()

    const biggerLen = Math.max(arr1.length, arr2.length)
    let carry = 0
    let result = []
    for (let i = 0; i < biggerLen; i++) {
        // Number(undefined)是NaN，无法处理
        const sum = Number(arr1[i] ?? 0) + Number(arr2[i] ?? 0) + carry
        result.push(String(sum % 10))
        carry = Math.floor(sum / 10)
    }

    // 处理最后的进位
    if (carry > 0) { 
        result.push(String(carry))
    }

    // 反转回来
    return result.reverse().join('')
};
```

### 2.双指针
不需要反转字符串，使用两个指针分别指向 `num1` 和 `num2` 的末尾，从最低位开始相加。每次取出当前位的数字（如果当前位不存在则视为 0），计算它们的和 `sum = digit1 + digit2 + carry`。将 `sum` 的个位数添加到结果数组中，更新 `carry = Math.floor(sum / 10)`。遍历结束后，如果 `carry` 大于 0，则将 `carry` 添加到结果数组中。最后将结果数组反转并连接成字符串返回。

- 时间复杂度：O(max(m, n))，其中 m 和 n 分别是字符串 num1 和 num2 的长度。我们需要遍历较长的字符串一次。
- 空间复杂度：O(max(m, n))，结果字符串的长度最多为 max(m, n) + 1（当两数相加产生进位时）。结果数组的长度也为 O(max(m, n))。

```ts
function addStrings(num1: string, num2: string): string {
    let i = num1.length - 1
    let j = num2.length - 1
    let carry = 0
    let ans = []

    while (i >= 0 || j >= 0 || carry !== 0) {
        let c1 = (i >= 0) ? num1.charCodeAt(i) - '0'.charCodeAt(0) : 0
        let c2 = (j >= 0) ? num2.charCodeAt(j) - '0'.charCodeAt(0) : 0
        let sum = c1 + c2 + carry
        ans.push(sum % 10)
        carry = Math.floor(sum / 10)
        i--
        j--
    }
    return ans.reverse().join('')
};
```

# 230.二叉搜索树中第K小的元素
给定一个二叉搜索树的根节点`root`，和一个整数`k`，请你设计一个算法查找其中第`k`小的元素（`k`从 1 开始计数）。
示例1：
![230-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/230-1.jpg)
- **输入**：root = [3,1,4,null,2], k = 1
- **输出**：1

示例2：
![230-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/230-2.jpg)
- **输入**：root = [5,3,6,2,4,null,null,1], k = 3
- **输出**：3

**进阶**：
- 如果二叉搜索树经常被修改（插入/删除操作）并且你需要频繁地查找第`k`小的值，你将如何优化算法？

## 解题思路
### 1.层序遍历后排序
既然是二叉树中第k小的元素，那么我们就可以对二叉树进行遍历，得到二叉树结点值的数组，然后对该数组进行排序，最后返回排序后数组的第k-1个元素即可。

时间复杂度：O(n log n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function kthSmallest(root: TreeNode | null, k: number): number {
    const queue = [root]
    const list = []
    while (queue.length) {
        let node = queue.shift() as TreeNode
        list.push(node.val)
        if (node.left) {
            queue.push(node.left)
        }
        if (node.right) {
            queue.push(node.right)
        }
    }

    list.sort((a, b) => a - b)
    return list[k - 1
    ]
};
```

### 2.中序遍历
对于二叉搜索树来说，中序遍历得到的数组内元素自然呈升序，那么通过对二叉树进行中序遍历，得到一个数组，返回该数组的第k-1个元素即可。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function kthSmallest(root: TreeNode | null, k: number): number {
    const valArr = []

    function getNodeVal(node: TreeNode | null) {
        if (node === null) {
            return
        }
        getNodeVal(node.left)
        valArr.push(node.val)
        getNodeVal(node.right)
    }
    getNodeVal(root)
    return valArr[k - 1]
};
```

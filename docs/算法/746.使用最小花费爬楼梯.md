# [746.使用最小花费爬楼梯](https://leetcode.cn/problems/min-cost-climbing-stairs/description/)   
给你一个整数数组 `cost` ，其中 `cost[i]` 是从楼梯第 `i` 个台阶向上爬需要支付的费用。一旦你支付此费用，即可选择向上爬一个或者两个台阶。

你可以选择从下标为 `0` 或下标为 `1` 的台阶开始爬楼梯。

请你计算并返回达到楼梯顶部的最低花费。

示例1：
- **输入**：cost = [10,15,20]
- **输出**：15
- **解释**：你将从下标为 1 的台阶开始。
```txt
- 支付 15 ，向上爬两个台阶，到达楼梯顶部。
总花费为 15 。
```

示例2：
- **输入**：cost = [1,100,1,1,1,100,1,1,100,1]
- **输出**：6
- **解释**：你将从下标为 0 的台阶开始。
```txt
- 支付 1 ，向上爬两个台阶，到达下标为 2 的台阶。
- 支付 1 ，向上爬两个台阶，到达下标为 4 的台阶。
- 支付 1 ，向上爬两个台阶，到达下标为 6 的台阶。
- 支付 1 ，向上爬一个台阶，到达下标为 7 的台阶。
- 支付 1 ，向上爬两个台阶，到达下标为 9 的台阶。
- 支付 1 ，向上爬一个台阶，到达楼梯顶部。
总花费为 6 。
```

## 解题思路
### 1.递归加记忆化
虽然给的cost数组，但是事实上要爬上楼顶需要走的是cost.length + 1步，所以我们可以定义一个函数dfs(i)表示爬上第i个索引台阶时的最小花费，那么我们可以从第i个索引台阶向上爬一步或者两步，分别对应的花费就是dfs(i-1)+cost[i-1]和dfs(i-2)+cost[i-2]，取最小值即可。最后返回的结果就是第cost.length个索引台阶的最小花费。   

- 时间复杂度：O(n)，其中 n 是数组 cost 的长度。我们需要计算每个台阶的最小花费，每个台阶只计算一次。
- 空间复杂度：O(n)，其中 n 是数组 cost 的长度。我们使用了一个大小为 n 的数组来存储每个台阶的最小花费。

```ts
function minCostClimbingStairs(cost: number[]): number {
    const memo = Array(cost.length + 1).fill(-1)
    function dfs(i) {
        if (i <= 1) {
            return 0
        }
        if (memo[i] !== -1) {
            return memo[i]
        }
        memo[i] = Math.min(dfs(i - 1) + cost[i - 1], dfs(i - 2) + cost[i - 2])
        return memo[i]
    }
    dfs(cost.length)
    return memo[cost.length]
};
```

### 2.递推
初始化记忆数组被0填充，原题说明了一定会有两个以上的楼梯，所以计算从索引2开始，每次计算当前索引的最小花费就是爬到先前一个或两个楼梯所用花费加上他们自身的花费这两个和的最小值，最终返回最后一个索引的最小花费即可。

- 时间复杂度：O(n)，其中 n 是数组 cost 的长度。我们需要计算每个台阶的最小花费，每个台阶只计算一次。
- 空间复杂度：O(n)，其中 n 是数组 cost 的长度。我们使用了一个大小为 n 的数组来存储每个台阶的最小花费。

```ts
function minCostClimbingStairs(cost: number[]): number {
    const memo = Array(cost.length + 1).fill(0)
    for (let i = 2; i <= cost.length; i++) {
        memo[i] = Math.min(memo[i - 1] + cost[i - 1], memo[i - 2] + cost[i - 2])
    }
    return memo[cost.length]
};
```

### 3.空间优化
注意到每个高度楼顶爬上去的花费只需要用到前一个和前两个台阶的花费，没必要再维护先前台阶的花费，那么用两个变量滚动更新这两个台阶的花费就可以得到最终的结果。注意使用一个temp保存新楼顶的花费，然后先更新memo0再更新memo1。

- 时间复杂度：O(n)，其中 n 是数组 cost 的长度。我们需要计算每个台阶的最小花费，每个台阶只计算一次。
- 空间复杂度：O(1)。我们只使用了两个变量来存储前两个台阶的最小花费。

```ts
function minCostClimbingStairs(cost: number[]): number {
    let memo0 = 0
    let memo1 = 0
    for (let i = 2; i <= cost.length; i++) {
        const newMemo = Math.min(memo1 + cost[i - 1], memo0 + cost[i - 2])
        memo0 = memo1
        memo1 = newMemo
    }
    return memo1
};
```

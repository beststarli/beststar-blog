# 15.三数之和
给你一个整数数组`nums`，判断是否存在三元组`[nums[i], nums[j], nums[k]]`满足`i != j`、`i != k`且`j != k`，同时还满足`nums[i] + nums[j] + nums[k] == 0`。请你返回所有和为`0`且不重复的三元组。

**注意**：答案中不可以包含重复的三元组。

示例1：
- **输入**：nums = [-1,0,1,2,-1,-4]
- **输出**：[[-1,-1,2],[-1,0,1]]
- **解释**：
    - nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
    - nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
    - nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
    - 不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
    - 注意，输出的顺序和三元组的顺序并不重要。

示例2：
- **输入**：nums = [0,1,1]
- **输出**：[]
- **解释**：唯一可能的三元组和不为 0 。

示例3：
- **输入**：nums = [0,0,0]
- **输出**：[[0,0,0]]
- **解释**：唯一可能的三元组和为0。

## 解题思路
### 1.双指针——两数之和变形
既然要寻找的是数组内三个数字和为0，那么将三数之和转化为两数之和，第一层循环固定一个数字，剩下的两数之和为第一个数字的相反数也就是要求的目标数，此时就变成了[167.两数之和II——输入有序数组](/docs/算法/两数之和II——输入有序数组)的思路了。这题目的难点在于不能有重复的三元组，如何避免重复？

首先需要对输入数组进行升序排序，在外层循环中如果nums[i]和前一个数字nums[i-1]相同，那么nums[i]作为第一个数字的三元组已经被前一个数字nums[i-1]的三元组包含了，所以直接跳过即可。

在内层循环中，当三数之和等于0时，为了避免计入重复的三元组，需要在移动左指针和右指针之前，先判断左指针和右指针当前的数字是否和下一个数字相同，如果相同则继续移动指针直到不相同为止。

可以做的优化有：
- 排序后的数组如果三个连续数字的和大于0了，那么后续的数字只会更大，所以直接跳出循环。
- 排序后的数组如果nums[i]与数组最尾部的两个数字的和小于0了，那么nums[i]作为第一个数字的三元组不可能满足条件，所以直接跳过。外层循环直接continue即可。

> ***写法二更好理解。***

时间复杂度：O(n²)

#### 写法一
```ts
function threeSum(nums: number[]): number[][] {
    const newNums = nums.sort((a, b) => a - b)
    const result = []

    for (let i = 0; i < newNums.length; i++) {
        if (i > 0 && newNums[i] === newNums[i - 1]) continue

        const target = -newNums[i]

        let left = i + 1
        let right = newNums.length - 1

        while (left < right) {
            if (newNums[left] + newNums[right] === target) {
                result.push([newNums[i], newNums[left], newNums[right]])

                while (left < right && newNums[left] === newNums[left + 1]) left++
                while (left < right && newNums[right] === newNums[right - 1]) right--

                left++
                right--
            } else if (newNums[left] + newNums[right] < target) {
                left++
            } else {
                right--
            }
        }
    }
    return result
}
```

#### 写法二
```ts
function threeSum(nums: number[]): number[][] {
    const length = nums.length
    nums.sort((a, b) => a - b)
    const result = []

    for (let i = 0; i < length - 2; i++) {
        const temp = nums[i]

        // 跳过重复数字
        if (i > 0 && temp === nums[i - 1]) {
            continue
        }
        // 优化一
        if (temp + nums[i + 1] + nums[i + 2] > 0) {
            break
        }
        // 优化二
        if (temp + nums[length - 2] + nums[length - 1] < 0) {
            continue
        }

        let left = i + 1
        let right = length - 1

        while (left < right) {
            const sum = temp + nums[left] + nums[right]
            if (sum > 0) {
                right--
            } else if (sum < 0) {
                left++
            // 三数之和为 0
            } else {
                result.push([temp, nums[left], nums[right]])
                left++
                right--
                // 跳过重复数字
                while (left < right && nums[left] === nums[left - 1]) {
                    left++
                }
                // 跳过重复数字
                while (right > left && nums[right] === nums[right + 1]) {
                    right--
                }
            }
        }
    }
    return result
};
```

# 121.买卖股票的最佳时机
给定一个数组`prices`，它的第`i`个元素`prices[i]`表示一支给定股票第`i`天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回`0`。

示例1：
- **输入**：[7,1,5,3,6,4]
- **输出**：5
- **解释**：在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

示例2：
- **输入**：[7,6,4,3,1]
- **输出**：0
- **解释**：在这种情况下, 没有交易完成, 所以最大利润为 0。

## 解题思路
### 1.动态更新最大利润与最低价格
我们可以在一次遍历中动态更新最大利润与最低价格。初始时，最大利润为0，最低价格为`prices[0]`。对于每一天的价格，我们首先更新最大利润为当前价格与最低价格的差值与当前最大利润的较大值，然后更新最低价格为当前价格与最低价格的较小值。最后返回最大利润。

时间复杂度：O(n)。

```ts
function maxProfit(prices: number[]): number {
    let ans = 0
    let minPrice = prices[0]

    for (const price of prices) {
        ans = Math.max(ans, price - minPrice) // 在遍历中不断变大
        minPrice = Math.min(minPrice, price)  // 在遍历中不断变小
    }

    return ans
};
```

## 思考题
### 1.如果一定要交易一次，代码应该如何修改？
- 答：如果必须交易一次，那么利润就不一定是正数了，在初始化最大利润时应该设置为`-Infinity`，而不是`0`。

### 2.返回一个长为`n`的`answer`数组，其中`answer[i]`等于删除`prices[i]`后（相当于禁止在第`i`天买卖股票）本题的答案。
- 答：对于每个被删除的`i`，最优交易发生在三中情况：
    - 全在删除的`i`的左边
    - 全在删除的`i`的右边
    - 跨越`i`的两边

用四个辅助数组来做预处理：
- prefix[i]：表示 `prices[0..i]` 的最大利润
- suffix[i]：表示 `prices[i..n-1]` 的最大利润
- minPricePrefix[i]：表示 `prices[0..i]` 的最小值
- maxPriceSuffix[i]：表示 `prices[i..n-1]` 的最大值

答案就是：`answer[i] = max(prefix[i-1], suffix[i+1], maxPriceSuffix[i+1] - minPricePrefix[i-1], 0)`

时间复杂度：O(n)。

空间复杂度：O(n)。

```ts
function maxProfitAfterRemoval(prices: number[]): number[] {
    const n = prices.length
    if (n === 0) return []

    // 预处理四个数组
    const prefix = new Array(n).fill(0)
    const minPrefix = new Array<number>(n)
    let curMin = prices[0]
    for (let i = 0; i < n; i++) {
        curMin = Math.min(curMin, prices[i])
        minPrefix[i] = curMin
        if (i > 0) {
            prefix[i] = Math.max(prefix[i - 1], prices[i] - minPrefix[i - 1])
        }
    }

    const suffix = new Array(n).fill(0)
    const maxSuffix = new Array<number>(n)
    let curMax = prices[n - 1]
    for (let i = n - 1; i >= 0; i--) {
        curMax = Math.max(curMax, prices[i])
        maxSuffix[i] = curMax
        if (i < n - 1) {
            suffix[i] = Math.max(suffix[i + 1], maxSuffix[i + 1] - prices[i])
        }
    }

    const answer = new Array<number>(n)
    for (let i = 0; i < n; i++) {
        const left = i > 0 ? prefix[i - 1] : 0
        const right = i < n - 1 ? suffix[i + 1] : 0
        const cross = (i > 0 && i < n - 1) ? maxSuffix[i + 1] - minPrefix[i - 1] : 0
        answer[i] = Math.max(0, left, right, cross)
    }
    return answer
}
```

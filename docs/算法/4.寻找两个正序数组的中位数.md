# 4.寻找两个正序数组的中位数
给定两个大小分别为`m`和`n`的正序（从小到大）数组`nums1`和`nums2`。请你找出并返回这两个正序数组的中位数。

算法的时间复杂度应该为`O(log (m+n))`。

示例1：
- **输入**：nums1 = [1,3], nums2 = [2]
- **输出**：2.00000
- **解释**：合并数组 = [1,2,3] ，中位数 2

示例2：
- **输入**：nums1 = [1,2], nums2 = [3,4]
- **输出**：2.50000
- **解释**：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5

## 解题思路
事实上就是要寻找两个有序数组中第`(m+n)/2`或第`(m+n)/2+1`小的数，取决于`m+n`的奇偶性。

### 1.双指针
解法参考自[灵茶山艾府](https://leetcode.cn/problems/median-of-two-sorted-arrays/solutions/2950686/tu-jie-xun-xu-jian-jin-cong-shuang-zhi-z-p2gd/?envType=study-plan-v2&envId=top-100-liked)。

![4-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/4-1.jpg)

设`nums1`的`nums2`的长度分别为`length1`和`length2`，且`length1 ≤ length2`（如果不满足则交换两个数组）：
- 为方便处理`i=−1`，即`nums1`有`0`个数在第一组的情况，我们可以往`nums1`的最左边插入一个哨兵`−∞`，这可以保证数组仍然是有序的。对于`j=−1`的情况也同理，往`nums2`的最左边插入一个`−∞`。
- 为方便处理 `i+1 = length1`，即`nums1`有`length1`个数在第一组的情况，我们可以往`nums1`的最右边插入一个哨兵`∞`，这可以保证数组仍然是有序的。对于`j+1 = length2`的情况也同理，往`nums2`的最右边插入一个`∞`。这可以避免 $nums1_{i+1}$ 和 $nums2_{j+1}$ 下标越界。
- 插入`−∞`和`∞`后，便可保证无论`nums1`和`nums2`是什么样的，一定存在一个`i`，满足 $nums1_i$ ≤ nums2_j+1$ 且 $nums1_i+1 > nums2_j$。
- `length1`和`length2`的值不变。

如此修改后，`i`的含义变成了`nums1`有`i`个数在第一组，`j`的含义变成了`nums2`有`j`个数在第一组。

初始化 `i=0`，那么 `j` 应该初始化成多少？

- 如果 $length1+length2$ 是偶数，那么每组的大小为 $\dfrac{length1+length2}{2}$，$j$ 应当初始化成 $\dfrac{length1+length2}{2}$。
- 如果 $length1+length2$ 是奇数，我们规定第一组比第二组多一个数，第一组的大小为 $\dfrac{length1+length2+1}{2}$，$j$ 应当初始化成 $\dfrac{length1+length2+1}{2}$。

两种情况可以合并为：$j$ 初始化成 $\left\lfloor\dfrac{length1+length2+1}{2}\right\rfloor$。

为了保证组的大小不变，$i$ 每增加 $1$，$j$ 就要减少 $1$。所以有：

$$
j = \left\lfloor\dfrac{length1+length2+1}{2}\right\rfloor - i
$$

根据图片中的结论，只要发现 $nums1_i \le nums2_{j+1}$ 且 $nums1_{i+1} > nums2_j$，那么：

- 如果 $length1+length2$ 是偶数，中位数为 $\dfrac{\max(nums1_i,\,nums2_j) + \min(nums1_{i+1},\,nums2_{j+1})}{2}$。
- 如果 $length1+length2$ 是奇数，中位数为 $\max(nums1_i,\,nums2_j)$。

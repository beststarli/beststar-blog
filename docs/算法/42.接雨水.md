# 42.接雨水
给定`n`个非负整数表示每个宽度为`1`的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例1：
![42](/img/docs/algorithm/42.png)
- **输入**：height = [0,1,0,2,1,0,1,3,2,1,2,1]
- **输出**：6
- **解释**：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

示例2：
- **输入**：height = [4,2,0,3,2,5]
- **输出**：9

## 解题思路
### 个人题解
#### 1.双指针暴力解法
本题理所应当会想到分块计算各雨水区的水量求和，然而可以采用更加原子化的思路解题，对于图中每列，向两侧遍历计算其左右两边的最高柱子，取较小值与当前列高度的差值即为该列能接的雨水量，值为正则进行累加，最后将累加的雨水量输出即为结果。

时间复杂度：O(n²)

:::warning **耗时极长**
本方法涉及两层for循环，虽可以通过全部测试用例完成解题，但实际耗时达到了**1132ms**，超级慢！！！
![42-1](/img/docs/algorithm/42-1.png)
:::

```ts
function trap(height: number[]): number {
    let vSum = 0

    if (height.length <= 2) {
        return vSum
    }

    for (let i = 1; i < height.length - 1; i++) {
        let leftSide = height[i]
        let rightSide = height[i]

        for (let l = i - 1; l >= 0; l--) {
            leftSide = Math.max(leftSide, height[l])
        }

        for (let r = i + 1; r <= height.length - 1; r++) {
            rightSide = Math.max(rightSide, height[r])
        }

        const vTemp = Math.min(leftSide, rightSide) - height[i]
        if (vTemp > 0) {
            vSum += vTemp
        }
    }

    return vSum
};
```

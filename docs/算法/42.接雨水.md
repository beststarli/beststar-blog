# 42.接雨水
给定`n`个非负整数表示每个宽度为`1`的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例1：
![42](/img/docs/algorithm/42.png)
- **输入**：height = [0,1,0,2,1,0,1,3,2,1,2,1]
- **输出**：6
- **解释**：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

示例2：
- **输入**：height = [4,2,0,3,2,5]
- **输出**：9

## 解题思路
### 1.双指针暴力解法
本题理所应当会想到分块计算各雨水区的水量求和，然而可以采用更加原子化的思路解题，对于图中每列，向两侧遍历计算其左右两边的最高柱子，取较小值与当前列高度的差值即为该列能接的雨水量，值为正则进行累加，最后将累加的雨水量输出即为结果。

时间复杂度：O(n²)

:::warning **耗时极长**
本方法涉及两层for循环，虽可以通过全部测试用例完成解题，但实际耗时达到了**1132ms**，超级慢！！！
![42-1](/img/docs/algorithm/42-1.png)
:::

```ts
function trap(height: number[]): number {
    let vSum = 0

    if (height.length <= 2) {
        return vSum
    }

    for (let i = 1; i < height.length - 1; i++) {
        let leftSide = height[i]
        let rightSide = height[i]

        for (let l = i - 1; l >= 0; l--) {
            leftSide = Math.max(leftSide, height[l])
        }

        for (let r = i + 1; r <= height.length - 1; r++) {
            rightSide = Math.max(rightSide, height[r])
        }

        const vTemp = Math.min(leftSide, rightSide) - height[i]
        if (vTemp > 0) {
            vSum += vTemp
        }
    }

    return vSum
};
```

### 2.双指针优化
对于**解法1**，在计算每列的左右最高柱子时，实际上存在大量重复计算，为了避免重复计算，可以把每列的左边最高柱子和右边最高柱子预先计算出来，存储在两个数组中，当前位置，左边的最高高度是前一个位置的左边最高高度和本高度的最大值，右边的最高高度同理，之后对于每列的雨水量计算就可以直接从两个数组中获取左右最高柱子高度。

时间复杂度：O(n)

```ts
function trap(height: number[]): number {
    let vSum = 0

    if (height.length <= 2) {
        return vSum
    }

    const maxLeft: number[] = []
    const maxRight: number[] = []

    maxLeft[0] = height[0]
    maxRight[height.length - 1] = height[height.length-1]

    for (let i = 1; i < height.length; i++) {
        maxLeft[i] = Math.max(maxLeft[i-1], height[i])
    }

    for (let j = height.length - 2; j >= 0; j--) {
        maxRight[j] = Math.max(maxRight[j+1], height[j])
    }

    for (let v = 0; v < height.length; v++) {
        vSum += Math.min(maxLeft[v], maxRight[v]) - height[v]
    }

    return vSum
};
```
### 3.单调栈
对于每列，是为了向左和向右寻找第一个比自己高的柱子来计算面积，应当能想到使用单调栈解法，以题目[739.每日温度](/docs/算法/每日温度)中[单调栈](/docs/算法/每日温度#单调栈)解法为思路，对于本题需要知道**单调栈是按照行方向来计算雨水的**。

单调栈内元素的顺序从栈底到栈顶应是递减的，因为一旦发现添加的柱子高度大于栈顶元素，此时就出现凹槽了，栈顶元素就是凹槽底部的柱子，栈顶第二个元素就是凹槽左边的柱子，而添加的元素就是凹槽右边的柱子。

当遇到相同高度的柱子时，更新栈内元素下标，将栈内旧元素下标弹出，将新元素下标推入栈中。

本题的逻辑主要是三种情况：
1. 当前遍历的元素（柱子）高度小于栈顶元素的高度 height[i] < height[st.top()]
2. 当前遍历的元素（柱子）高度等于栈顶元素的高度 height[i] == height[st.top()]
3. 当前遍历的元素（柱子）高度大于栈顶元素的高度 height[i] > height[st.top()]

注意，先将下标`0`的柱子加入到栈中，`st.push(0)`。 栈中存放我们遍历过的元素，所以先将下标`0`加进来。然后开始从下标`1`开始遍历所有的柱子，`for (int i = 1; i < height.size(); i++)`。

时间复杂度：O(n)

```ts
function trap(height: number[]): number {
    let vSum = 0

    const length = height.length
    const stack: number[] = []

    stack.push(0)
    for (let i = 1; i < length; i++) {
        let top = stack[stack.length - 1]
        if (height[top] > height[i]) {
            stack.push(i)
        } else if (height[top] === height[i]) {
            stack.pop()
            stack.push(i)
        } else {
            while (stack.length > 0 && height[top] < height[i]) {
                let mid = stack.pop()
                if (stack.length > 0) {
                    let left = stack[stack.length - 1]
                    let h = Math.min(height[left], height[i]) - height[mid]
                    let w = i - left - 1

                    vSum += h * w
                    top = stack[stack.length - 1]
                }
            }
            stack.push(i)
        }
    }
    return vSum
};
```

## 总结
1. 双指针解题关键：当前列雨水面积：`Math.min(leftMax, rightMax) - height[i]`。
2. 单调栈解题关键：单调栈是按照行方向来计算雨水的。

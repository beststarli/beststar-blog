# 20.有效的括号
给定一个只包括`'('`，`)`，`'{'`，`'}'`，`'['`，`']'` 的字符串`s`，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

示例1：
- **输入**：s = "()"
- **输出**：true

示例2：
- **输入**：s = "()[]{}"
- **输出**：true

示例3：
- **输入**：s = "(]"
- **输出**：false

示例4：
- **输入**：s = "([])"
- **输出**：true

示例5：
- **输入**：s = "([)]"
- **输出**：false

## 解题思路
### 1.栈
题意简单来说就是括号的开闭必须是包含或是并列，交叉闭合是错误的。注意到如果字符串的长度如果为奇数，那一定是错误的。我们可以使用栈来解决这个问题，遍历字符串，如果遇到左括号就入栈对应的右括号，如果遇到右括号就和栈顶元素比较，如果不相等即交叉闭合、或者栈为空即无法全部闭合则说明字符串无效。最后如果栈为空则说明字符串有效。

时间复杂度：O(n)。

```ts
function isValid(s: string): boolean {
    if (s.length % 2 !== 0) {
        return false
    }
    
    const stack: string[] = []

    for (const item of s) {
        if (item === '(') {
            stack.push(')')
        } else if (item === '[') {
            stack.push(']')
        } else if (item === '{') {
            stack.push('}')
        } else if (stack.length === 0 || stack.pop() !== item) {
            return false
        }
    }
    return stack.length === 0
};
```

### 2.哈希表优化if-else
我们可以使用哈希表来优化上面的if-else判断，减少代码量。实际上是创建了一个映射表对象。

时间复杂度：O(n)。

```ts
function isValid(s: string): boolean {
    if (s.length % 2 !== 0) {
        return false
    }
    
    const stack: string[] = []
    const map = {
        '(': ')',
        '[': ']',
        '{': '}'
    }

    for (const item of s) {
        if (map[item] !== undefined) {
            stack.push(map[item])
        } else if (stack.length === 0 || stack.pop() !== item) {
            return false
        }
    }
    return stack.length === 0
};
```

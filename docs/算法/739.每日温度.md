# 739.每日温度
给定一个整数数组`temperatures`，表示每天的温度，返回一个数组`answer`，其中`answer[i]`是指对于第`i`天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用`0`来代替。

示例1：
- **输入**：temperatures = [73,74,75,71,69,72,76,73]
- **输出**：[1,1,4,2,1,1,0,0]

示例2：
- **输入**：temperatures = [30,40,50,60]
- **输出**：[1,1,1,0]

示例3：
- **输入**：temperatures = [30,60,90]
- **输出**：[1,1,0]

## 解题思路
### 1.暴力循环
对于每一天，向后遍历寻找第一个比当前温度更高的温度，计算两者之间的天数差值并存入结果数组中，如果没有找到更高的温度则存入0。

时间复杂度：O(n²)

:::danger **超出时间限制**
本方法不能通过所用测试用例，不是一个可通过题解。
:::

```ts
function dailyTemperatures(temperatures: number[]): number[] {
    let answer: number[] = []

    for (let i = 0; i < temperatures.length; i++) {

        let j = i + 1
        while (j < temperatures.length && temperatures[i] >= temperatures[j]) {
            j++
        }

        if (j < temperatures.length) {
            answer.push(j - i)
        } else {
            answer.push(0)
        }
    }

    return answer
};
```

### 2.单调栈
用单调栈来维护遍历温度的索引，新推入栈索引的所指向的温度与栈顶索引指向的温度进行比较，大于则求差值推入结果数组并将栈顶元素推出，小于等于则将新索引推入栈中作为栈顶，属于前者情况则对栈内元素重复以上过程，直至遍历结束或后者情况发生则继续遍历。具体解法见[代码随想录](https://programmercarl.com/0739.%E6%AF%8F%E6%97%A5%E6%B8%A9%E5%BA%A6.html#%E6%80%9D%E8%B7%AF)

时间复杂度：O(n)

```ts
function dailyTemperatures(temperatures: number[]): number[] {
    const length = temperatures.length
    const stack: number[] = []
    const result: number[] = new Array(length).fill(0)

    stack.push(0)
    for (let i = 0; i < length; i++) {
        let top = stack[stack.length - 1]
        while (stack.length > 0 && temperatures[top] < temperatures [i]) {
            result[top] = i - top
            stack.pop()
            top = stack[stack.length - 1]
        }

        stack.push(i)
    }
    return result
};
```

## 总结
### 单调栈
:::secondary **何时想到使用单调栈？**
通常是一维数组，要寻找任一个元素的右边或者左边第一个比自己大或者小的元素的位置，此时我们就要想到可以用单调栈了。时间复杂度为O(n)。
:::

单调栈的本质是**空间换时间**，因为在遍历的过程中需要用一个栈来记录右边第一个比当前元素高的元素，优点是整个数组只需要遍历一次。

更直白来说，就是**用一个栈来记录遍历过的元素**，因为遍历数组的时候，并不知道之前都遍历了哪些元素，以至于遍历一个元素找不到是不是之前遍历过一个更小的，所以就需要用一个容器（这里用单调栈）来记录之前遍历过的元素。

在使用单调栈的时候首先要明确如下几点：
#### 1.单调栈里存放的元素是什么？
单调栈里只需要存放元素的下标`i`就可以了，如果需要使用对应的元素，直接用 `temperatures[i]` 获取。
#### 2.单调栈里元素是递增呢？ 还是递减呢？
单调栈的“递增/递减”一定要先约定**按哪个方向描述**，更常见的约定是**从栈底到栈顶**看单调性。

以本题「找右边第一个更大温度」为例：栈里存的是下标，栈中对应的温度 `temperatures[stack[*]]` 会保持**从栈底到栈顶递减**（也等价于：**从栈顶到栈底递增**）。
原因是：遇到更高温度 `temperatures[i]` 时，会不断弹出栈顶那些更低温度的下标并结算答案，直到栈顶温度不小于当前温度，再把当前下标入栈。

单调栈主要有三个判断条件：
- 当前遍历的元素`T[i]`小于栈顶元素`T[st.top()]`的情况
- 当前遍历的元素`T[i]`等于栈顶元素`T[st.top()]`的情况
- 当前遍历的元素`T[i]`大于栈顶元素`T[st.top()]`的情况

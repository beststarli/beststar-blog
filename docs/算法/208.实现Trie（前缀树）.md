# 208.实现Trie（前缀树）
[Trie](https://baike.baidu.com/item/%E5%AD%97%E5%85%B8%E6%A0%91/9825209?fr=aladdin)（发音类似 "try"）或者说 **前缀树** 是一种树形数据结构，用于高效地存储和检索字符串数据集中的键。这一数据结构有相当多的应用情景，例如自动补全和拼写检查。

请你实现 Trie 类：
- `Trie()` 初始化前缀树对象。
- `void insert(String word)` 向前缀树中插入字符串 `word` 。
- `boolean search(String word)` 如果字符串 `word` 在前缀树中，返回 `true`（即，在检索之前已经插入）；否则，返回 `false` 。
- `boolean startsWith(String prefix)` 如果之前已经插入的字符串 `word` 的前缀之一为 `prefix` ，返回 `true` ；否则，返回 `false` 。

示例1：
- **输入**：
```txt
["Trie", "insert", "search", "search", "startsWith", "insert", "search"]
[[], ["apple"], ["apple"], ["app"], ["app"], ["app"], ["app"]]
```
- **输出**：[null, null, true, false, true, null, true]
- **解释**：
```txt
Trie trie = new Trie();
trie.insert("apple");
trie.search("apple");   // 返回 True
trie.search("app");     // 返回 False
trie.startsWith("app"); // 返回 True
trie.insert("app");
trie.search("app");     // 返回 True
```

## 解题思路
### 1.二十六叉树
这道题的本意是建立一棵二十六叉树，树上的每个节点也都有26个子节点，对应字母表中的每个字母，当然开始的时候子节点是不存在的。对于insert字符串的每个字母都是向子节点前进的过程，直到字符串的最后一个字母，此时将当前节点标记为一个单词的结尾。对于search和startsWith方法也是类似的过程，只不过search需要判断最后一个节点是否是单词的结尾，而startsWith只需要判断是否存在这个前缀。

解法参考自[灵茶山艾府](https://leetcode.cn/problems/implement-trie-prefix-tree/solutions/2993894/cong-er-cha-shu-dao-er-shi-liu-cha-shu-p-xsj4/?envType=study-plan-v2&envId=top-100-liked)

- 初始化：创建一棵`26`叉树，一开始只有一个根节点`root`。`26`叉树的每个节点包含一个长为`26`的儿子节点列表`son`，以及一个布尔值`end`，表示是否为终止节点。
- `insert`：
    1. 遍历字符串`word`，同时用一个变量`curr`表示当前在`26`叉树的哪个节点，初始值为`root`。
    2. 如果`word[i]`不是`curr`的儿子，那么创建一个新的节点`node`作为`curr`的儿子。如果`word[i]=a`，那么把`node`记录到`curr`的`son[0]`中。如果`word[i]=b`，那么把`node`记录到`curr`的`son[1]`中。依此类推。
    3. 更新`curr`为儿子列表中的相应节点。
    4. 遍历结束，把`curr`的`end`标记为`true`。
- `search`和`startsWith`可以复用同一个函数`find`：
    1. 遍历字符串 `word`，同时用一个变量`curr`表示当前在`26`叉树的哪个节点，初始值为`root`。
    2. 如果`word[i]`不是`curr`的儿子，返回`0`。`search`和`startsWith`收到`0`之后返回`false`。
    3. 更新`curr`为儿子列表中的相应节点。
    4. 遍历结束，如果`curr`的`end`是`false`，返回`1`，否则返回`2`。`search`如果收到的是`2`，返回`true`，否则返回`false`。`startsWith`如果收到的是非`0`数字，返回`true`，否则返回`false`。

时间复杂度：初始化为O(1)，insert为O(n∣Σ∣)，其余为O(n)，其中n是word的长度，∣Σ∣=26是字符集合的大小。注意创建一个节点需要O(∣Σ∣)的时间（如果用的是数组）。

```ts
class Trie {
    root: Node

    constructor() {
        this.root = new Node()
    }

    insert(word: string): void {
        let curr = this.root
        for (const item of word) {
            const index = item.charCodeAt(0) - 'a'.charCodeAt(0)
            if (curr.son[index] === null) {
                curr.son[index] = new Node()
            }
            curr = curr.son[index]
        }
        curr.end = true
    }

    _find(word: string): number {
        let curr = this.root
        for (const item of word) {
            const index = item.charCodeAt(0) - 'a'.charCodeAt(0)
            if (curr.son[index] === null) {
                return 0
            }
            curr = curr.son[index]
        }
        return curr.end ? 2 : 1
    }

    search(word: string): boolean {
        return this._find(word) === 2
    }

    startsWith(prefix: string): boolean {
        return this._find(prefix) !== 0
    }
}

class Node {
    son: (Node | null)[]
    end: boolean

    constructor() {
        this.son = Array(26).fill(null)
        this.end = false
    }
}

/**
 * Your Trie object will be instantiated and called as such:
 * var obj = new Trie()
 * obj.insert(word)
 * var param_2 = obj.search(word)
 * var param_3 = obj.startsWith(prefix)
 */
```

## 二刷
### 1.哈希map优化
用哈希map替代数组来存储子节点，可以节省空间，因为不是每个节点都会有26个子节点。

- 时间复杂度：初始化为O(1)，insert为O(n)，其余为O(n)，其中n是word的长度。
- 空间复杂度：O(n)，其中n是所有插入的字符串的长度之和。

```ts
class Node {
    son: Map<string, Node>
    end: boolean

    constructor() {
        this.son = new Map()
        this.end = false
    }
}

class Trie {
    root: Node

    constructor() {
        this.root = new Node()
    }

    _find(word) {
        let curr = this.root
        for (const char of word) {
            if (curr.son.has(char) === false) {
                return 0
            }
            curr = curr.son.get(char)
        }
        return curr.end ? 2 : 1
    }

    insert(word: string): void {
        let curr = this.root
        for (const char of word) {
            if (curr.son.has(char) === false) {
                curr.son.set(char, new Node())
            }
            curr = curr.son.get(char)
        }
        curr.end = true
    }

    search(word: string): boolean {
        return this._find(word) === 2
    }

    startsWith(prefix: string): boolean {
        return this._find(prefix) !== 0
    }
}

 */
```

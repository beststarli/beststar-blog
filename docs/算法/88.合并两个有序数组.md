# 88.合并两个有序数组
给你两个按 非递减顺序 排列的整数数组 nums1 和 nums2，另有两个整数 m 和 n ，分别表示 nums1 和 nums2 中的元素数目。

请你 合并 nums2 到 nums1 中，使合并后的数组同样按 非递减顺序 排列。

注意：最终，合并后数组不应由函数返回，而是存储在数组 nums1 中。为了应对这种情况，nums1 的初始长度为 m + n，其中前 m 个元素表示应合并的元素，后 n 个元素为 0 ，应忽略。nums2 的长度为 n 。

**叶子节点** 是指没有子节点的节点。

示例1：
- **输入**：nums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
- **输出**：[1,2,2,3,5,6]
- **解释**：
```txt
需要合并 [1,2,3] 和 [2,5,6] 。
合并结果是 [1,2,2,3,5,6] ，其中斜体加粗标注的为 nums1 中的元素。
```

示例2：
- **输入**：nums1 = [1], m = 1, nums2 = [], n = 0
- **输出**：[1]
- **解释**：
```txt
需要合并 [1] 和 [] 。
合并结果是 [1] 。
```

示例3：
- **输入**：nums1 = [0], m = 0, nums2 = [1], n = 1
- **输出**：[1]
- **解释**：
```txt
需要合并的数组是 [] 和 [1] 。
合并结果是 [1] 。
注意，因为 m = 0 ，所以 nums1 中没有元素。nums1 中仅存的 0 仅仅是为了确保合并结果可以顺利存放到 nums1 中。
```

**进阶**：
- 你可以设计实现一个时间复杂度为 O(m + n) 的算法解决此问题吗？

## 解题思路
### 1.倒序遍历——进阶
理解题意后第一反应是感觉和[21.合并两个有序链表](/docs/算法/21.合并两个有序链表)类似，但是这道题并不允许开辟新的数组来存放结果，正序遍历因为num2元素的插入可能会导致num1头部的元素被覆盖，注意到num1的长度初始化已经是m + n了，尾部被0填充，那么可以倒序遍历。

首先分别对num1和num2分别初始化两个指向尾部的指针，在循环内比较两个指针指向的元素，将较大的元素放在num1的尾部，并将对应的指针向前移动，直到其中一个指针越界。最后可能会出现其中一个指针没有到达数组头部的情况：
- 如果是pointer1没有到达数组头部，无需处理，因为num1的前面部分已经是有序的了。
- 如果是pointer2没有到达数组头部，则需要将num2剩余的元素复制到num1的前面。

- 时间复杂度：O(m + n)，其中 m 和 n 分别是数组 nums1 和 nums2 的长度。我们需要遍历两个数组的所有元素来完成合并。
- 空间复杂度：O(1)，我们只使用了常数级别的额外空间来存储指针和临时变量。

```ts
/**
 Do not return anything, modify nums1 in-place instead.
 */
function merge(nums1: number[], m: number, nums2: number[], n: number): void {
    let pointer1 = m - 1
    let pointer2 = n - 1
    let index = m + n - 1
    while (pointer1 >= 0 && pointer2 >= 0 && index >= 0) {
        if (nums1[pointer1] > nums2[pointer2]) {
            nums1[index--] = nums1[pointer1]
            pointer1--
        } else {
            nums1[index--] = nums2[pointer2]
            pointer2--
        }
    }

    while (pointer2 >= 0) {
        nums1[index--] = nums2[pointer2]
        pointer2--
    }
};
```
### 2.逻辑优化
对[解法1](/#1.倒序遍历——进阶)的逻辑进行优化，事实上两个while循环可以进行合并，在第一个while循环内就可以处理pointer2没有到达数组头部的情况了。

- 时间复杂度：O(m + n)，其中 m 和 n 分别是数组 nums1 和 nums2 的长度。我们需要遍历两个数组的所有元素来完成合并。
- 空间复杂度：O(1)，我们只使用了常数级别的额外空间来存储指针和临时变量。

```ts
/**
 Do not return anything, modify nums1 in-place instead.
 */
function merge(nums1: number[], m: number, nums2: number[], n: number): void {
    let pointer1 = m - 1
    let pointer2 = n - 1
    let index = m + n - 1
    while (pointer2 >= 0) {
        if (pointer1 >= 0 && nums1[pointer1] > nums2[pointer2]) {
            nums1[index--] = nums1[pointer1]
            pointer1--
        } else {
            nums1[index--] = nums2[pointer2]
            pointer2--
        }
    }
};
```

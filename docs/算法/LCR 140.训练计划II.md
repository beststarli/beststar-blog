# [LCR 140.训练计划II](https://leetcode.cn/problems/lian-biao-zhong-dao-shu-di-kge-jie-dian-lcof/description/)
给定一个头节点为`head`的链表用于记录一系列核心肌群训练项目编号，请查找并返回倒数第`cnt`个训练项目编号对应的节点。

示例1：
- **输入**：head = [2,4,7,8], cnt = 1
- **输出**：8

## 解题思路
### 1.双指针
还是使用双指针，倒数的长度看作是一把尺子的长度，先让右指针走到这边尺子的右端，然后同时移动尺子，也就同时移动双指针，直到右侧指针超出指向了null，也就是while循环跳出的条件是`right === null`，这时左侧指针就指向了倒数第`cnt`个节点。

- 时间复杂度：O(n)，其中 n 是链表的节点数。
- 空间复杂度：O(1)，只需要常数空间存放若干变量。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function trainingPlan(head: ListNode | null, cnt: number): ListNode | null {
    let left = head
    let right = head

    for (let i = 0; i < cnt; i++) {
        right = right.next 
    }

    while (right !== null) {
        left = left.next
        right = right.next
    }
    return left
};
```

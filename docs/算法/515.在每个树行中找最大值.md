# [515.在每个树行中找最大值](https://leetcode.cn/problems/find-largest-value-in-each-tree-row/description/)
给定一棵二叉树的根节点 root ，请找出该二叉树中每一层的最大值。

示例1：
![515](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/515.png)
- **输入**：root = [1,3,2,5,3,null,9]
- **输出**：[1,3,9]

示例2：
- **输入**：root = [1,2,3]
- **输出**：[1,3]

## 解题思路
### 1.层序遍历——BFS
既然是输出每一层的最大值，那么就可以想到使用层序遍历的方式，遍历每一层的节点，记录每一层的最大值即可。

- 时间复杂度：O(n)，其中 n 是二叉树的节点数。每个节点恰好被访问一次。
- 空间复杂度：O(n)，其中 n 是二叉树的节点数。空间主要为队列的开销，最坏情况下队列中需要存储 n/2 个节点。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function largestValues(root: TreeNode | null): number[] {
    if (root === null) {
        return []
    }
    const ans = []
    let curr = [root]
    while (curr.length !== 0) {
        const next = []
        let max = -Infinity
        for (const node of curr) {
            if (node.left) {
                next.push(node.left)
            }
            if (node.right) {
                next.push(node.right)
            }
            max = Math.max(max, node.val)
        }
        curr = next
        ans.push(max)
    }
    return ans
};
```

### 2.递归——DFS
维护一个最大深度变量 `maxDepth`，以及一个 Map 用于存储每一层的最大值。通过深度优先搜索遍历二叉树，在遍历过程中更新每一层的最大值。当再次到达同一深度时，比较当前节点的值与已存储的最大值，更新最大值。最后根据 `maxDepth` 输出每一层的最大值。

- 时间复杂度：O(n)，其中 n 是二叉树的节点数。每个节点恰好被访问一次。
- 空间复杂度：O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function largestValues(root: TreeNode | null): number[] {
    const ans = []
    let maxDepth = 0
    const map = new Map()
    function dfs(node, depth) {
        if (node === null) {
            return
        }
        maxDepth = Math.max(maxDepth, depth)
        map.set(depth, Math.max(map.get(depth) ?? -Infinity, node.val))
        dfs(node.left, depth + 1)
        dfs(node.right, depth + 1)
    }
    dfs(root, 1)
    for (let i = 1; i <= maxDepth; i++) {
        ans.push(map.get(i))
    }  
    return ans
};
```

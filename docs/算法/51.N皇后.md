# 51.N皇后
按照国际象棋的规则，皇后可以攻击与之处在同一行或同一列或同一斜线上的棋子。

**n 皇后问题** 研究的是如何将`n`个皇后放置在`n×n`的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数`n`，返回所有不同的 **n 皇后问题** 的解决方案。

每一种解法包含一个不同的 **n 皇后问题** 的棋子放置方案，该方案中`'Q'`和`'.'`分别代表了皇后和空位。

示例1：
![51-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/51-1.jpg)
- **输入**：n = 4
- **输出**：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
- **解释**：如上图所示，4 皇后问题存在两个不同的解法。

示例2：
- **输入**：n = 1
- **输出**：[["Q"]]

## 解题思路
根据国际象棋中皇后的行进方式可以知道，在结果数组中，每一行每一列都只能有一个皇后，并且每条斜线上也只能有一个皇后。如果忽略斜线的限制，这道题事实上就成为了对列索引的全排列问题，参考[46.全排列](/docs/算法/46.全排列)。

下面我们只需要考虑斜线的限制即可。注意沿↗方向的格子，行号加列号是不变的；沿↖方向的格子，行号减列号是不变的。因此判断两个皇后是否在同一条斜线上，如果两个皇后行号加列号相同，或者行号减列号相同，那么这两个皇后就在一条斜线上即互相攻击。

### 1.两数组维护已放置皇后的行列号和与差
用两个数组 diag<sub>1</sub>和 diag<sub>2</sub> 分别标记之前放置的皇后的行号加列号，以及行号减列号。如果当前位置的行号加列号在 diag<sub>1</sub> 中（标记为`true`），或者当前位置的行号减列号在 diag<sub>2</sub> 中（标记为`true`），那么当前位置被之前放置的皇后攻击到，不能放皇后。

时间复杂度：O(n^2 * n!)，其中 n 是棋盘的大小。

```ts
function solveNQueens(n: number): string[][] {
    const ans = []
    const queens = Array(n).fill(0) // 皇后放在 (r,queens[r])
    const col = Array(n).fill(false)
    const diag1 = Array(n * 2 - 1).fill(false)
    const diag2 = Array(n * 2 - 1).fill(false)

    function dfs(r) {
        if (r === n) {
            ans.push(queens.map(c => '.'.repeat(c) + 'Q' + '.'.repeat(n - 1 - c)))
            return 
        }

        // 在 (r,c) 放皇后
        for (let c = 0; c < n; c++) {
            const rc = r - c + n - 1
            // 判断能否放皇后
            if (!col[c] && !diag1[r + c] && !diag2[rc]) {
                queens[r] = c   // 直接覆盖，无需恢复现场
                col[c] = diag1[r + c] = diag2[rc] = true  // 皇后占用了 c 列和两条斜线
                dfs(r + 1)
                col[c] = diag1[r + c] = diag2[rc] = false  // 恢复现场
            }
        }
    }

    dfs(0)
    return ans
};
```

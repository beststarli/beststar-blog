# 5.最长回文子串
给你一个字符串`s`，找到`s`中最长的 **回文** **子串**。

:::info **回文子串**
**回文性**：如果字符串向前和向后读都相同，则它满足 **回文性**。

**子字符串**：**子字符串** 是字符串中连续的 **非空** 字符序列。
:::

示例1：
- **输入**：s = "babad"
- **输出**："bab"
- **解释**："aba" 同样是符合题意的答案。

示例2：
- **输入**：s = "cbbd"
- **输出**："bb"

## 解题思路

### 1.中心扩散
对于字符串的中心子串，像两边扩散，如果两个字母相同，则继续扩散，直到不相同为止。这样做需要判断字符串长度的奇偶性，中心子串可能是一个字母，也可能是两个字母。

对于奇数字符串，枚举i = 0， 1，...，n-1，以s[i]为中心子串进行扩散：
- 如果left = right = i，说明中心子串是一个字母。
- 如果s[left] == s[right]，那么向左右两侧扩展，将left向左移动一位，right向右移动一位。继续判断s[left]和s[right]是否相同，直到不相同为止。
- 循环结束，最后一轮的s[left + 1]和s[right - 1]是相同的，说明以s[i]为中心子串的最长回文子串是s[left + 1...right - 1]。

对于偶数字符串，枚举i = 0， 1，...，n-2，以s[i]和s[i + 1]为中心子串进行扩散，其余同上。

时间复杂度：$O(n^2)$，其中 n 是字符串 s 的长度。

空间复杂度：$O(1)$，只需要常数空间存储若干变量。


#### 写法一——分别处理奇偶
```ts
function longestPalindrome(s: string): string {
    const length = s.length
    let ansLeft = 0
    let ansRight = 0

    // 奇数回文串
    for (let i = 0; i < length; i++) {
        let left = i
        let right = i

        while (left >= 0 && right < length && s[left] === s[right]) {
            left--
            right++
        }

        // 循环结束，s[left + 1]到s[right - 1]是答案
        if (right - left - 1 > ansRight - ansLeft) {
            ansLeft = left + 1
            ansRight = right    // 左闭右开区间
        }
    }

    // 偶数回文串
    for (let i = 0; i < length - 1; i++) {
        let left = i
        let right = i + 1

        while (left >= 0 && right < length && s[left] === s[right]) {
            left--
            right++
        }

        if (right - left - 1 > ansRight - ansLeft) {
            ansLeft = left + 1
            ansRight = right    // 左闭右开区间
        }
    }

    return s.substring(ansLeft, ansRight)
};
```

#### 写法二——合并处理奇偶
枚举i = 0, 1, 2, ..., 2n - 2，规定：
- 当i为偶数时，使用枚举奇数回文串的规则，初始化left = right = i / 2。
- 当i为奇数时，使用枚举偶数回文串的规则，初始化$left = \lfloor i / 2 \rfloor$，$right = \lceil i / 2 \rceil$。

两种情况合并后得到：$left = \lfloor i / 2 \rfloor$，$right = \lceil i / 2 \rceil = \lfloor {(i + 1)} / 2 \rfloor$。

```ts
function longestPalindrome(s: string): string {
    const length = s.length
    let ansLeft = 0
    let ansRight = 0

    for (let i = 0; i < 2 * length - 1; i++) {
        let left = Math.floor(i / 2)
        let right = Math.floor((i + 1) / 2)

        while (left >= 0 && right < length && s[left] === s[right]) {
            left--
            right++
        }

        if (right - left - 1 > ansRight - ansLeft) {
            ansLeft = left + 1
            ansRight = right
        }
    }

    return s.substring(ansLeft, ansRight)
};
```

## 2.Manacher算法
解法参考自[灵茶山艾府](https://leetcode.cn/problems/longest-palindromic-substring/solutions/2958179/mo-ban-on-manacher-suan-fa-pythonjavacgo-t6cx/?envType=study-plan-v2&envId=top-100-liked)

Manacher算法的核心思想是利用回文串的对称性来减少不必要的比较，从而将时间复杂度降低到O(n)。该算法通过预处理字符串来统一处理奇偶长度的回文串，并使用一个辅助数组来记录每个位置的回文半径。然后根据下标转换关系来计算出最长回文子串的起始位置和长度。

时间复杂度：$O(n)$，其中 n 是字符串 s 的长度。

空间复杂度：$O(n)$，其中 n 是字符串 s 的长度，主要用于存储预处理后的字符串和回文半径数组。

```ts
function longestPalindrome(s: string): string {
    // 预处理：在每个字符之间插入“#”，并在首尾添加哨兵
    let t = '#'
    for (const char of s) {
        t += char + '#'
    }
    t += '$'

    const length = t.length
    const radius = new Array(length).fill(0)

    let center = 0  // 当前回文中心
    let right = 0   // 当前回文的最右边界
    let maxLength = 0   // 最长回文长度
    let maxCenter = 0   // 最长回文中心在字符串t中的位置

    // Manacher算法核心
    for (let i = 1; i < length - 1; i++) {
        // 利用对称性跳过已知部分
        const mirror = 2 * center - i
        if (i < right) {
            radius[i] = Math.min(right - i, radius[mirror])
        }

        // 向两边扩展
        while (t[i + radius[i] + 1] === t[i - radius[i] - 1]) {
            radius[i]++
        }

        // 更新right和center
        if (i + radius[i] > right) {
            center = i
            right = i + radius[i]
        }

        // 记录最长回文
        if (radius[i] > maxLength) {
            maxLength = radius[i]
            maxCenter = i
        }
    }

    // 从处理后的字符串t中还原原始回文子串
    const start = Math.floor((maxCenter - maxLength) / 2)
    return s.substring(start, start + maxLength)
};
```

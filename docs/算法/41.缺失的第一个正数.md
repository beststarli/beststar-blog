# 41.缺失的第一个正数
给你一个未排序的整数数组`nums`，请你找出其中没有出现的最小的正整数。

请你实现时间复杂度为`O(n)`并且只使用常数级别额外空间的解决方案。

示例1：
- **输入**：nums = [1,2,0]
- **输出**：3
- **解释**：范围 [1,2] 中的数字都在数组中。

示例2：
- **输入**：nums = [3,4,-1,1]
- **输出**：2
- **解释**：1 在数组中，但 2 没有。

示例3：
- **输入**：nums = [7,8,9,11,12]
- **输出**：1
- **解释**：最小的正数 1 没有出现。

## 解题思路
### 1.筛选、去重、排序
注意到题干关注的是正整数，那么第一步首先想多对输入数组内元素进行筛选，只保留正数，筛选时应注意边界条件，数组内元素全为非正数时直接返回1。对于筛选后数组，可能存在重复元素，此时想到哈希set自带去重，于是第二步是把筛选后的元素放入set中。最后一步是对set内元素添入新的数组，并对新数组进行排序，排序后遍历新数组内元素，找到第一个不连续的元素的索引即为答案。

:::danger **超出时间复杂度限制**
题干要求时间复杂度为O(n)，而排序的时间复杂度为O(nlogn)，因此该解法并不满足题干要求。
:::

时间复杂度：O(nlogn)

```ts
function firstMissingPositive(nums: number[]): number {
    nums.sort((a, b) => a - b)

    let filterArray = []
    filterArray = nums.filter(num => num > 0)

    if (filterArray.length === 0) {
        return 1
    }

    const indexSet = new Set()

    filterArray.map(item => {
        indexSet.add(item)
    })

    const newArray = []
    for (const item of indexSet) {
        newArray.push(item)
    }

    newArray.sort((a, b) => a - b)

    if (newArray[0] > 1) {
        return 1
    }

    let index = 1
    for (let i = 0; i < newArray.length; i++) {
        if (newArray[i] === index) {
            index++
        } else if (newArray[i] > index) {
            return index
        }
    }

    return index
};
```

### 2.交换位置
解法参考自[灵茶山艾府](https://leetcode.cn/problems/first-missing-positive/solutions/3655377/huan-zuo-wei-tong-guo-li-zi-li-jie-suan-qa94e/?envType=study-plan-v2&envId=top-100-liked)。

想象有一间教室，座位从左到右编号为`1`到`n`。有`n`个学生坐在教室的座位上，把`nums[i]`当作坐在第`i`个座位上的学生的学号。我们要做的事情，就是让学号在`1`到`n`中的学生，都坐到编号与自己学号相同的座位上（学号与座位编号匹配）。学号不在`[1,n]`中的学生可以忽略。学生们交换座位后，从左往右看，第一个学号与座位编号不匹配的学生，其座位编号就是答案。特别地，如果所有学生都坐在正确的座位上，那么答案是`n+1`。

判断「学生是否坐在正确的座位上」，能用`nums[i] === i`判断吗？注意有影分身（重复元素）。

一般地，为了兼容「当前学生是真身，坐在正确的座位上」和「当前学生是影分身，且其真身坐在正确的座位上」两种情况，我们可以把`i === nums[i]`套一层`nums`，用`nums[i] === nums[nums[i]]`判断:
- 无论「当前学生是真身，坐在正确的座位上」还是「当前学生是影分身，且其真身坐在正确的座位上」，上式都是成立的。
- 如果「当前学生是真身，不坐在正确的座位上」，那么上式左边是当前学生的学号，右边是要交换的学生的学号。
- 如果「当前学生是影分身，且其真身不坐在正确的座位上」，那么上式左边是当前学生的学号，右边是要交换的学生的学号。虽然是用影分身交换的，但交换后，可以认为真身已经坐在了正确的座位上。

代码实现时，由于`nums`的下标是从`0`开始的，通过学号访问下标，要把学号减一。

时间复杂度：O(n)

空间复杂度：O(1)

```ts
function firstMissingPositive(nums: number[]): number {
    const n = nums.length;
    for (let i = 0; i < n; i++) {
        // 如果当前学生的学号在 [1,n] 中，但（真身）没有坐在正确的座位上
        while (1 <= nums[i] && nums[i] <= n && nums[nums[i] - 1] !== nums[i]) {
            // 那么就交换 nums[i] 和 nums[j]，其中 j 是 i 的学号
            const j = nums[i] - 1; // 减一是因为数组下标从 0 开始
            [nums[i], nums[j]] = [nums[j], nums[i]];
        }
    }

    // 找第一个学号与座位编号不匹配的学生
    for (let i = 0; i < n; i++) {
        if (nums[i] !== i + 1) {
            return i + 1;
        }
    }

    // 所有学生都坐在正确的座位上
    return n + 1;
};
```

### 3.原地哈希
核心思路是把每个数`x`（如果它在`1～n`范围内）“放到”它应该在的位置（索引`x-1`），用**负号**来标记“这个数出现过”。最后哪个位置还是正数，说明对应的数（`i+1`）没出现过，那就是答案。

时间复杂度：O(n)

空间复杂度：O(1)

```ts
function firstMissingPositive(nums: number[]): number {
    const n = nums.length;

    // 1. 负数/0 置为 n+1
    for (let i = 0; i < n; ++i) {
        if (nums[i] <= 0) nums[i] = n + 1;
    }

    // 2. 标记
    for (let i = 0; i < n; ++i) {
        const idx = Math.abs(nums[i]) - 1;
        if (idx < n) nums[idx] = -Math.abs(nums[idx]);
    }

    // 3. 找第一个正数
    for (let i = 0; i < n; ++i) {
        if (nums[i] > 0) return i + 1;
    }

    return n + 1;
};
```

## 二刷
### 交换位置
- 时间复杂度：O(n)
- 空间复杂度：O(1)

```ts
function firstMissingPositive(nums: number[]): number {
    const length = nums.length
    for (let i = 0; i < length; i++) {
        while (nums[i] >= 1 && nums[i] <= length && nums[nums[i] - 1] !== nums[i]) {
            const j = nums[i] - 1;
            [nums[i], nums[j]] = [nums[j], nums[i]]
        }
    }

    for (let i = 0; i < length; i++) {
        if (nums[i] !== i + 1) {
            return i + 1
        }
    }
    return length + 1
};
```

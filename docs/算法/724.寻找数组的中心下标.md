# [724.寻找数组的中心下标](https://leetcode.cn/problems/find-pivot-index/description/)
给你一个下标从 **0** 开始的整数数组 `nums` ，请你找到 **最左边** 的中间位置 `middleIndex` （也就是所有可能中间位置下标最小的一个）。

中间位置 `middleIndex` 是满足 `nums[0] + nums[1] + ... + nums[middleIndex-1] == nums[middleIndex+1] + nums[middleIndex+2] + ... + nums[nums.length-1]` 的数组下标。

如果 `middleIndex == 0` ，左边部分的和定义为 `0` 。类似的，如果 `middleIndex == nums.length - 1` ，右边部分的和定义为 `0` 。

请你返回满足上述条件 **最左边** 的 `middleIndex` ，如果不存在这样的中间位置，请你返回 `-1` 。

示例1：
- **输入**：nums = [2,3,-1,8,4]
- **输出**：3
- **解释**：
```txt
下标 3 之前的数字和为：2 + 3 + -1 = 4
下标 3 之后的数字和为：4 = 4
```

示例2：
- **输入**：nums = [1,-1,4]
- **输出**：2
- **解释**：
```txt
下标 3 之前的数字和为：2 + 3 + -1 = 4
下标 3 之后的数字和为：4 = 4
```

示例3：
- **输入**：nums = [2,5]
- **输出**：-1
- **解释**：不存在符合要求的 middleIndex 。

示例4：
- **输入**：nums = [1]
- **输出**：0
- **解释**：
```txt
下标 0 之前的数字和为：0
下标 0 之后的数字和为：0
```

注意：本题与主站 1991 题相同：[https://leetcode.cn/problems/find-the-middle-index-in-array/](https://leetcode.cn/problems/find-the-middle-index-in-array/)

## 解题思路
### 1.前缀和
根据题意可以注意到，在中心下标处，左侧和的二倍加上中心下标数值等于整个数组的和，即：
```txt
2 * leftSum + nums[i] = totalSum
```
用一个哈希map来维护至当前下标的前缀和，这样通过`value - nums[index]`就可以得到左侧的和，`allSum - value`就可以得到右侧的和。通过比较当相等时返回当前下标即可。

- 时间复杂度：O(n)，其中 n 是数组的长度。我们需要遍历数组一次来计算总和，并在遍历过程中计算前缀和。
- 空间复杂度：O(n)，用于存储前缀和的哈希表。

```ts
function pivotIndex(nums: number[]): number {
    const allSum = nums.reduce((a, b) => a + b)
    const prefixMap = new Map()
    let sum = 0
    for (let i = 0; i < nums.length; i++) {
        sum += nums[i]
        prefixMap.set(i, sum)
    }
    for (const [index, value] of prefixMap.entries()) {
        const left = value - nums[index]
        const right = allSum - value
        if (left === right) {
            return index
        }
    }
    return -1
};
```

### 2.前缀和优化
我们可以在遍历数组的同时计算前缀和，而不需要额外的哈希表来存储前缀和。我们可以使用一个变量 `leftSum` 来维护当前下标左侧的和，然后通过 `totalSum - leftSum - nums[i]` 来计算右侧的和。这样可以节省空间。注意应先计算右侧的和，比对后再更新左侧的和。

- 时间复杂度：O(n)，其中 n 是数组的长度。我们需要遍历数组一次来计算总和，并在遍历过程中计算前缀和。
- 空间复杂度：O(1)，我们只使用了常数级别的额外空间。

```ts
function pivotIndex(nums: number[]): number {
    const allSum = nums.reduce((a, b) => a + b)
    let leftSum = 0
    for (let i = 0; i < nums.length; i++) {
        const rightSum = allSum - leftSum - nums[i]
        if (rightSum === leftSum) {
            return i
        }
        leftSum += nums[i]
    }
    return -1
};
```

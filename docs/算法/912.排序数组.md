# [912.排序数组](https://leetcode.cn/problems/sort-an-array/description/)
给你一个整数数组 `nums`，请你将该数组升序排列。

你必须在 **不使用任何内置函数** 的情况下解决问题，时间复杂度为 `O(nlog(n))`，并且空间复杂度尽可能小。

示例1：
- **输入**：nums = [5,2,3,1]
- **输出**：[1,2,3,5]
- **解释**：数组排序后，某些数字的位置没有改变（例如，2 和 3），而其他数字的位置发生了改变（例如，1 和 5）。

示例2：
- **输入**：nums = [5,1,1,2,0,0]
- **输出**：[0,0,1,1,2,5]
- **解释**：请注意，nums 的值不一定唯一。


## 解题思路
### 1.快速排序
参考[215.数组中的第K个最大元素](/docs/算法/215.数组中的第K个最大元素.md)中快速选择算法的思想，对数组应用快速选择后递归对左右两半再进行快速排序，直到order为true，说明已经排好序。

- 时间复杂度：O(nlog(n))，其中 n 是数组 nums 的长度。快速排序的平均时间复杂度为 O(nlog(n))。
- 空间复杂度：O(log(n))，其中 n 是数组 nums 的长度。

```ts
function partition(nums: number[], left: number, right: number) {
    const index = Math.floor(Math.random() * (right - left + 1)) + left
    const pivot = nums[index]
    [nums[left], nums[index]] = [nums[index], nums[left]]
    // 上式交换在LC的TS会报错
    // const tmp = nums[left]
    // nums[left] = nums[index]
    // nums[index] = tmp

    let i = left + 1
    let j = right
    while (true) {
        while (i <= j && pivot > nums[i]) {
            i++
        }
        while (i <= j && pivot < nums[j]) {
            j--
        }
        if (i > j) {
            break
        }
        [nums[i], nums[j]] = [nums[j], nums[i]]
        i++
        j--
    }
    [nums[left], nums[j]] = [nums[j], nums[left]]
    return j
}

function quickSort(nums: number[], left: number, right: number) {
    let order = true
    for (let i = left; i < right; i++) {
        if (nums[i] > nums[i + 1]) {
            order = false
            break
        }
    }
    if (order) {
        return
    }
    const index = partition(nums, left, right)
    quickSort(nums, left, index - 1)
    quickSort(nums, index + 1, right)
}

function sortArray(nums: number[]): number[] {
    quickSort(nums, 0, nums.length - 1)
    return nums
};
```

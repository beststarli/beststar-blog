# [409.最长回文串](https://leetcode.cn/problems/longest-palindrome/description/)
给定一个包含大写字母和小写字母的字符串 `s` ，返回 *通过这些字母构造成的* ***最长的*** 回文串 的长度。

:::info **回文串**
**回文** 串是向前和向后读都相同的字符串。
:::

在构造过程中，请注意 **区分大小写** 。比如 `"Aa"` 不能当做一个回文字符串。

示例1：
- **输入**：s = "abccccdd"
- **输出**：7
- **解释**：我们可以构造的最长的回文串是"dccaccd", 它的长度是 7。

示例2：
- **输入**：s = "a"
- **输出**：1
- **解释**：可以构造的最长回文串是"a"，它的长度是 1。

## 解题思路
### 1.哈希map频率表
根据回文串的特性，只要一个字母出现的频率大于等于2，就可以用来组成回文串，那么考虑长度为奇数的回文串的情况，事实上这个多出的一个字符可以从出现频率大于2且为奇数次的字母处取，也可以从出现频率为1的字母处取。那么怎么能确保取到这样的字符呢？可以在建立偶数字符串时累加完结果后从频率表中删除所有出现频率为偶数的字母，如果处理完后哈希map的size不为零，剩下的就是出现频率为奇数的字母，取其中任意一个即可。

- 时间复杂度：O(n)，其中 n 是字符串 s 的长度。我们需要遍历字符串一次来统计每个字符的出现次数，然后再遍历哈希表来计算最长回文串的长度。
- 空间复杂度：O(1)。由于字符集的大小是固定的（只有大写字母和小写字母），哈希表的大小最多为 52，因此空间复杂度为 O(1)。

```ts
function longestPalindrome(s: string): number {
    const count = new Map()
    let ans = 0
    s.split('').forEach(char => {
        count.set(char, (count.get(char) ?? 0) + 1)
    })
    for (const [char, times] of count.entries()) {
        ans += Math.floor(times / 2) * 2
        if (times % 2 === 0) {
            count.delete(char)
        }
    }
    return count.size !== 0 ? ans + 1 : ans
};
```

### 2.哈希set
事实上根据回文串中两个字符一对的特性，我们可以用一个哈希set来存储出现过的字符，如果当前字符已经在set中出现过了，那么就说明我们可以组成一对回文串，累加结果并从set中删除该字符；如果当前字符没有在set中出现过，那么就将其加入set中。最后如果set中还有剩余的字符，那么说明我们可以从这些字符中取一个作为回文串的中心字符。

- 时间复杂度：O(n)，其中 n 是字符串 s 的长度。我们需要遍历字符串一次来统计每个字符的出现次数。
- 空间复杂度：O(1)。由于字符集的大小是固定的（只有大写字母和小写字母），哈希set的大小最多为 52，因此空间复杂度为 O(1)。

```ts
function longestPalindrome(s: string): number {
    const hasSet = new Set()
    let ans = 0
    for (const char of s) {
        if (hasSet.has(char)) {
            ans += 2
            hasSet.delete(char)
        } else {
            hasSet.add(char)
        }
    }
    return hasSet.size !== 0 ? ans + 1 : ans
};
```

# [718.最长重复子数组](https://leetcode.cn/problems/maximum-length-of-repeated-subarray/description/)   
给两个整数数组 `nums1` 和 `nums2`，返回 *两个数组中 **公共的** 、长度最长的子数组的长度* 。

示例1：
- **输入**：nums1 = [1,2,3,2,1], nums2 = [3,2,1,4,7]
- **输出**：3
- **解释**：长度最长的公共子数组是 [3,2,1] 。

示例2：
- **输入**：nums1 = [0,0,0,0,0], nums2 = [0,0,0,0,0]
- **输出**：5

## 解题思路
### 1.动态规划
与[1143.最长公共子序列](./1143.最长公共子序列.md)相比，这题的判断条件要简单一些，子数组要求必须是连续的，所以只比较两个遍历指针所指数字是否相同即可，相同就更新状态转移方程，并更新最大值。否则不操作。

- 时间复杂度：O(m*n)，其中 m 和 n 分别是数组 nums1 和 nums2 的长度。我们需要遍历整个二维数组一次。
- 空间复杂度：O(m*n)，其中 m 和 n 分别是数组 nums1 和 nums2 的长度。我们需要创建一个大小为 `m*n` 的二维数组来存储状态。

```ts
function findLength(nums1: number[], nums2: number[]): number {
    const m = nums1.length
    const n = nums2.length
    const memo = Array.from({ length: m + 1 }, () => Array(n + 1).fill(0))
    let ans = 0
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            if (nums1[i] === nums2[j]) {
                memo[i + 1][j + 1] = memo[i][j] + 1
                ans = Math.max(ans, memo[i + 1][j + 1])
            } 
        }
    }
    return ans
};
```

### 2.空间优化
用一维数组代替二维数组，注意更新顺序，必须从后往前更新。pre表示上一次的状态即左上角，temp表示当前的状态。

- 时间复杂度：O(m*n)，其中 m 和 n 分别是数组 nums1 和 nums2 的长度。我们需要遍历整个二维数组一次。
- 空间复杂度：O(n)，其中 n 是数组 nums2 的长度。我们只需要创建一个大小为 `n` 的一维数组来存储状态。

```ts
function findLength(nums1: number[], nums2: number[]): number {
    const n = nums2.length
    let ans = 0
    const memo = Array(n + 1).fill(0)
    for (const num of nums1) {
        let pre = 0
        for (let j = 0; j < n; j++) {
            const temp = memo[j + 1]
            if (num === nums2[j]) {
                memo[j + 1] = pre + 1
                ans = Math.max(ans, memo[j + 1])
            } else {
                memo[j + 1] = 0
            }
            pre = temp
        }
    }
    return ans
};
```

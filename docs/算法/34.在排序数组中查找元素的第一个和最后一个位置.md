# 34.在排序数组中查找元素的第一个和最后一个位置
给你一个按照非递减顺序排列的整数数组`nums`，和一个目标值`target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值`target`，返回`[-1, -1]`。

你必须设计并实现时间复杂度为`O(log n)`的算法解决此问题。

示例1：
- **输入**：nums = [5,7,7,8,8,10], target = 8
- **输出**：[3,4]

示例2：
- **输入**：nums = [5,7,7,8,8,10], target = 6
- **输出**：[-1,-1]

示例3：
- **输入**：nums = [], target = 0
- **输出**：[-1,-1]

## 解题思路
### 1.回家等通知法
对于这个非递减数组，直接使用indexOf()和lastIndexOf()方法找到目标元素的起始和结束位置索引即可。

:::danger **回家等通知！**
这种解法的时间复杂度可以达到O(n)，但这不符合这道算法题的出题要求和目的！
:::

时间复杂度：O(n)。

```ts
function searchRange(nums: number[], target: number): number[] {
    if (nums.indexOf(target) === -1) {
        return [-1, -1]
    }

    let start = nums.indexOf(target)
    let end = nums.lastIndexOf(target)

    return [start, end]
};
```

### 2.两次二分查找
我们可以使用两次二分查找来分别找到目标元素的起始位置和结束位置。第一次二分查找用于找到目标元素的起始位置，第二次二分查找用于找到目标元素后一个元素的起始位置，对于这个后一个元素的起始位置索引减一就是目标元素的结束位置索引。

时间复杂度：O(log n)。

```ts
function searchRange(nums: number[], target: number): number[] {
    const startIndex = lowerBound(nums, target)
    if (startIndex === nums.length || nums[startIndex] !== target) {
        return [-1, -1]
    }
    const endIndex = lowerBound(nums, target + 1) - 1
    return [startIndex, endIndex]
};

function lowerBound(nums, target) {
    let left = 0
    let right = nums.length - 1

    while (left <= right) {
        const mid = Math.floor((left + right) / 2)
        if (nums[mid] >= target) {
            right = mid - 1
        } else {
            left = mid + 1
        }
    }

    return left
}
```

# [470.用Rand7()实现Rand10()](https://leetcode.cn/problems/implement-rand10-using-rand7/description/)
给定方法 `rand7` 可生成 `[1,7]` 范围内的均匀随机整数，试写一个方法 `rand10` 生成 `[1,10]` 范围内的均匀随机整数。

你只能调用 `rand7()` 且不能调用其他方法。请不要使用系统的 `Math.random()` 方法。

每个测试用例将有一个内部参数 `n`，即你实现的函数 `rand10()` 在测试时将被调用的次数。请注意，这不是传递给 `rand10()` 的参数。

示例1：
- **输入**：1
- **输出**：[2]

示例2：
- **输入**：2
- **输出**：[2,8]

示例3：
- **输入**：3
- **输出**：[2,8,10]

**进阶**：
- `rand7()`调用次数的 [期望值](https://en.wikipedia.org/wiki/Expected_value) 是多少 ?
- 你能否尽量少调用 `rand7()` ?

## 解题思路
### 1.进制转换
注意题干说明rand7()和rand10()都是生成闭区间内的整数，那么对于有限的仅可使用rand7()的条件下，我们可以将rand7()生成的[1,7]范围内的整数看作是一个7进制数的个位数，若我们调用两次rand7()，则可以得到一个两位的7进制数，范围为[1,49]，然后我们只需要将[1,40]范围内的整数映射到[1,10]即可。这个转换公式就是：
```ts
num = (rand7() - 1) * 7 + rand7() // 7进制转十进制得到的[1, 49]范围内的整数
```
注意到41-49只有9个数，无法映射到[1,10]，所以我们需要舍弃掉这9个数，重新生成一个[1,49]范围内的整数，直到生成的整数在[1,40]范围内为止。所以对于40以内的数，我们可以直接映射到[1,10]，通过对10取余数再加1即可得到[1,10]范围内的整数。

- 时间复杂度：O(1)，因为舍弃的概率为9/49，所以期望调用rand7()的次数为49/40，仍然是常数级别。
- 空间复杂度：O(1)
- rand7()调用期望值：2 * 49/40 = 2.45

```ts
/**
 * The rand7() API is already defined for you.
 * function rand7(): number {}
 * @return a random integer in the range 1 to 7
 */

function rand10(): number {
    while (true) {
        const num = (rand7() - 1) * 7 + rand7() // 7进制转十进制得到的[1, 49]范围内的整数
        if (num <= 40) {
            return num % 10 + 1
            // 为了更好的建立num与rand10值的映射
            // 可以写成
            // return (num - 1) % 10 + 1
        }
    }
};
```

### 2.优化期望值
由于解法一将41-49舍弃掉了，实际上我们可以将41-49映射到[1,9]范围内的整数，然后再调用rand7()生成一个[1,63]范围内的整数，舍弃掉61-63，映射到[1,60]范围内的整数，然后再映射到[1,10]范围内的整数。这样可以减少舍弃的概率，从而减少rand7()的调用次数。以此类推，我们可以继续优化期望值，对于41-49映射到[1,9]范围内的整数后，我们可以继续调用rand7()生成一个[1,63]范围内的整数，然后将61-63映射到[1,3]范围内的整数，再调用rand7()生成一个[1,21]范围内的整数，舍弃掉22-21，映射到[1,20]范围内的整数，然后再映射到[1,10]范围内的整数。这样可以进一步减少舍弃的概率，从而减少rand7()的调用次数。

- 时间复杂度：O(1)
- 空间复杂度：O(1)
- rand7()调用期望值：约 **2.19** 次
  ```
  E = 2(第一层) + (9/49)×1(第二层额外) + (9/49×3/63)×1(第三层额外) + (9/49×3/63×1/21)×E
  E = 329/150 ≈ 2.19
  ```

```ts
/**
 * The rand7() API is already defined for you.
 * function rand7(): number {}
 * @return a random integer in the range 1 to 7
 */

function rand10(): number {
    while (true) {
        let num = (rand7() - 1) * 7 + rand7() // 7进制转十进制得到的[1, 49]范围内的整数
        if (num <= 40) {
            return num % 10 + 1
            // 为了更好的建立num与rand10值的映射
            // 可以写成
            // return (num - 1) % 10 + 1
        }

        // 处理[41, 49]
        num = (num - 40 - 1) * rand7() + rand7()
        if (num <= 60) {
            return num % 10 + 1
        }

        // 处理[61, 63]
        num = (num - 60 - 1) * rand7() + rand7()
        if (num <= 20) {
            return num % 10 + 1
        }

        // 处理[1]
        return rand7()
    }
};
```

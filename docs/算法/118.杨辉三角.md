# 118.杨辉三角
给定一个非负整数`numRows`，生成「杨辉三角」的前`numRows`行。

在「**杨辉三角**」中，每个数是它左上方和右上方的数的和。

![118-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/118-1.gif)
示例1：
- **输入**：numRows = 5
- **输出**：[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]

示例2：
- **输入**：numRows = 1
- **输出**：[[1]]

## 解题思路
### 1.动态规划
我们可以使用动态规划来解决这个问题。首先创建一个二维数组`ans`，其中`ans[i]`表示第`i`行的元素。对于每一行，我们首先将第一个元素和最后一个元素设置为`1`，然后对于每个元素`ans[i][j]`，可以发现它等于`ans[i-1][j-1] + ans[i-1][j]`。

时间复杂度：O(numRows^2)。

```ts
function generate(numRows: number): number[][] {
    const ans = new Array(numRows)

    for (let i = 0; i < numRows; i++) {
        ans[i] = new Array(i + 1)
        ans[i][0] = ans[i][i] = 1
        for (let j = 1; j < i; j++) {
            ans[i][j] = ans[i - 1][j - 1] + ans[i - 1][j]
        }
    }

    return ans
};
```

# 226.翻转二叉树
给你一棵二叉树的根节点`root`，翻转这棵二叉树，并返回其根节点。

示例1：
![226-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/226-1.jpg)
- **输入**：root = [4,2,7,1,3,6,9]
- **输出**：[4,7,2,9,6,3,1]

示例2：
![226-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/226-2.jpg)
- **输入**：root = [2,1,3]
- **输出**：[2,3,1]

示例3：
- **输入**：root = []
- **输出**：[]

## 解题思路
### 1.递归——自底向上
还是递归的思路，将问题转变为最小单元，也就是一个根节点左子树只有一个叶子节点，右子树也只有一个叶子节点的情况，此时翻转后就是交换左右子树的位置。对于更大的树，我们可以先递归地翻转左右子树，再交换左右子树的位置，这样就完成了整棵树的翻转。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function invertTree(root: TreeNode | null): TreeNode | null {
    if (root === null) {
        return null
    }

    const left = invertTree(root.left)
    const right = invertTree(root.right)
    root.left = right
    root.right = left
    
    return root
};
```

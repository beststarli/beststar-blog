# 112.路径总和
给你二叉树的根节点 `root` 和一个表示目标和的整数 `targetSum` 。判断该树中是否存在 **根节点到叶子节点** 的路径，这条路径上所有节点值相加等于目标和 `targetSum` 。如果存在，返回 `true` ；否则，返回 `false` 。

**叶子节点** 是指没有子节点的节点。

示例1：
![112-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/112-1.png)
- **输入**：root = [5,4,8,11,null,13,4,7,2,null,null,null,1], targetSum = 22
- **输出**：true
- **解释**：等于目标和的根节点到叶节点路径如上图所示。

示例2：
![112-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/112-2.png)
- **输入**：root = [1,2,3], targetSum = 5
- **输出**：false
- **解释**：树中存在两条根节点到叶子节点的路径：
```txt
(1 --> 2): 和为 3
(1 --> 3): 和为 4
不存在 sum = 5 的根节点到叶子节点的路径。
```

示例3：
- **输入**：root = [], targetSum = 0
- **输出**：false
- **解释**：由于树是空的，所以不存在根节点到叶子节点的路径。

## 解题思路
### 1.递归
在从根节点往下递归的过程无非就是左子树和右子树两条路可选，每次往下走一层都要将目标值减去当前节点值以更新目标值,递归的边界条件是：
- 当前到达的节点为null，说明没有路径了，返回false。
- 当前节点为叶子节点，说明已经到达路径的末尾了，如果此时目标值为0，说明存在一条路径的和等于目标值，返回true。

最终返回的结果应是对左右子树分别进行递归后的或运算结果，因为向下的路径不折返，所以只要存在一条路径满足条件即可。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function hasPathSum(root: TreeNode | null, targetSum: number): boolean {
    if (root === null) {
        return false
    }
    targetSum -= root.val
    if (root.left === null && root.right === null) {
        return targetSum === 0
    }
    return hasPathSum(root.left, targetSum) || hasPathSum(root.right, targetSum)
};
```

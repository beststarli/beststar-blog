# 73.矩阵置零
给定一个`m x n`的矩阵，如果一个元素为0，则将其所在行和列的所有元素都设为0。请使用 **原地** 算法。
:::info **原地算法**
原地算法（In-place Algorithm） 是指在算法执行过程中，额外使用的空间复杂度为 O(1) 的算法，也就是说，除了输入本身占用的空间之外，几乎不使用额外的空间（常数级别的辅助变量是允许的）。

“原地” = 大部分工作直接在输入数组/数据结构上完成，不额外开大块空间。
:::

示例1：
![73-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/73-1.jpg)
- **输入**：matrix = [[1,1,1],[1,0,1],[1,1,1]]
- **输出**：[[1,0,1],[0,0,0],[1,0,1]]

示例2：
![73-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/73-2.jpg)
- **输入**：matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
- **输出**：[[0,0,0,0],[0,4,5,0],[0,3,1,0]]

**进阶**：
- 一个直观的解决方案是使用`O(mn)`的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用`O(m + n)`的额外空间，但这仍然不是最好的解决方案。
- 你能想出一个仅使用常量空间的解决方案吗？

## 解题思路
### 1.暴力循环
题干为了把0所在行列所有元素置0，那很直接想到对整个矩阵遍历并记录为0元素的行列索引，使用额外的数组记录存在0的行和列，然后对矩阵的该行该列分别更新元素为0即可。

:::warning **注意**
这样做不满足原地算法的要求
:::

时间复杂度：O(mn)，其中 m 和 n 分别是 matrix 的行数和列数。

空间复杂度：O(m+n)。

```ts
function setZeroes(matrix: number[][]): void {
    const height = matrix.length
    const width = matrix[0].length

    const rowHasZero = Array(height).fill(false)
    const colHasZero = Array(width).fill(false)

    const zeroItem = new Map()
    for (let i = 0; i < height; i++) {
        for (let j = 0; j < width; j++) {
            if (matrix[i][j] === 0) {
                rowHasZero[i] = colHasZero[j] = true
            }
        }
    }

    for (let i = 0; i < height; i++) {
        for (let j = 0; j < width; j++) {
            if (rowHasZero[i] || colHasZero[j]) {
                matrix[i][j] = 0
            }
        }
    }
};
```

### 2.优化

时间复杂度：O(mn)，其中 m 和 n 分别是 matrix 的行数和列数。

空间复杂度：O(1)。

```ts
/**
 Do not return anything, modify matrix in-place instead.
 */
function setZeroes(matrix: number[][]): void {
    const m = matrix.length, n = matrix[0].length;
    const firstRowHasZero = matrix[0].includes(0);

    for (let i = 1; i < m; i++) {
        for (let j = 0; j < n; j++) {
            if (matrix[i][j] === 0) {
                matrix[i][0] = matrix[0][j] = 0;
            }
        }
    }

    for (let i = 1; i < m; i++) {
        // 倒着遍历，避免提前把 matrix[i][0] 改成 0，误认为这一行要全部变成 0
        for (let j = n - 1; j >= 0; j--) {
            if (matrix[i][0] === 0 || matrix[0][j] === 0) {
                matrix[i][j] = 0;
            }
        }
    }

    if (firstRowHasZero) {
        matrix[0].fill(0);
    }
};
```

# [491.非递减子序列](https://leetcode.cn/problems/non-decreasing-subsequences/description/)
给你一个整数数组 `nums` ，找出并返回所有该数组中不同的递增子序列，递增子序列中 **至少有两个元素** 。你可以按 **任意顺序** 返回答案。

数组中可能含有重复元素，如出现两个整数相等，也可以视作递增序列的一种特殊情况。

示例1：
- **输入**：nums = [4,6,7,7]
- **输出**：[[4,6],[4,6,7],[4,6,7,7],[4,7],[4,7,7],[6,7],[6,7,7],[7,7]]

示例2：
- **输入**：nums = [4,4,3,2,1]
- **输出**：[[4,4]]

## 解题思路
### 1.回溯——选哪个以及能否选
与[90.子集II](/docs/算法/90.子集II)主体类似，需要在递归时判断当前元素是否大于等于路径中最后一个元素，如果是则可以选择该元素，否则跳过该元素。为了避免重复子序列的出现，我们需要在每一层递归中使用一个哈希集合来记录已经使用过的元素，如果当前元素已经在集合中出现过，则跳过该元素。注意要求子序列长度至少为2，所以在递归函数中需要判断当前路径的长度是否大于等于2，如果是则将当前路径加入结果中。

- 时间复杂度：O(n * 2^n)，其中n是输入数组的长度。生成子序列的时间复杂度是O(2^n)，每个子序列需要O(n)的时间来构建。
- 空间复杂度：O(n)。递归调用栈的深度为n，最坏情况下需要O(n)的空间来存储递归调用栈。

```ts
function findSubsequences(nums: number[]): number[][] {
    const ans = []
    const path = []
    function dfs(index) {
        if (path.length >= 2) {
            ans.push([...path])
        }
        const used = new Set()
        for (let i = index; i < nums.length; i++) {
            if (path.length > 0 && nums[i] < path[path.length - 1]) {
                continue
            }
            if (used.has(nums[i])) {
                continue
            }
            path.push(nums[i])
            used.add(nums[i])
            dfs(i + 1)
            path.pop()
        }
    }
    dfs(0)
    return ans
};
```

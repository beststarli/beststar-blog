# [135.分发糖果](https://leetcode.cn/problems/candy/description/)
`n` 个孩子站成一排。给你一个整数数组 `ratings` 表示每个孩子的评分。

你需要按照以下要求，给这些孩子分发糖果：
- 每个孩子至少分配到 `1` 个糖果。
- 相邻两个孩子中，评分更高的那个会获得更多的糖果。

请你给每个孩子分发糖果，计算并返回需要准备的 **最少糖果数目** 。

示例1：
- **输入**：ratings = [1,0,2]
- **输出**：5
- **解释**：你可以分别给第一个、第二个、第三个孩子分发 2、1、2 颗糖果。

示例2：
- **输入**：ratings = [1,2,2]
- **输出**：4
- **解释**：你可以分别给第一个、第二个、第三个孩子分发 1、2、1 颗糖果。第三个孩子只得到 1 颗糖果，这满足题面中的两个条件。

## 解题思路
### 1.寻找上下坡
这个题其实有点像[162.寻找峰值](/docs/算法/162.寻找峰值.md)的意思，我们需要将评分数组分割成一段段的上下坡，寻找每段坡的长度以及坡峰坡谷。

因为题目要求每个孩子至少分配到 1 个糖果，下面分情况讨论：
- 对于上坡段，最左边的孩子旁边没有比他评分更低的孩子，所以再给他0个糖果就行，下一个孩子再比他多给1个就行。
- 对于下坡段，最右边的孩子旁边没有比他评分更低的孩子，所以再给他0个糖果就行，上一个孩子再比他多给1个就行。
- 最后确定坡峰的孩子的糖果数为坡峰左边的孩子和坡峰右边的孩子中糖果数较多的那个。

如果把上坡段+坡峰+下坡段看作一座山，那么对于多座山就需要分组循环，外层循环负责遍历组的起点与数量统计，内层循环负责遍历组的终点。根据等差数列求和公式，外循环计数更新为`ans += (increase * (increase - 1) + decrease * (decrease - 1)) / 2 + Math.max(increase, decrease)`。

- 时间复杂度：O(n)，其中 n 是数组 ratings 的长度。我们只需要遍历一次数组即可。
- 空间复杂度：O(1)，我们只需要常数级别的额外空间来存储计数器和结果。

```ts
function candy(ratings: number[]): number {
    const n = ratings.length
    let ans = n
    for (let i = 0; i < n; i++) {
        const start = (i > 0) && (ratings[i - 1] < ratings[i]) ? i - 1 : i
        while (i + 1 < n && ratings[i] < ratings[i + 1]) {
            i++
        }
        const top = i
        while (i + 1 < n && ratings[i] > ratings[i + 1]) {
            i++
        }
        const increase = top - start
        const decrease = i - top
        ans += (increase * (increase - 1) + decrease * (decrease - 1)) / 2 + Math.max(increase, decrease)
    }
    return ans
};
```

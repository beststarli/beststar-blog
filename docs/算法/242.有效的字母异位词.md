# [242.有效的字母异位词](https://leetcode.cn/problems/valid-anagram/description/)   
给定两个字符串 `s` 和 `t` ，编写一个函数来判断 `t` 是否是 `s` 的 字母异位词。

:::info **字母异位词**
字母异位词是通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次。
:::

示例1：
- **输入**：s = "anagram", t = "nagaram"
- **输出**：true

示例2：
- **输入**：s = "rat", t = "car"
- **输出**：false

**进阶**: 如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

## 解题思路
### 1.频率哈希map——两次遍历
用一个map来统计字符串s中每个字符出现的次数，然后再遍历字符串t，检查每个字符是否在map中出现过，并且次数是否大于0，如果是，则将该字符的次数减1，否则返回false。最后检查map中是否还有字符的次数大于0，如果有，则返回false，否则返回true。

- 时间复杂度：O(n)，其中n是字符串s的长度。我们需要遍历字符串s和t各一次。
- 空间复杂度：O(1)，因为字符集是固定的，最多只有26个小写字母，所以map的大小是常数级别的。

```ts
function isAnagram(s: string, t: string): boolean {
    if (s.length !== t.length) {
        return false
    }
    const map = new Map()
    for (const char of s) {
        map.set(char, (map.get(char) ?? 0) + 1)
    }
    for (const char of t) {
        if (map.has(char) && map.get(char) > 0) {
            map.set(char, map.get(char) - 1)
        } 
    }
    
    return Math.max(...map.values()) > 0 ? false : true
};
```

### 2.频率数组——两次遍历
因为只有26个小写字母，所以可以用一个长度为26的数组来统计每个字符出现的次数。遍历字符串s时，将对应字符的计数加1，遍历字符串t时，将对应字符的计数减1。最后检查数组中是否有不为0的元素，这里用到的是数组的every()方法，如果有，则返回false，否则返回true。

- 时间复杂度：O(n)，其中n是字符串s的长度。我们需要遍历字符串s和t各一次。
- 空间复杂度：O(1)，因为字符集是固定的，最多只有26个小写字母，所以数组的大小是常数级别的。

```js
function isAnagram(s: string, t: string): boolean {
    if (s.length !== t.length) {
        return false
    }
    const count = Array(26).fill(0)
    for (const char of s) {
        count[char.charCodeAt(0) - 'a'.charCodeAt(0)]++
    }
    for (const char of t) {
        count[char.charCodeAt(0) - 'a'.charCodeAt(0)]--
    }

    return count.every(times => times === 0)
};
```

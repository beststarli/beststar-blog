# [974.和可被K整除的子数组](https://leetcode.cn/problems/subarray-sums-divisible-by-k/description/)
给定一个整数数组 `nums` 和一个整数 `k`，返回其中元素之和可被 `k` 整除的非空 **子数组** 的数目。

**子数组** 是数组中 **连续** 的部分。

示例1：
- **输入**：nums = [4,5,0,-2,-3,1], k = 5
- **输出**：7
- **解释**：
```txt
有 7 个子数组满足其元素之和可被 k = 5 整除：
[4, 5, 0, -2, -3, 1], [5], [5, 0], [5, 0, -2, -3], [0], [0, -2, -3], [-2, -3]
```

示例2：
- **输入**：nums = [5], k = 9
- **输出**：0

## 解题思路
### 1.前缀和 + 哈希表
本题与[560.和为K的子数组](/docs/算法/560.和为k的子数组.md)思考基本类似，用哈希map来维护前缀和的余数出现的次数，假设当前前缀和为`s`，`s % k = r`，则我们需要找出之前有多少个前缀和的余数也是`r`，这样就可以组成一个子数组，其和可被`k`整除。需要确保余数为非负数，因此在计算余数时需要加上`k`再取模。所以s的更新公式为`s = (s + num % k + k) % k`。

- 时间复杂度：O(n)，其中 n 是数组 nums 的长度。
- 空间复杂度：O(k)，其中 k 是整数 k 的大小。

```ts
function subarraysDivByK(nums: number[], k: number): number {
    const count = new Map()
    let ans = 0
    let s = 0
    for (const num of nums) {
        count.set(s, (count.get(s) ?? 0) + 1)
        s = ((s + num) % k + k) % k   // 保证余数为非负数
        ans += count.get(s) ?? 0
    }
    return ans
};
```

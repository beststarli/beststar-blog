# 25.K个一组翻转链表
给你链表的头节点`head`，每`k`个节点一组进行翻转，请你返回修改后的链表。

`k`是一个正整数，它的值小于或等于链表的长度。如果节点总数不是`k`的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

示例1：
![25-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/25-1.jpg)
- **输入**：head = [1,2,3,4,5], k = 2
- **输出**：[2,1,4,3,5]

示例2：
![25-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/25-2.jpg)
- **输入**：head = [1,2,3,4,5], k = 3
- **输出**：[3,2,1,4,5]

**进阶**:
- 你可以设计一个只用`O(1)`额外内存空间的算法解决此问题吗？

## 解题思路
### 1.计数区域反转
这道题的大体思路与[92.反转链表II](/docs/算法/92.反转链表II)类似，都是需要反转链表的一部分。不同之处在于，我们需要反转链表的多个部分，每个部分的长度为`k`。在每次反转前应先对剩余结点数进行判断，不足`k`个结点时直接返回结果。需要注意的点是每次反转前需更新前一个部分的尾结点和当前部分的头结点的连接关系。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseKGroup(head: ListNode | null, k: number): ListNode | null {
    let n = 0
    for (let cur = head; cur; cur = cur.next) {
        n++
    }

    const dummy = new ListNode(0, head)
    let p0 = dummy
    let pre = null
    let cur = head

    for (; n>=k; n -= k) {
        for (let i = 0; i < k; i++) {
            const nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt
        }

        const nxt = p0.next
        p0.next.next = cur
        p0.next = pre
        p0 = nxt
    }
    return dummy.next
};
```

## 二刷
### 1.计数反转——辅助反转函数
还是每k个一组反转链表，借助[92.反转链表II](/docs/算法/92.反转链表II)辅助函数来实现链表反转，需要注意重置新的要反转的链表头以及连接反转后的前部分链表。

- 时间复杂度：O(n)
- 空间复杂度：O(1)
```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */
function reverseListNode(node: ListNode | null): ListNode | null {
    let pre = null
    let curr = node
    while (curr !== null) {
        const temp = curr.next
        curr.next = pre
        pre = curr
        curr = temp
    }
    return pre
}

function reverseKGroup(head: ListNode | null, k: number): ListNode | null {
    let count = 0
    for (let cur = head; cur !== null; cur = cur.next) {
        count++
    }
    const dummy = new ListNode(0, head)
    let p0 = dummy
    while (count >= k) {
        let groupHead = p0.next
        let groupTail = groupHead
        for (let i = 1; i < k; i++) {
            groupTail = groupTail.next
        }

        let nextGroup = groupTail.next
        groupTail.next = null
        
        let reverseHead = reverseListNode(groupHead)

        p0.next = reverseHead
        groupHead.next = nextGroup

        p0 = groupHead
        count -= k
    }
    return dummy.next
};
```

## 总结
递归题目的解法无论是使用迭代还是递归求解，关键还是要把题目最小化，以小见大，找到递归关系。

# 144.二叉树的前序遍历
给你一个二元数组 `nums` ，和一个整数 `goal` ，请你统计并返回有多少个和为 `goal` 的 非空 子数组。

子数组 是数组的一段连续部分。

示例1：
- **输入**：root = [1,null,2,3]
- **输出**：[1,2,3]

## 解题思路
### 1.递归
二叉树前序遍历的顺序是先根节点，再左子树然后再右子树，那么在递归方城内就是先将当前节点的值推入结果数组，然后递归左子树再递归右子树，边界条件是节点为空则返回。

时间复杂度：O(n)，其中n是二叉树的结点数。

空间复杂度：O(n)，其中n是二叉树的结点数，最坏情况下二叉树退化成链表，递归调用栈的深度为n。平均情况下，二叉树的高度为log(n)，递归调用栈的深度为log(n)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function preorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    function dfs(node: TreeNode | null, result: number[]) {
        if (node === null) {
            return
        }

        result.push(node.val)
        dfs(node.left, result)
        dfs(node.right, result)
    }
    dfs(root, ans)
    return ans
};
```
### 2.Morris遍历



```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function preorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    let cur = root
    while (cur !== null) {
        if (cur.left === null) {
            // 没有左子树，直接访问根
            ans.push(cur.val)
            cur = cur.right
        } else {
            // 找到左子树的最右节点
            let prev = cur.left
            while (prev.right !== null && prev.right !== cur) {
                prev = prev.right
            }

            if (prev.right === null) {
                // 第一次到达，建立线索
                prev.right = cur
                ans.push(cur.val)   // 前序：在这里访问根
                cur = cur.left
            } else {
                // 第二次到达，拆线索
                prev.right = null
                cur = cur.right
            }
        }
    }
    return ans
};
```

### 3.迭代——栈
时间复杂度：O(n)。

空间复杂度：O(n)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function preorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    const stack = []

    // 根节点先入栈
    if (root) {
        stack.push(root)
    }
    while (stack.length > 0) {
        const cur = stack.pop() // 弹出当前节点
        ans.push(cur.val)       // 前序先访问根

        // 由于栈先进后出，所以先推右子树再推左子树
        if (cur.right) {
            stack.push(cur.right)
        }
        if (cur.left) {
            stack.push(cur.left)
        }
    }
    return ans
};
```

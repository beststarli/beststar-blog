# 1143.最长公共子序列
给定两个字符串 `text1` 和 `text2`，返回这两个字符串的最长 **公共子序列** 的长度。如果不存在 **公共子序列**，返回 0 。

一个字符串的 **子序列** 是指这样一个新的字符串：它是由原字符串在不改变字符的相对顺序的情况下删除某些字符（也可以不删除任何字符）后组成的新字符串。

例如，`"ace"` 是 `"abcde"` 的子序列，但 `"aec"` 不是 `"abcde"` 的子序列。
两个字符串的 **公共子序列** 是这两个字符串所共同拥有的子序列。

示例1：
- **输入**：text1 = "abcde", text2 = "ace" 
- **输出**：3
- **解释**：最长公共子序列是 "ace" ，它的长度为 3 。

示例2：
- **输入**：text1 = "abc", text2 = "abc"
- **输出**：3
- **解释**：最长公共子序列是 "abc" ，它的长度为 3 。

示例3：
- **输入**：text1 = "abc", text2 = "def"
- **输出**：0
- **解释**：两个字符串没有公共子序列，返回 0 。

## 解题思路

### 1.递归加记忆化
为了保证两个字符串公共子序列的顺序，将两个字符串分别转换成字符数组`s`和`t`。定义函数`dfs(i, j)`表示`s[0...i]`和`t[0...j]`的最长公共子序列的长度。
- 如果`s[i]`和`t[j]`相同，那么最长公共子序列的长度为`s[0...i-1]`和`t[0...j-1]`的最长公共子序列的长度加1，即`dfs(i, j) = dfs(i - 1, j - 1) + 1`。
- 如果`s[i]`和`t[j]`不同，那么最长公共子序列的长度为`s[0...i-1]`和`t[0...j]`的最长公共子序列的长度与`s[0...i]`和`t[0...j-1]`的最长公共子序列的长度的较大值，即`dfs(i, j) = Math.max(dfs(i - 1, j), dfs(i, j - 1))`。

递归边界为`i < 0`或`j < 0`，此时最长公共子序列的长度为0。为了避免重复计算，使用二维数组memo进行记忆化，memo[i][j]表示`s[0...i]`和`t[0...j]`的最长公共子序列的长度。递归入口是`dfs(length1 - 1, length2 - 1)`，其中length1和length2分别是`s`和`t`的长度。

时间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 text1 和 text2 的长度。

```ts
function longestCommonSubsequence(text1: string, text2: string): number {
    const s = text1.split('')
    const t = text2.split('')
    const length1 = s.length
    const length2 = t.length
    const memo = Array.from({ length: length1 }, () => Array(length2).fill(-1))

    function dfs(i, j) {
        if (i < 0 || j < 0) {
            return 0
        }
        if (memo[i][j] !== -1) {
            return memo[i][j]
        }
        if (s[i] === t[j]) {
            memo[i][j] = dfs(i - 1, j - 1) + 1
        } else {
            memo[i][j] = Math.max(dfs(i - 1, j), dfs(i, j - 1))
        }

        return memo[i][j]
    }

    return dfs(length1 - 1, length2 - 1)
};
```

### 2.递推
定义二维数组`memo`，其中`memo[i][j]`表示`s[0...i]`和`t[0...j]`的最长公共子序列的长度。对`s[i]`和`t[j]`分类讨论：
- 如果`s[i]`和`t[j]`相同，那么`memo[i][j] = memo[i - 1][j - 1] + 1`。
- 如果`s[i]`和`t[j]`不同，那么`memo[i][j] = Math.max(memo[i - 1][j], memo[i][j - 1])`。

之所以建立的二维数组的长度为`length1 + 1`和`length2 + 1`，是为了处理递推边界。`memo[0][j]`表示`s[0...-1]`和`t[0...j]`的最长公共子序列的长度，`memo[i][0]`表示`s[0...i]`和`t[0...-1]`的最长公共子序列的长度，这两种情况最长公共子序列的长度都为`0`。

```ts
function longestCommonSubsequence(text1: string, text2: string): number {
    const s = text1.split('')
    const t = text2.split('')
    const length1 = s.length
    const length2 = t.length
    const memo = Array.from({ length: length1 + 1 }, () => Array(length2 + 1).fill(0))

    for (let i = 1; i <= length1; i++) {
        for (let j = 1; j <= length2; j++) {
            memo[i][j] = (s[i - 1] === t[j - 1])
                ? memo[i - 1][j - 1] + 1
                : Math.max(memo[i - 1][j], memo[i][j - 1])
        }
    }
    return memo[length1][length2]
};
```

### 3.空间优化
由于`memo[i][j]`的计算只依赖于`memo[i - 1][j - 1]`、`memo[i - 1][j]`和`memo[i][j - 1]`，所以可以用两个一维数组来维护当前行和上一行的最长公共子序列的长度，空间复杂度降为$O(n)$，其中 n 是字符串 text2 的长度。

```ts
function longestCommonSubsequence(text1: string, text2: string): number {
    const s = text1.split('')
    const t = text2.split('')
    const length1 = s.length
    const length2 = t.length
    const memo = Array(length2 + 1).fill(0)

    for (const char of text1) {
        let pre = 0
        for (let j = 0; j < length2; j++) {
            const temp = memo[j + 1]
            memo[j + 1] = (char === t[j])
                ? pre + 1
                : Math.max(memo[j + 1], memo[j])
            pre = temp
        }
    }

    return memo[length2]
};
```

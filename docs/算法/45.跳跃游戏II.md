# 45.跳跃游戏II
给定一个长度为`n`的 **0索引**整数数组`nums`。初始位置在下标`0`。

每个元素`nums[i]`表示从索引`i`向后跳转的最大长度。换句话说，如果你在索引`i`处，你可以跳转到任意`(i + j)`处：

- `0 <= j <= nums[i]`且
- `i + j < n`

返回到达`n - 1`的最小跳跃次数。测试用例保证可以到达`n - 1`。

示例1：
- **输入**：nums = [2,3,1,1,4]
- **输出**：2
- **解释**：
```txt
跳到最后一个位置的最小跳跃数是 2。
从下标为 0 跳到下标为 1 的位置，跳 1 步，然后跳 3 步到达数组的最后一个位置。
```

示例2：
- **输入**：nums = [2,3,0,1,4]
- **输出**：2

## 解题思路
题目本意是如何用最少的跳跃次数到达数组的最后位置。

### 1.贪心
本题应该是默认会走到最后一个位置的，只是要找出到达的最少跳跃次数。第一步一定是`nums[0]`步跳跃，对于`i`位置来说，能到达的最远位置是`i + nums[i]`，用一个变量`nextRight`来记录`i + nums[i]`与先前`nextRight`的最大值，用变量`currRight`来记录当前已经建造的桥的右端点，遍历数组时，如果发现`i`位置已经到达了`currRight`，说明必须建桥了，此时将`currRight`更新为`nextRight`，并将跳跃次数加1。

时间复杂度：O(n)。

```ts
function jump(nums: number[]): number {
    let ans = 0
    let currRight = 0   // 已建造的桥的右端点
    let nextRight = 0   // 下一座桥的右端点的最大值

    for (let i = 0; i < nums.length - 1; i++) {
        // 遍历的过程中，记录下一座桥的最远点
        nextRight = Math.max(nextRight, i + nums[i])
        // 无路可走，必须建桥
        if (i === currRight) {
            currRight = nextRight   // 建桥后，最远可以到达 next_right
            ans++
        }
    }
    return ans
};
```

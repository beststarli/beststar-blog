# 279.完全平方数
给你一个整数`n`，返回 *和为`n` 的完全平方数的最少数量* 。

**完全平方数** 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，`1`、`4`、`9`和`16`都是完全平方数，而`3`和`11`不是。

示例1：
- **输入**：n = 12
- **输出**：3
- **解释**：12 = 4 + 4 + 4

示例2：
- **输入**：n = 13
- **输出**：2
- **解释**：13 = 4 + 9

**提示**：1 ≤ n ≤ 10<sup>4</sup>

## 解题思路
### 1.递归与记忆化
各个平方数可以多次选择，最终和为`n`，求最少的平方数数量。由于每个数都可以选择多次，所以是一个[完全背包](https://www.bilibili.com/video/BV16Y411v7Y6/?vd_source=8449e39df4a29ece9841f1cd40a3a9ae)问题。

定义递归函数`dfs(i, j)`，表示在前`i`个平方数中选择，最终和为`j`的最少平方数数量。则有以下状态转移方程：
- 如果不选择第`i`个平方数，则有`dfs(i - 1, j)`
- 如果选择第`i`个平方数，则有`dfs(i, j - i * i)`

这两种情况取最小值，就有了`dfs(i, j) = min(dfs(i - 1, j), dfs(i, j - i * i) + 1)`。前提是`j >= i * i`，否则只能选择不选第`i`个平方数，即`dfs(i, j) = dfs(i - 1, j)`。

递归的边界是`dfs(0,0) = 0`，因为没有数可以选了，且要得到的数等于`0`，那么答案为`0`。如果 `j > 0`，那么 `dfs(0,j) = ∞`，这里用`∞`表示不合法的状态，从而保证上式中的`min`取到合法的状态。注意本题是一定有解的，因为`1`是完全平方数。


由于$i^2 \le n$，所以$i \le \lfloor\sqrt{n}\rfloor$，所以递归入口为$dfs(\lfloor\sqrt{n}\rfloor, n)$，也就是答案。

在计算`n = 7`的时候，如果选了`4`，会递归到`n = 3`的情况。这个例子意味着多个测试数据之间可以共享记忆化搜索的结果。因此，把记忆化搜索的`memo`数组声明为全局变量，这样可以在多个测试数据之间共享，从而减少计算量。

时间复杂度：$O(\sqrt{n} * n)$，其中`sqrt(n)`是平方数的数量，`n`是目标数。

```ts
// 写在外面，多个测试数据之间可以共享，减少计算量
const memo = Array.from({ length: 101 }, () => Array(10001).fill(-1))   // -1 表示没有计算过

function numSquares(n: number): number {
    function dfs(i, j) {
        if (i === 0) {
            return (j === 0) ? 0 : Infinity
        }
        // 之前计算过
        if (memo[i][j] !== -1) {
            return memo[i][j]
        }
        if (j < i * i) {
            memo[i][j] = dfs(i - 1, j)  // 只能不选
        } else {
            memo[i][j] = Math.min(dfs(i - 1, j), dfs(i, j - i * i) + 1)  // 不选 还是 选
        }
        return memo[i][j]
    }

    return dfs(Math.floor(Math.sqrt(n)), n)
};
```

### 2.递推
将[解法1](/docs/算法/完全平方数#1递归与记忆化)的dfs(i, j)改写为dp[i][j]，其中`i`表示前`i`个平方数，`j`表示目标数。则有以下状态转移方程：
- 当j < i * i时，则有`dp[i - 1][j]`
- 当j >= i * i时，则有`min(dp[i - 1][j], dp[i][j - i * i] + 1)`

由递归边界dfs(0,0)=0 和 dfs(0,j)=∞ (j>0)得到初始值dp[0][0] = 0, dp[0][j] = ∞ (j>0)

由递归入口$dfs(\lfloor\sqrt{n}\rfloor, n)$得到答案为$dp[\lfloor\sqrt{n}\rfloor][n]$。

时间复杂度：$O(\sqrt{n} * n)$，其中`sqrt(n)`是平方数的数量，`n`是目标数。

```ts
const memo = Array.from({ length: 101 }, () => Array(10001).fill(Infinity)) // 设为无穷大方便取小
memo[0][0] = 0
for (let i = 1; i * i <= 10000; i++) {
    for (let j = 0; j <= 10000; j++) {
        if (j < i * i) {
            memo[i][j] = memo[i - 1][j]; // 只能不选
        } else {
            memo[i][j] = Math.min(memo[i - 1][j], memo[i][j - i * i] + 1); // 不选 vs 选
        }
    }
}

function numSquares(n: number): number {
    return memo[Math.floor(Math.sqrt(n))][n]; // 也可以写 memo[100][n]
};
```

### 3.空间优化
由于`dp[i][j]`只和`dp[i - 1][j]`和`dp[i][j - i * i]`有关，所以可以将二维数组优化成一维数组`dp[j]`，其中`j`表示目标数。则有以下状态转移方程：
- 当j < i * i时，则有`dp[j] = dp[j]`，即不选第`i`个平方数
- 当j >= i * i时，则有`dp[j] = min(dp[j], dp[j - i * i] + 1)`，即不选第`i`个平方数

时间复杂度：$O(\sqrt{n} * n)$，其中`sqrt(n)`是平方数的数量，`n`是目标数。

空间复杂度：$O(n)$，其中`n`是目标数。

```ts
const memo = Array(10001).fill(Infinity)// 设为无穷大方便取小
memo[0] = 0
for (let i = 1; i * i <= 10000; i++) {
    for (let j = i * i; j <= 10000; j++) {
        memo[j] = Math.min(memo[j], memo[j - i * i] + 1); // 不选 vs 选
    }
}

function numSquares(n: number): number {
    return memo[n]
};
```

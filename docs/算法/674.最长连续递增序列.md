# [674.最长连续递增序列](https://leetcode.cn/problems/longest-continuous-increasing-subsequence/description/)
给定一个未经排序的整数数组，找到最长且 **连续递增** 的子序列，并返回该序列的长度。

连续递增的子序列 可以由两个下标 `l` 和 `r`（`l < r`）确定，如果对于每个 `l <= i < r`，都有 `nums[i] < nums[i + 1]` ，那么子序列 `[nums[l], nums[l + 1], ..., nums[r - 1], nums[r]]` 就是连续递增子序列。

示例1：
- **输入**：nums = [1,3,5,4,7]
- **输出**：3
- **解释**：最长连续递增序列是 [1,3,5], 长度为3。尽管 [1,3,5,7] 也是升序的子序列, 但它不是连续的，因为 5 和 7 在原数组里被 4 隔开。 

示例2：
- **输入**：nums = [2,2,2,2,2]
- **输出**：1
- **解释**：最长连续递增序列是 [2], 长度为1。

## 解题思路
### 1.序列起点指针
维护一个指针start指向递增序列的起点，遍历数组，如果当前数字小于等于前一个数字，则更新start指针为当前下标，计算当前递增序列的长度并更新最大长度。

- 时间复杂度：O(n)
- 空间复杂度：O(1)

```ts
function findLengthOfLCIS(nums: number[]): number {
    let start = 0
    let ans = 0
    for (let i = 0; i < nums.length; i++) {
        if (i > 0 && nums[i] <= nums[i - 1]) {
            start = i
        }
        ans = Math.max(i - start + 1, ans)
    }
    return ans
};
```

### 2.贪心
初始化当前段长度为1，从第二个元素开始遍历数组，如果当前数字小于等于前一个数字，则重置当前段长度为1，否则当前段长度加1，更新最大长度。

- 时间复杂度：O(n)
- 空间复杂度：O(1)

```ts
function findLengthOfLCIS(nums: number[]): number {
    let length = 1
    let ans = 1
    for (let i = 1; i < nums.length; i++) {
        if (nums[i] <= nums[i - 1]) {
            length = 1
        } else {
            length++
        }
        ans = Math.max(ans, length)
    }
    return ans
};
```

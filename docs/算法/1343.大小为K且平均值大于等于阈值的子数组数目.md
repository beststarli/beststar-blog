# [1343.大小为K且平均值大于等于阈值的子数组数目](https://leetcode.cn/problems/number-of-sub-arrays-of-size-k-and-average-greater-than-or-equal-to-threshold/description/)
给你一个整数数组 `arr` 和两个整数 `k` 和 `threshold` 。

请你返回长度为 `k` 且平均值大于等于 `threshold` 的子数组数目。

示例1：
- **输入**：arr = [2,2,2,2,5,5,5,8], k = 3, threshold = 4
- **输出**：3
- **解释**：
```txt
子数组 [2,5,5],[5,5,5] 和 [5,5,8] 的平均值分别为 4，5 和 6 。其他长度为 3 的子数组的平均值都小于 4 （threshold 的值)。
```

示例2：
- **输入**：arr = [11,13,17,23,29,31,7,5,2,3], k = 3, threshold = 5
- **输出**：6
- **解释**：
```txt
前 6 个长度为 3 的子数组平均值都大于 5 。注意平均值不是整数。
```

## 解题思路
### 1.固定滑动窗口
实际上本题就是建立一个长度固定为k的滑动窗口，计算窗口内的元素和平均值是否大于等于阈值threshold，如果大于等于则计数器加一。不需要一个额外的数组来维护窗口，只需要一个变量sum来维护窗口内的元素和即可。每次滑动窗口时，更新sum的值即可。

- 时间复杂度：O(n)，其中 n 是数组的长度。我们只需要遍历一次数组。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间。

```ts
function numOfSubarrays(arr: number[], k: number, threshold: number): number {
    let ans = 0
    let sum = 0
    for (let i = 0; i < k; i++) {
        sum += arr[i]
    }
    if (sum / k >= threshold) {
        ans++
    }

    for (let i = k; i < arr.length; i++) {
        sum += arr[i] - arr[i - k]
        if (sum / k >= threshold) {
            ans++
        }
    }
    return ans
};
```

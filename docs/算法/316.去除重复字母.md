# [316.去除重复字母](https://leetcode.cn/problems/remove-duplicate-letters/description/)
给你一个字符串 `s` ，请你去除字符串中重复的字母，使得每个字母只出现一次。需保证 **返回结果的字典序最小**（要求不能打乱其他字符的相对位置）。

:::info **字典序**
考虑字符串 `a` 与 字符串 `b`，如果字符串 `a` 在 `a` 与 `b` 相异的第一处的字符在字母表上先于对应 `b` 在此处的字符出现，则称字符串 `a` 字典序小于 `b`。如果 `a` 或 `b` 其中较短的字符串为另一个字符串的前半部分，则较短的字符串字典序小于另一个字符串。
:::

示例1：
- **输入**：s = "bcabc"
- **输出**："abc"

示例2：
- **输入**：s = "cbacdcbc"
- **输出**："acdb"

**注意**：该题与 1081 [https://leetcode.cn/problems/smallest-subsequence-of-distinct-characters](https://leetcode.cn/problems/smallest-subsequence-of-distinct-characters) 相同

## 解题思路
### 1.频率哈希表与单调栈
对于示例2，我们分析一下。对于 s=cbacdcbc，从左到右遍历其中的字母：
- s[0]=c。由于只遍历了一个字母，目前已知字典序最小的字符串是 c。
- s[1]=b。如果右边没有字母 c，那么 s[0]=c 必须保留；实际上右边还有字母 c，由于 b < c ，我们可以去掉 c，改用 b 当作目前字典序最小的字符串。
- s[2]=a。同样的，由于 a < b 且比右边还有字母 b，我们可以去掉 b，改用 a 当作目前字典序最小的字符串（下面记作 ans）。
- s[3]=c。由于 c 比 a 大，只能添加到 a 后面，现在 ans=ac。
- s[4]=d。由于 d 比 c 大，只能添加到 c 后面，现在 ans=acd。
- s[5]=c。由于 acd 里面已经有 c 了，直接跳过。现在 ans=acd。
- s[6]=b。我们发现 b 比 d 小，能不能像上面 s[1] 和 s[2] 那样，去掉 d，添加 b 呢？这是不行的，因为后面没有 d 了，我们只能老老实实地添加到 d 后面，现在 ans=acdb。
- s[7]=c。由于 acdb 里面已经有 c 了，直接跳过。

那么怎么知道右边是否还有某个字母呢？可以在最开始遍历一遍字符串s得到每个字母的频率哈希表，然后在遍历处理时对当前遍历的字母的频率减一，如果频率为0，说明右边没有这个字母了。

具体实现如下：
1. 统计每个字母的出现次数，记到一个哈希map中。
2. 遍历字符串s，当前遍历的字母频率减一，如果当前字母已经在结果中，直接跳过。对于ans中字母是否存在可以用一个哈希set来记录。
3. 如果当前字母不在结果中，判断当前字母是否比结果中的最后一个字母小，且结果中最后一个字母在右边还有出现的机会，如果是，则将结果中的最后一个字母弹出。
4. 重复第3步，直到ans为空或者当前字母比结果中的最后一个字母大，或者结果中最后一个字母在右边没有出现的机会了。
5. 将当前字母添加到结果中，并标记当前字母已经在结果中。
6. 遍历结束后，返回结果。

- 时间复杂度：O(n)，其中 n 是字符串 s 的长度。每个字母最多入栈和出栈各一次，所以均摊时间复杂度为 O(n)。
- 空间复杂度：O(n)，哈希表和栈的空间复杂度为 O(n)。

```ts
function removeDuplicateLetters(s: string): string {
    const countMap = new Map()
    s.split('').forEach(char => {
        countMap.set(char, (countMap.get(char) ?? 0) + 1)
    })
    const ans = []
    const ansSet = new Set()
    for (const char of s) {
        // 当前遍历的字母频率减一
        countMap.set(char, countMap.get(char) - 1)
        // 如果当前字母已经在结果中，直接跳过
        if (ansSet.has(char)) {
            continue
        }
        // 结果不为空，且当前字母比结果中的最后一个字母小，且结果中最后一个字母在右边还有出现的机会
        while (ans.length !== 0 && char < ans[ans.length - 1] && countMap.get(ans[ans.length - 1]) > 0) {
            const last = ans.pop()
            ansSet.delete(last)
        }
        ans.push(char)
        ansSet.add(char)
    }
    return ans.join('')
};
```

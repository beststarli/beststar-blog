# 92.反转链表II
给你单链表的头指针`head`和两个整数`left`和`right`，其中`left <= right`。请你反转从位置`left`到位置`right`的链表节点，返回反转后的链表。

示例1：
![92](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/92.jpg)
- **输入**：head = [1,2,3,4,5], left = 2, right = 4
- **输出**：[1,4,3,2,5]

示例2：
- **输入**：head = [5], left = 1, right = 1
- **输出**：[5]

**进阶**：
- 你可以使用一趟扫描完成反转吗？

## 解题思路
### 1.头结点后移再计数反转并连接
这道题和[206.反转链表](/docs/算法/反转链表)的区别在于我们需要反转链表的一部分，而不是整个链表。事实上把问题简化可以发现，本质和206题还是一样的，无非是开始的头结点需要后移，并且有一个计数器来限制反转的次数，最重要的是要把反转后的链表和原链表连接起来。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseBetween(head: ListNode | null, left: number, right: number): ListNode | null {
    if (!head || left === right) return head;

    const dummy = new ListNode(0, head);
    let prev = dummy;

    for (let i = 1; i < left; i++) {
        prev = prev.next!;
    }

    let curr: ListNode | null = prev.next;
    let next: ListNode | null = null;
    let reversePrev: ListNode | null = null;
    const leftNode = curr; 

    for (let i = left; i <= right; i++) {
        if (curr === null) break;

        next = curr.next;
        curr.next = reversePrev;
        reversePrev = curr;
        curr = next;
    }

    prev.next = reversePrev; 
    leftNode!.next = curr;   

    return dummy.next;
};
```

### 2.只修改一个next
解法参考自[灵茶山艾府](https://leetcode.cn/problems/reverse-linked-list-ii/solutions/1992226/you-xie-cuo-liao-yi-ge-shi-pin-jiang-tou-teqq/d)。

思路和解法1一致，不过优化了反转链表的过程，反转链表的过程不需要使用三个指针来完成，而是只需要一个指针来完成反转链表的过程。需要注意的是在拼接时应先接反转链表的头结点再接反转链表的尾结点。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseBetween(head: ListNode | null, left: number, right: number): ListNode | null {
    const dummy = new ListNode(0, head)
    let p0 = dummy
    for (let i = 0; i < left - 1; i++) {
        p0 = p0.next
    }

    let pre = null
    let cur = p0.next
    for (let i = 0; i < right-left+1; i++) {
        const nxt = cur.next
        cur.next = pre
        pre = cur
        cur = nxt
    }
    p0.next.next = cur
    p0.next = pre
    return dummy.next
};
```

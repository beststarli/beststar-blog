# 48.旋转图像
给定一个*n × n*的二维矩阵`matrix`表示一个图像。请你将图像顺时针旋转 90 度。

你必须在 **原地** 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。

:::info **原地算法**
原地算法（In-place Algorithm） 是指在算法执行过程中，额外使用的空间复杂度为 O(1) 的算法，也就是说，除了输入本身占用的空间之外，几乎不使用额外的空间（常数级别的辅助变量是允许的）。

“原地” = 大部分工作直接在输入数组/数据结构上完成，不额外开大块空间。
:::

## 解题思路
### 1.转置加反转
这道题是将原矩阵顺时针旋转了90度，根据矩阵变换的知识，发现先对原矩阵做转置操作后再对每行进行反转操作就得到了旋转后的矩阵。转置操作是将矩阵的行列进行交换，反转操作是将每行元素进行反转。

时间复杂度：O(mn)，其中 m 和 n 分别是 matrix 的行数和列数。

```ts
function rotate(matrix: number[][]): void {
    for (let i = 0; i < matrix.length; i++) {
        for (let j = i + 1; j < matrix[0].length; j++) {
            [matrix[i][j], matrix[j][i]] = [matrix[j][i], matrix[i][j]] // 数值解构赋值
        }
    }

    for (let k = 0; k < matrix.length; k++) {
        matrix[k].reverse()
    }
};
```

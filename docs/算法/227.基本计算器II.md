# [227.基本计算器II](https://leetcode.cn/problems/basic-calculator-ii/description/)
给你一个字符串表达式 `s` ，请你实现一个基本计算器来计算并返回它的值。

整数除法仅保留整数部分。

你可以假设给定的表达式总是有效的。所有中间结果将在 `[-231, 231 - 1]` 的范围内。

**注意**：不允许使用任何将字符串作为数学表达式计算的内置函数，比如 `eval()` 。

**提示**：
- `s` 由整数和算符 `('+', '-', '*', '/')` 组成，中间由一些空格隔开
- 表达式中的所有整数都是非负整数，且在范围 `[0, 231 - 1]` 内

示例1：
- **输入**：s = "3+2*2"
- **输出**：7

示例2：
- **输入**：s = " 3/2 "
- **输出**：1

示例3：
- **输入**：s = " 3+5 / 2 "
- **输出**：5

## 解题思路
### 1.栈
用栈来处理，遇到数字则入栈；遇到算符则取出栈顶两个数字进行计算，并将结果压入栈中。需要注意的是，除法需要向零截断，所以在js中需要用`Math.trunc()`来处理，而不是直接用`Math.floor()`。注意本题提示，字符串存在空格，需要先对字符串进行处理，去掉空格。且数字不只是一位数，可能是两位数，所以需要用一个变量来存储数字。

- 时间复杂度：O(n)，其中 n 是字符串 s 的长度。
- 空间复杂度：O(n)，其中 n 是字符串 s 的长度。

```ts
function calculate(s: string): number {
    s = s.replaceAll(' ', '')
    const stack = []
    let num = 0     // 数字可能是两位数
    let sign = '+'  // 前一个符号
    for (let i = 0; i < s.length; i++) {
        const char = s[i]
        if (char >= '0' && char <= '9') {
            num = num * 10 + (char.charCodeAt(0) - '0'.charCodeAt(0))
        }
        if (!(char >= '0' && char <= '9') || i === s.length - 1) {
            if (sign === '+') {
                stack.push(num)
            } else if (sign === '-') {
                stack.push(-num)
            } else if (sign === '*') {
                stack.push(stack.pop() * num)
            } else if (sign === '/') {
                stack.push(Math.trunc(stack.pop() / num))
            }
            sign = char
            num = 0
        }
    }
    return stack.reduce((a, b) => a + b)
};
```

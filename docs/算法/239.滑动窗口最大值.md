# 239.滑动窗口最大值
给你一个整数数组`nums`，有一个大小为`k`的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的`k`个数字。滑动窗口每次只向右移动一位。返回*滑动窗口中的最大值*。

示例1：
- **输入**：nums = [1,3,-1,-3,5,3,6,7], k = 3
- **输出**：[3,3,5,5,6,7]
- **解释**：
```text
滑动窗口的位置                最大值
---------------               -----------
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

示例2：
- **输入**：nums = [1], k = 1
- **输出**：[1]

## 解题思路
### 1.暴力循环
既然是为了计算数组切片内最大值，那么最直接的思路就是每次切片后都遍历一次切片来计算最大值，直到遍历完整个数组。

时间复杂度：O(n·klogk)

:::danger **超出时间限制**
本方法不能通过所用测试用例，不是一个可通过的题解。
:::

```ts
function maxSlidingWindow(nums: number[], k: number): number[] {
    let window: number[] = []
    const result: number[] = []

    if (k === nums.length) {
        let tempMax = nums[0]
        for (const num of nums) {
            tempMax = Math.max(num, tempMax)
        }
        result.push(tempMax)
    } else {
        for (let i = 0; i <= nums.length - k; i++) {
            for (let j = i; j < i + k; j++) {
                window.push(nums[j])
            }

            const sortWindow = window.sort((a,b) => a-b)
            result.push(sortWindow[k - 1])

            window = []
        }
    }

    return result
};
```
或更简洁些，直接使用`nums.slice(i, i + k)`来获取切片，`Math.max()`和`...`展开语法来进行窗口内部排序：

时间复杂度：O(n·k)

```ts
function maxSlidingWindow(nums: number[], k: number): number[] {
    const result: number[] = []

    for (let i = 0; i <= nums.length - k; i++) {
        const window = nums.slice(i, i + k)
        result.push(Math.max(...window))
    }

    return result
};
```

### 2.单调队列
对于这个固定大小的窗口，实际是一个从队头到队尾数字由大到小的单调队列。设计单调队列时`pop`和`push`操作要保持以下原则：
- pop(value):如果窗口移除的元素value等于单调队列的出口元素，那么队列弹出元素，否则不用任何操作。
- push(value):如果push的元素value大于入口元素的数值，那么就将队列入口的元素弹出，直到push元素的数值小于等于队列入口元素的数值为止。

具体解法见[代码随想录](https://programmercarl.com/0239.%E6%BB%91%E5%8A%A8%E7%AA%97%E5%8F%A3%E6%9C%80%E5%A4%A7%E5%80%BC.html#%E6%80%9D%E8%B7%AF)

时间复杂度：O(n)

```ts
function maxSlidingWindow(nums: number[], k: number): number[] {
    let result: number[] = []

    class MonoQueue {
        private queue: number[]
        constructor() {
            this.queue = []
        }

        public top(): number|undefined {
            return this.queue[0]
        }

        public enqueue(value: number): void {
            let back: number|undefined = this.queue[this.queue.length - 1]
            while (back !== undefined && back < value) {
                this.queue.pop()
                back = this.queue[this.queue.length - 1]
            }
            this.queue.push(value)
        }

        public dequeue(value: number): void { 
            let top: number|undefined = this.top()
            if (top !== undefined && top === value) {
                this.queue.shift()
            }
        }
    }

    const helperQueue: MonoQueue = new MonoQueue()
    let i = 0
    let j = 0

    while(j < k) {
        helperQueue.enqueue(nums[j++])
    }

    result.push(helperQueue.top()!)

    while (j < nums.length) {
        helperQueue.enqueue(nums[j])
        helperQueue.dequeue(nums[i])

        result.push(helperQueue.top()!)
        j++
        i++
    }

    return result
};
```
## 进阶
**在线性时间复杂度内解决此题。**

要做到时间复杂度为`O(n)`，关键是用**单调队列（双端队列 Deque）**维护“当前窗口可能成为最大值的候选下标”。

### 思路（Deque 存下标）
- 队列里存的是下标 `i`，并且让对应的值 `nums[i]` 保持**从队头到队尾递减**。
- 每次右边界 `r` 向右移动：
    - 把所有比 `nums[r]` 小的下标从队尾弹出（它们不可能再成为后续窗口最大值）。
    - 把 `r` 入队。
    - 把已经滑出窗口左边界 `l = r - k + 1` 的下标从队头弹出。
    - 当 `r >= k - 1` 时，窗口形成，队头就是当前窗口最大值的下标。

### 为什么是线性时间？
每个下标最多：
- 入队一次
- 从队尾被弹出一次（被更大的元素“淘汰”）
- 或从队头被弹出一次（滑出窗口）

所以总操作次数与 `n` 成正比，整体是 $O(n)$。

### 代码

```ts
function maxSlidingWindow(nums: number[], k: number): number[] {
    const result: number[] = []
    const deque: number[] = [] // 存下标，nums[deque[0]] 始终是窗口最大值

    for (let r = 0; r < nums.length; r++) {
        // 1) 维护单调性：把队尾所有更小的值弹掉
        while (deque.length > 0 && nums[deque[deque.length - 1]] <= nums[r]) {
            deque.pop()
        }
        deque.push(r)

        // 2) 移除滑出窗口的队头下标
        const l = r - k + 1
        if (deque[0] < l) {
            deque.shift()
        }

        // 3) 窗口形成后，记录最大值
        if (r >= k - 1) {
            result.push(nums[deque[0]])
        }
    }

    return result
}
```


## 总结
**Deque**：双端队列，允许在头部和尾部执行元素的添加或删除操作。

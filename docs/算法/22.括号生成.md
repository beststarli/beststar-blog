# 22.括号生成
数字`n`代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

示例1：
- **输入**：n = 3
- **输出**：["((()))","(()())","(())()","()(())","()()()"]

示例2：
- **输入**：n = 1
- **输出**：["()"]

## 解题思路
这道题需要在0,1,2,...,2n-1的位置上选n个数字符'('，剩下的位置上放')'。我们可以使用回溯算法来生成所有可能的括号组合，并在生成过程中确保括号的有效性。本题和[77.组合](/docs/算法/77.组合)类似，但是有额外约束，也就是**对于括号字符串的任意前缀，右括号的个数不能超过左括号的个数**。

### 1.枚举当前位置填左还是右
观察示例可以发现，对于字符串的第一项，只能填左括号'('，对于字符串的最后一项，只能填右括号')'。对于其他位置，我们可以选择填左括号'('或者右括号')'，但是需要满足前缀约束。我们可以使用两个变量`left`和`right`来分别记录当前已经填入的左括号和右括号的数量。当`left`小于`n`时，我们可以选择填入左括号'('；当`right`小于`left`时，我们可以选择填入右括号')'。当`right`等于`n`时，说明已经生成了一个有效的括号组合，我们将其加入结果中。于是根据要填入的括号类型，我们可以分为两种情况进行递归。

时间复杂度：O(n * C(2n, n))，其中C(2n, n)是组合数，表示从2n个元素中选取n个元素的组合数。生成每个组合需要O(n)的时间。

```ts
function generateParenthesis(n: number): string[] {
    const ans = []
    const path = Array(n * 2)

    function dfs(left, right) {
        if (right === n) {
            ans.push(path.join(''))
            return
        }
        if (left < n) {
            path[left + right] = '('
            dfs(left + 1, right)
        }
        if (right < left) {
            path[left + right] = ')'
            dfs(left, right + 1)
        }
    }

    dfs(0, 0)
    return ans
};
```

### 2.枚举下一个左括号的位置
我们把整个字符串先用右括号填满，用`path`数组记录左括号的位置。对字符串从左往右开始填入左括号，过程中要时刻保证右括号的个数不能超过左括号的个数。对于还可填入右括号的数量用`balance`来记录，当`balance`为`0`时，说明当前路径已经不合法了，我们需要回退到上一个可选项继续尝试。

```ts
function generateParenthesis(n: number): string[] {
    const ans = []
    const path = []

    function dfs(index, balance) {
        if (path.length === n) {
            const string = Array(n * 2).fill(')')
            for (const item of path) {
                string[item] = '('
            }
            ans.push(string.join(''))
            return
        }

        for (let right = 0; right <= balance; right++) {
            path.push(index + right)
            dfs(index + right + 1, balance - right + 1)
            path.pop()
        }
    }
    dfs(0, 0)
    return ans
};
```

# [143.重排链表](https://leetcode.cn/problems/reorder-list/description/)   
给定一个单链表 `L` 的头节点 `head` ，单链表 `L` 表示为：
```txt
L0 → L1 → … → Ln - 1 → Ln
```
请将其重新排列后变为：
```txt
L0 → Ln → L1 → Ln - 1 → L2 → Ln - 2 → …
```
不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

示例1：
![143-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/143-1.png)
- **输入**：head = [1,2,3,4]
- **输出**：[1,4,2,3]

- **输入**：head = [1,2,3,4,5]
- **输出**：[1,5,2,4,3]

## 解题思路
### 1.寻找中点反转后交叉改变指针
原题的意思是将链表改为由原链表一头一尾交替向中心夹逼来排列的形式，高效的办法是找到原链表的中点后将原链表看作是两段，对后半段反转后就得到了原链表最尾部节点为起点的第二段链表，对于前一段链表，交替改变指针即可。

用到的方法可以参考：
- [876.链表的中间结点](/docs/算法/876.链表的中间结点.md)
- [206.反转链表](/docs/算法/206.反转链表.md)

- 时间复杂度：O(n)，其中 n 是链表的节点数。我们只需要遍历链表一次。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

/**
 Do not return anything, modify head in-place instead.
 */
function findMid(node) {
    let slow = node
    let fast = node
    while (fast && fast.next) {
        slow = slow.next
        fast = fast.next.next
    }
    return slow
}

function reverse(node) {
    let curr = node
    let prev = null
    while (curr !== null) {
        const next = curr.next
        curr.next = prev
        prev = curr
        curr = next
    }
    return prev
}

function reorderList(head: ListNode | null): void {
    const mid = findMid(head)
    let head2 = reverse(mid)
    while (head2.next) {
        const next = head.next
        const next2 = head2.next
        head.next = head2
        head2.next = next
        head = next
        head2 = next2
    }
};
```

# 75.颜色分类
给定一个包含红色、白色和蓝色、共`n`个元素的数组`nums`，**原地** 对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数`0`、`1`和`2`分别表示红色、白色和蓝色。

必须在不使用库内置的 sort 函数的情况下解决这个问题。

示例1：
- **输入**：nums = [2,0,2,1,1,0]
- **输出**：[0,0,1,1,2,2]

示例2：
- **输入**：nums = [2,0,1]
- **输出**：[0,1,2]

**进阶**：
- 你能想出一个仅使用常数空间的一趟扫描算法吗？

## 解题思路
### 1.双指针
在对数组每次进行遍历时，用一个常量记录一下原来的颜色，然后先将颜色都刷成`2`，再根据原来的颜色，用`pointer1`去记录可能是`0`或者`1`颜色排序后的终止位置，并把数组头部颜色用`1`覆盖；同理`pointer2`去记录可能是`0`颜色排序后的终止位置，并把数组头部颜色用`0`覆盖。即完成排序

时间复杂度：O(n)。

```ts
/**
 Do not return anything, modify nums in-place instead.
 */
function sortColors(nums: number[]): void {
    let pointer1 = 0
    let pointer2 = 0

    for (let i = 0; i < nums.length; i++) {
        const temp = nums[i]
        nums[i] = 2
        if (temp <= 1) {
            nums[pointer1] = 1
            pointer1++
        }
        if (temp === 0) {
            nums[pointer2] = 0
            pointer2++
        }
    }
};
```

# [662.二叉树最大宽度](https://leetcode.cn/problems/maximum-width-of-binary-tree/description/)   
给你一棵二叉树的根节点 `root` ，返回树的 **最大宽度** 。

树的 最大宽度 是所有层中最大的 宽度 。

每一层的 宽度 被定义为该层最左和最右的非空节点（即，两个端点）之间的长度。将这个二叉树视作与满二叉树结构相同，两端点间会出现一些延伸到这一层的 `null` 节点，这些 `null` 节点也计入长度。

题目数据保证答案将会在  **32** 位 带符号整数范围内。

示例1：
![662-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/662-1.jpg)
- **输入**：root = [1,3,2,5,3,null,9]
- **输出**：4
- **解释**：最大宽度出现在树的第 3 层，宽度为 4 (5,3,null,9) 。

示例2：
![662-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/662-2.jpg)
- **输入**：root = [1,3,2,5,null,null,9,6,null,7]
- **输出**：7
- **解释**：最大宽度出现在树的第 4 层，宽度为 7 (6,null,null,null,null,null,7) 。

示例3：
![662-3](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/662-3.jpg)
- **输入**：root = [1,3,2,5]
- **输出**：2
- **解释**：最大宽度出现在树的第 2 层，宽度为 2 (3,2) 。


## 解题思路
### 1.队列
利用堆式索引（heap-like indexing）为每个节点编号，根节点编号为 0。对于任意节点编号为 `i`，其左子节点编号为 `2*i`，右子节点编号为 `2*i + 1`。这样每一层的宽度 = 该层最右节点编号 - 最左节点编号 + 1。

由于二叉树深度最多 3000，索引会指数级增长超出 Number 安全范围，因此使用 **BigInt** 存储编号。同时每层做 **归一化**（减去该层首节点编号），避免 BigInt 数值过大。

**算法步骤**：
1. BFS 层序遍历，队列存 `[节点, 编号]`
2. 每层记录第一个节点的编号 `firstIndex`
3. 遍历该层所有节点，当前节点编号减去 `firstIndex` 得到归一化索引
4. 将子节点编号为 `normalized * 2n` 和 `normalized * 2n + 1n` 入队
5. 该层宽度 = `lastIndex + 1`，更新最大宽度

- 时间复杂度：O(n)，其中 n 是二叉树的节点数。我们需要遍历每个节点一次。
- 空间复杂度：O(n)，其中 n 是二叉树的节点数。我们使用了一个队列来存储每一层的节点，最坏情况下队列中可能包含所有节点。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function widthOfBinaryTree(root: TreeNode | null): number {
    const queue: [TreeNode, bigint][] = [[root, 0n]]
    let max = 0
    while (queue.length) {
        const level = queue.length
        const firtIndex = queue[0][1]
        let lastIndex = 0n

        for (let i = 0; i < level; i++) {
            const [node, index] = queue.shift()
            const normalized = index - firtIndex
            lastIndex = normalized
            if (node.left) {
                queue.push([node.left, normalized * 2n])
            }
            if (node.right) {
                queue.push([node.right, normalized * 2n + 1n])
            }
        }
        const width = Number(lastIndex) + 1
        max = Math.max(max, width)
    }
    return max
};
```

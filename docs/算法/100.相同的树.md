# 100.相同的树
给你两棵二叉树的根节点`p`和`q`，编写一个函数来检验这两棵树是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

示例1：
![100-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/100-1.jpg)
- **输入**：p = [1,2,3], q = [1,2,3]
- **输出**：true

示例2：
![100-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/100-2.jpg)
- **输入**：p = [1,2], q = [1,null,2]
- **输出**：false

示例3：
![100-3](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/100-3.jpg)
- **输入**：p = [1,2,1], q = [1,1,2]
- **输出**：false

## 解题思路
### 1.递归
还是递归到最小单元，也就是两个树都是空树的情况，此时它们是相同的；如果其中一个树是空树，另一个树不是空树，此时它们不相同；如果两个树的根节点值不同，此时它们不相同。对于更大的树，我们可以先比较根节点的值是否相同，再递归地比较左右子树是否相同，如果都相同，那么整棵树就相同了。

时间复杂度：O(n)，其中n是两棵树中较小的树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSameTree(p: TreeNode | null, q: TreeNode | null): boolean {
    if (p === null || q === null) {
        return p === q
    }

    return (p.val === q.val)
        && isSameTree(p.left, q.left)
        && isSameTree(p.right, q.right)
};
```

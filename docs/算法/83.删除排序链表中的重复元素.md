# [83.删除排序链表中的重复元素](https://leetcode.cn/problems/remove-duplicates-from-sorted-list/description/)   
给定一个已排序的链表的头 `head` ， *删除所有重复的元素，使每个元素只出现一次* 。返回 ***已排序的链表*** 。

示例1：
![83-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/83-1.jpg)
- **输入**：head = [1,1,2]
- **输出**：[1,2]

示例2：
![83-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/83-2.jpg)
- **输入**：head = [1,1,2,3,3]
- **输出**：[1,2,3]

## 解题思路
### 1.指针
因为要返回排序后的链表，所以要建立守卫节点dummy，初始化指针p0指向dummy，然后遍历链表，如果当前节点的值和下一个节点的值相同，则删除下一个节点，也就是将p0.next指向p0.next.next，继续循环体，否则指针后移。不过要注意在建立dummy的时候其val不能填具体数值，否则会导致特定测试用例无法通过。

- 时间复杂度：O(n)，其中 n 是链表的节点数。我们只需要遍历链表一次。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function deleteDuplicates(head: ListNode | null): ListNode | null {
    const dummy = new ListNode(null, head)
    let p0 = dummy
    while (p0 && p0.next) {
        if (p0.val === p0.next.val) {
            p0.next = p0.next.next
            continue
        } else {
            p0 = p0.next
        }
    }
    return dummy.next
};
```

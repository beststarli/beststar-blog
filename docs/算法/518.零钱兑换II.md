# 518.零钱兑换II
给你一个整数数组`coins`表示不同面额的硬币，另给一个整数`amount`表示总金额。

请你计算并返回可以凑成总金额的硬币组合数。如果任何硬币组合都无法凑出总金额，返回`0`。

假设每一种面额的硬币有无限个。 

题目数据保证结果符合 32 位带符号整数。

示例1：
- **输入**：amount = 5, coins = [1, 2, 5]
- **输出**：4
- **解释**：有四种方式可以凑成总金额：
```txt
5=5
5=2+2+1
5=2+1+1+1
5=1+1+1+1+1
```

示例2：
- **输入**：amount = 3, coins = [2]
- **输出**：0
- **解释**：只用面额 2 的硬币不能凑成总金额 3 。

示例3：
- **输入**：amount = 10, coins = [10] 
- **输出**：1

## 解题思路
本题和[322.零钱兑换](/docs/算法/零钱兑换)的思路一样。

### 1.递归加记忆化
定义递归`dfs(i, c)`表示使用前`i`种硬币凑成总金额`c`的组合数。那么有选或不选两张情况：
- 不选第i种硬币：`dfs(i - 1, c)`
- 选第i种硬币：`dfs(i, c - coins[i - 1])`

由**加法原理**可得状态转移方程：$dfs(i,c)=dfs(i−1,c)+dfs(i,c−coins[i])$

:::info **加法原理**
如果事件 A 和事件 B 是互斥的（即不能同时发生，不再选硬币的同时，又继续选同一种硬币），那么发生事件 A 或事件 B 的总数等于事件 A 的数量加上事件 B 的数量。
:::

递归入口为`dfs(coins.length - 1, amount)`，边界条件为当`coins.length == 0`时，只有当`amount == 0`时才返回1，否则返回`0`。

时间复杂度：$O(n * amount)$，其中`n`为硬币种类数，`amount`为总金额。

空间复杂度：$O(n * amount)$，其中`n`为硬币种类数，`amount`为总金额。

```ts
function change(amount: number, coins: number[]): number {
    const length = coins.length
    const memo = Array.from({ length: length}, () => Array(amount + 1).fill(-1))
    return dfs(length - 1, amount, coins, memo)
};

function dfs(i, c, coins, memo) {
    if (i < 0) {
        return (c === 0) ? 1 : 0
    }
    if (memo[i][c] !== -1) {
        return memo[i][c]
    }
    if (c < coins[i]) {
        return memo[i][c] = dfs(i - 1, c, coins, memo)
    }

    return memo[i][c] = dfs(i - 1, c, coins, memo) + dfs(i, c - coins[i], coins, memo)
}
```

### 2.递推
一比一将「归」的过程翻译为递推，从下往上计算，结果数组预选用无法凑满金额的`0`填满。

时间复杂度：$O(n * amount)$，其中`n`为硬币种类数，`amount`为总金额。

空间复杂度：$O(n * amount)$，其中`n`为硬币种类数，`amount`为总金额。

```ts
function change(amount: number, coins: number[]): number {
    const length = coins.length
    const memo = Array.from({ length: length + 1}, () => Array(amount + 1).fill(0))
    memo[0][0] = 1

    for (let i = 0; i < length; i++) {
        for (let c = 0; c <= amount; c++) {
            if (c < coins[i]) {
                memo[i + 1][c] = memo[i][c]
            } else {
                memo[i + 1][c] = memo[i][c] + memo[i + 1][c - coins[i]]
            }
        }
    }

    return memo[length][amount]
};
```

### 3.空间优化
由于状态转移方程中`dfs(i, c)`只依赖于`dfs(i - 1, c)`和`dfs(i, c - coins[i])`，所以可以将二维数组优化为一维数组。

时间复杂度：$O(n * amount)$，其中`n`为硬币种类数，`amount`为总金额。

空间复杂度：$O(amount)$，其中`amount`为总金额。

```ts
function change(amount: number, coins: number[]): number {
    const memo = Array(amount + 1).fill(0)
    memo[0] = 1

    for (const coin of coins) {
        for (let c = coin; c <= amount; c++) {
            memo[c] += memo[c - coin]
        }
    }

    return memo[amount]
};
```

## 解题思路
### 1.反转、剔除、再反转
既然题干是删除链表中倒数某个位置的结点，我们知道想要顺序删除链表中特定结点是简单的，那么可以先反转链表这样就是顺序删除了，删除完后再反转回来就好了。注意边界条件的处理，删除的结点可能是头结点或者尾结点。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
    if (!head) return null

    let prev = null
    let cur = head

    while (cur !== null) {
        const nextTemp = cur.next
        cur.next = prev
        prev = cur
        cur = nextTemp
    }

    let newNode = new ListNode(0, prev)
    let prev2 = newNode

    for (let i = 0; i < n-1; i++) {
        if (!prev2.next) {
            return head
        }
        prev2 = prev2.next
    }

    prev2.next = prev2.next ? prev2.next.next : null

    prev = null
    cur = newNode.next
    
    while (cur !== null) {
        const nextTemp = cur.next
        cur.next = prev
        prev = cur
        cur = nextTemp
    }

    return prev
};
```

### 2.双指针
解法参考自[灵茶山艾府](https://leetcode.cn/problems/remove-nth-node-from-end-of-list/solutions/2004057/ru-he-shan-chu-jie-dian-liu-fen-zhong-ga-xpfs/?envType=study-plan-v2&envId=top-100-liked)。

想象有一把长度固定的尺子，左端点在链表头部，右端点在正数第`n`个节点。向右移动尺子，当尺子右端点到达链表末尾时，左端点就在倒数第`n`个节点。由于需要删除节点，我们需要找倒数第`n`个节点的前一个节点（倒数第`n+1`个节点），这样才能做删除操作。这把尺子就是双指针，右指针先向右走`n`步，然后左指针和右指针一起向右走，直到右指针到达链表末尾，此时左指针就停在了倒数第`n+1`个节点上。最后删除左指针的下一个节点即可。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
    // 由于可能会删除链表头部，用哨兵节点简化代码
    const dummy = new ListNode(0, head);
    let left = dummy;
    let right = dummy;
    while (n--) {
        right = right.next; // 右指针先向右走 n 步
    }
    while (right.next) {
        left = left.next;
        right = right.next; // 左右指针一起走
    }
    left.next = left.next.next; // 左指针的下一个节点就是倒数第 n 个节点
    return dummy.next;
};
```

## 二刷
### 1.反转剔除再反转
- 时间复杂度：O(n)，其中 n 是链表的节点数。
- 空间复杂度：O(1)，只使用了常数级别的额外空间。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */
 function reverseListNode(node: ListNode | null) {
    let cur = node
    let prev = null
    while (cur !== null) {
        const next = cur.next
        cur.next = prev
        prev = cur
        cur = next
    }
    return prev
 }

function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
    const reverseHead = reverseListNode(head)
    let newNode = new ListNode(0, reverseHead)
    let prev2 = newNode
    for (let count = 1; count <= n - 1; count++) {
        prev2 = prev2.next
    }

    prev2.next = prev2.next.next
    const newHead = reverseListNode(newNode.next)
    return newHead


};
```

### 2.双指针
- 时间复杂度：O(n)，其中 n 是链表的节点数。
- 空间复杂度：O(1)，只使用了常数级别的额外空间。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function removeNthFromEnd(head: ListNode | null, n: number): ListNode | null {
    const dummy = new ListNode(0, head)
    let slow = dummy
    let fast = dummy

    for (let i = 0; i < n; i++) {
        fast = fast.next
    }

    while (fast.next !== null) {
        fast = fast.next
        slow = slow.next
    }

    slow.next = slow.next.next
    return dummy.next
};
```

## 总结
### 删除结点
如何删除链表中特定位置的结点，说明来自于[Hello算法 4.2.1链表常用操作](https://www.hello-algo.com/chapter_array_and_linkedlist/linked_list/#3)：
![19-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/19-1.png)
```ts
/* 删除链表的节点 n0 之后的首个节点 */
function remove(n0: ListNode): void {
    if (!n0.next) {
        return;
    }
    // n0 -> P -> n1
    const P = n0.next;
    const n1 = P.next;
    n0.next = n1;
}
```

### 链表题循环条件
一般在做链表题时，如果要遍历到最后一个节点，需要写`while node`；如果要遍历到倒数第二个节点，需要写`while node.next`。

# [209.长度最小的子数组](https://leetcode.cn/problems/minimum-size-subarray-sum/description/)
给定一个含有 n 个正整数的数组和一个正整数 `target` 。

找出该数组中满足其总和大于等于 `target` 的长度最小的 子数组 `[numsl, numsl+1, ..., numsr-1, numsr]` ，并返回其长度。如果不存在符合条件的子数组，返回 `0` 。

:::info **子数组**
**子数组** 是数组中连续的 非空 元素序列。
:::

示例1：
- **输入**：target = 7, nums = [2,3,1,2,4,3]
- **输出**：2
- **解释**：子数组 [4,3] 是该条件下的长度最小的子数组。

示例2：
- **输入**：target = 4, nums = [1,4,4]
- **输出**：1

示例3：
- **输入**：target = 11, nums = [1,1,1,1,1,1,1,1]
- **输出**：0

## 解题思路
### 1.变长滑动窗口
遍历推进窗口右边界，每次都把当前右边界指向的数加入`sum`，然后对窗口的左边界进行收缩，保证当前窗口内数字总和减去左边界所指数字小于`target`，如果此时窗口内数字总和大于等于`target`，则可能是要找的子数组，更新子数组长度结果。

- 时间复杂度：O(n)，其中 n 是数组 nums 的长度。每个元素被访问至多两次（一次被右边界访问，一次被左边界访问）。
- 空间复杂度：O(1)，只使用了常数级别的额外空间。

```ts
function minSubArrayLen(target: number, nums: number[]): number {
    let ans = Infinity
    let sum = 0
    let left = 0
    for (let right = 0; right < nums.length; right++) {
        sum += nums[right]
        while (sum - nums[left] >= target) {
            sum -= nums[left]
            left++
        }
        if (sum >= target) {
            ans = Math.min(ans, right - left + 1)
        }
    }
    return ans !== Infinity ? ans : 0
};
```

### 2.优化更新逻辑
将ans的更新逻辑放入`while`循环中，将右边界数字加入`sum`后，收缩左边界保证`sum`小于`target`，这次在收缩时先更新ans再更新`sum`和收缩左边界，这样更新后的`ans`一定是当前窗口内数字总和大于等于`target`的子数组长度。

- 时间复杂度：O(n)，其中 n 是数组 nums 的长度。每个元素被访问至多两次（一次被右边界访问，一次被左边界访问）。
- 空间复杂度：O(1)，只使用了常数级别的额外空间。

```ts
function minSubArrayLen(target: number, nums: number[]): number {
    let ans = Infinity
    let sum = 0
    let left = 0
    for (let right = 0; right < nums.length; right++) {
        sum += nums[right]
        while (sum >= target) {
            ans = Math.min(ans, right - left + 1)
            sum -= nums[left]
            left++
        }
    }
    return ans !== Infinity ? ans : 0
};
```

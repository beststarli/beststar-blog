# 347.前K个高频元素
给你一个整数数组`nums`和一个整数`k`，请你返回其中出现频率前`k`高的元素。你可以按 **任意顺序** 返回答案。

示例1：
- **输入**：nums = [1,1,1,2,2,3], k = 2
- **输出**：[1,2]

示例2：
- **输入**：nums = [1], k = 1
- **输出**：[1]

示例3：
- **输入**：nums = [1,2,1,2,1,2,3,1,3,2], k = 2
- **输出**：[1,2]

**进阶**：
- 你所设计算法的时间复杂度 **必须** 优于`O(n log n)`，其中`n`是数组大小。

## 解题思路
### 1.桶排序
答案与元素出现频率相关，首先想到用哈希map来统计所以元素的频率。

把出现次数相同的元素，放到同一个桶中。同一个桶内的元素出现次数相同，无需对桶内元素排序。`buckets[i]`存储出现频率为`i`的元素列表。因此桶的长度为`maxCount + 1`，`buckets[0]`为空。最后对buckets反向遍历，直到答案数组的长度达到`k`停止。

时间复杂度：O(n)。

```ts
function topKFrequent(nums: number[], k: number): number[] {
    const countMap = new Map()
    for (const item of nums) {
        countMap.set(item, (countMap.get(item) || 0) + 1)
    }
    const maxCount = Math.max(...countMap.values())

    const buckets = Array.from({ length: maxCount + 1}, () => [])
    for (const [item, count] of countMap.entries()) {
        buckets[count].push(item)
    }

    const ans = []
    for (let i = maxCount; ans.length < k; i--) {
        ans.push(...buckets[i])
    }
    return ans
};
```

## 二刷
### 1.桶排序
先用一个哈希map来统计每个元素的出现频率，再用数组来建立每个出现频率对应的元素子数组，这个数组的长度为最大元素出现频率加1，最后倒着遍历这个数组，直到答案数组的长度达到`k`停止。

- 时间复杂度：O(n)。
- 空间复杂度：O(n)。

```ts
function topKFrequent(nums: number[], k: number): number[] {
    // 统计每个元素的出现频率
    const count = new Map()
    for (const num of nums) {
        count.set(num, (count.get(num) ?? 0) + 1)
    }
    const countMax = Math.max(...count.values())

    // 相同频率的放在同一个桶中
    const buckets = Array.from({length: countMax + 1}, () => [])
    for (const [num, times] of count.entries()) {
        buckets[times].push(num)
    }

    // 倒序遍历桶，把出现次数前k大的元素加入答案
    const ans = []
    for (let i = countMax; ans.length < k; i--) {
        ans.push(...buckets[i])
    }
    return ans
};
```

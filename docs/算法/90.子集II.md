# [90.子集II](https://leetcode.cn/problems/subsets-ii/description/)
给你一个整数数组 `nums` ，其中可能包含重复元素，请你返回该数组所有可能的 子集（幂集）。

:::info **子集**
数组的 **子集** 是从数组中选择一些元素（可能为空）。
:::

解集 **不能** 包含重复的子集。返回的解集中，子集可以按 **任意顺序** 排列。

示例1：
- **输入**：nums = [1,2,2]
- **输出**：[[],[1],[1,2],[1,2,2],[2],[2,2]]

示例2：
- **输入**：nums = [0]
- **输出**：[[],[0]]

## 解题思路
### 1.回溯——选哪个
与[78.子集](/docs/算法/78.子集)主体类似，只是这题有重复元素，而结果数组不允许存在重复的子集，所以我们需要对输入数组进行排序，然后在递归时添加一个判断跳过重复的元素即可。

- 时间复杂度：O(n * 2^n)，其中n是输入数组的长度。生成子集的时间复杂度是O(2^n)，每个子集需要O(n)的时间来构建。
- 空间复杂度：O(n)。递归调用栈的深度为n，最坏情况下需要O(n)的空间来存储递归调用栈。

```ts
function subsetsWithDup(nums: number[]): number[][] {
    nums.sort()
    const ans = []
    const path = []
    function dfs(index) {
        ans.push([...path])
        for (let i = index; i < nums.length; i++) {
            if (i > index && nums[i] === nums[i - 1]) {
                continue
            }
            path.push(nums[i])
            dfs(i + 1)
            path.pop()
        }
    }
    dfs(0)
    return ans
};
```

# [494.目标和](https://leetcode.cn/problems/target-sum/description/)
给你一个非负整数数组 `nums` 和一个整数 `target` 。

向数组中的每个整数前添加 `'+'` 或 `'-'` ，然后串联起所有整数，可以构造一个 **表达式** ：
- 例如，`nums = [2, 1]` ，可以在 2 之前添加 `'+'` ，在 1 之前添加 `'-'` ，然后串联起来得到表达式 `"+2-1"` 。
返回可以通过上述方法构造的、运算结果等于 `target` 的不同 **表达式** 的数目。

示例1：
- **输入**：nums = [1,1,1,1,1], target = 3
- **输出**：5
- **解释**：一共有 5 种方法让最终目标和为 3 。
```txt
-1 + 1 + 1 + 1 + 1 = 3
+1 - 1 + 1 + 1 + 1 = 3
+1 + 1 - 1 + 1 + 1 = 3
+1 + 1 + 1 - 1 + 1 = 3
+1 + 1 + 1 + 1 - 1 = 3
```

示例2：
- **输入**：nums = [1], target = 1
- **输出**：1

## 解题思路
### 1.递归加记忆化
设nums的元素和为sum，其中添加正号的元素和为p，添加负号的元素和为q，则有p + q = sum，p - q = target，于是有p = (sum + target) / 2, q = (sum - target) / 2。这意味着，只要我们选出的取正号的元素和恰好等于 p，或者取负号的元素和恰好等于 q，就等价于表达式的结果恰好等于 target。所以问题变成从 nums 中选一些数，使得这些数的元素和恰好等于 p（或者 q）的方案数。
- 如果target >= 0, 则p >= q，取q = (sum - target) / 2为背包容量正好
- 如果target < 0, 则p < q，取p = (sum + target) / 2 = (sum - |target|) / 2为背包容量正好

综上可取背包容量为：`(sum - |target|) / 2`

设`s = sum − ∣target∣`，有两种情况需要特判，可以直接返回0：
- 如果`s<0`，由于`nums[i] ≥ 0`，我们无法得到负数，方案数是0。
- 如果`s`是奇数，由于`nums[i]`都是整数，我们无法得到非整数`s / 2`，方案数是0。

- 时间复杂度：O(n * m)，其中 n 是数组 nums 的长度，m 是背包容量。
- 空间复杂度：O(n * m)，其中 n 是数组 nums 的长度， m 是背包容量。

```ts
function findTargetSumWays(nums: number[], target: number): number {
    const length = nums.length
    let sum = nums.reduce((a, b) => a + b)
    sum -= Math.abs(target)
    if (sum < 0 || sum % 2 === 1) {
        return 0
    }
    const n = sum / 2
    const memo = Array.from({ length: length }, () => Array(n + 1).fill(-1))

    function dfs(i, c) {
        if (i < 0) {
            return c === 0 ? 1 : 0
        }
        if (memo[i][c] !== -1) {
            return memo[i][c]
        }
        let res = dfs(i - 1, c)
        if (c >= nums[i]) {
            res += dfs(i - 1, c - nums[i])
        }
        return res
    }
    return dfs(nums.length - 1, n)
};
```

### 2.递推
将递归一比一翻译为递推，在建立memo数组时，数组的第一维长度应为length + 1，数组内元素初始化为0，memo[0][0] = 1，表示不选任何元素时，背包容量为0的方案数为1。根据加法原理，memo[i + 1][c] = memo[i][c] + memo[i][c - nums[i]]，表示不选nums[i]时的方案数加上选nums[i]时的方案数。

- 时间复杂度：O(n * m)，其中 n 是数组 nums 的长度，m 是背包容量。
- 空间复杂度：O(n * m)，其中 n 是数组 nums 的长度， m 是背包容量。

```ts
function findTargetSumWays(nums: number[], target: number): number {
    const length = nums.length
    let sum = nums.reduce((a, b) => a + b)
    sum -= Math.abs(target)
    if (sum < 0 || sum % 2 === 1) {
        return 0
    }
    const n = sum / 2
    const memo = Array.from({ length: length + 1 }, () => Array(n + 1).fill(0))
    memo[0][0] = 1

    for (let i = 0; i < nums.length; i++) {
        for (let c = 0; c <= n; c++) {
            if (c < nums[i]) {
                memo[i + 1][c] = memo[i][c]
            } else {
                memo[i + 1][c] = memo[i][c] + memo[i][c - nums[i]]
            }
        }
    }
    return memo[nums.length][n]
};
```

### 3.空间优化——滚动数组——两个数组
事实上更新memo[i + 1][c]时，只依赖于memo[i][c]和memo[i][c - nums[i]]，用不到memo[i - 1]等之前的数据了,因此可以使用滚动数组优化空间复杂度。将memo数组的第一维长度改为2，表示当前行和上一行。然后通过对i取模2来表示当前行和上一行。

- 时间复杂度：O(n * m)，其中 n 是数组 nums 的长度，m 是背包容量。
- 空间复杂度：O(m)，其中 m 是背包容量。

```ts
function findTargetSumWays(nums: number[], target: number): number {
    const length = nums.length
    let sum = nums.reduce((a, b) => a + b)
    sum -= Math.abs(target)
    if (sum < 0 || sum % 2 === 1) {
        return 0
    }
    const n = sum / 2
    const memo = Array.from({ length: 2 }, () => Array(n + 1).fill(0))
    memo[0][0] = 1

    for (let i = 0; i < nums.length; i++) {
        for (let c = 0; c <= n; c++) {
            if (c < nums[i]) {
                memo[(i + 1) % 2][c] = memo[i % 2][c]
            } else {
                memo[(i + 1) % 2][c] = memo[i % 2][c] + memo[i % 2][c - nums[i]]
            }
        }
    }
    return memo[nums.length % 2][n]
};
```

### 4.空间优化——滚动数组——一个数组
可以只用一个数组来表示当前行，更新memo[c]时，memo[c] = memo[c] + memo[c - nums[i]]，表示不选nums[i]时的方案数加上选nums[i]时的方案数。注意这里需要从后往前更新memo数组，否则会出现重复计算。内层循环的c从n到nums[i]，表示背包容量从大到小更新。

- 时间复杂度：O(n * m)，其中 n 是数组 nums 的长度，m 是背包容量。
- 空间复杂度：O(m)，其中 m 是背包容量。

```ts
function findTargetSumWays(nums: number[], target: number): number {
    const length = nums.length
    let sum = nums.reduce((a, b) => a + b)
    sum -= Math.abs(target)
    if (sum < 0 || sum % 2 === 1) {
        return 0
    }
    const n = sum / 2
    const memo = Array(n + 1).fill(0)
    memo[0] = 1

    for (const num of nums) {
        for (let c = n; c >= num; c--) {
            memo[c] += memo[c - num]
        }
    }
    return memo[n]
};
```

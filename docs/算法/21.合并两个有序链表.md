# 21.合并两个有序链表
将两个升序链表合并为一个新的 **升序** 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例1：
![21](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/21.jpg)
- **输入**：l1 = [1,2,4], l2 = [1,3,4]
- **输出**：[1,1,2,3,4,4]

示例2：
- **输入**：l1 = [], l2 = []
- **输出**：[]

示例3：
- **输入**：l1 = [], l2 = [0]
- **输出**：[0]

## 解题思路
### 1.迭代
题干说明了要合并的两个链表是升序的，那么可以新建一个虚拟头节点，我们只需要比较两个链表的当前节点的值，将较小的节点连接到新链表的末尾，并将对应链表的指针向后移动一位。重复这个过程，直到其中一个链表被完全遍历。最后，将另一个链表剩余的部分连接到新链表的末尾。返回新链表的头节点即可。

注意：
- 虚拟头节点是一个辅助节点，它的值可以是任意的，因为我们最终返回的是虚拟头节点的下一个节点。
- 链表剩余部分的连接是必要的，因为当其中一个链表被完全遍历时，另一个链表可能还有未处理的节点，这些节点已经是升序的了，所以直接连接到新链表的末尾即可。

时间复杂度：O(n + m)，其中 n 和 m 分别是两个链表的长度。我们需要遍历两个链表一次。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function mergeTwoLists(list1: ListNode | null, list2: ListNode | null): ListNode | null {
    const newNode = new ListNode(0)
    let current = newNode

    while (list1 && list2) {
        if (list1.val <= list2.val) {
            current.next = list1
            list1 = list1.next
        } else {
            current.next = list2
            list2 = list2.next
        }
        current = current.next
    }
    current.next = list1 ? list1 : list2

    return newNode.next
};
```

### 2.递归
递归解法的思路与迭代类似。我们比较两个链表的当前节点的值，选择较小的节点作为新链表的当前节点，并递归地合并剩余的部分。递归终止条件是当其中一个链表为空时，返回另一个链表。

时间复杂度：O(n + m)，其中 n 和 m 分别是两个链表的长度。我们需要遍历两个链表一次。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function mergeTwoLists(list1: ListNode | null, list2: ListNode | null): ListNode | null {
    // 递归终止条件
    if (!list1) return list2
    if (!list2) return list1

    // 递归调用，比较当前节点的值，连接较小的节点，并继续递归处理剩余的部分
    if (list1.val <= list2.val) {
        list1.next = mergeTwoLists(list1.next, list2);
        return list1;
    } else {
        list2.next = mergeTwoLists(list1, list2.next);
        return list2;
    }
};
```

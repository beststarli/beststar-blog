# 131.分割回文串
给你一个字符串`s`，请你将 `s` 分割成一些 **子串**，使每个子串都是 **回文串** 。返回 `s` 所有可能的分割方案。

:::info **子串** **回文串**
- **子字符串** 是字符串中连续的 **非空** 字符序列。
- **回文** 串是向前和向后读都相同的字符串。
:::

示例1：
- **输入**：s = "aab"
- **输出**：[["a","a","b"],["aa","b"]]

示例2：
- **输入**：s = "a"
- **输出**：[["a"]]

## 解题思路
### 1.对分割位置递归
是否要在`i`和`i+1`处分割，把`s[i]`作为当前子串的最后一个字符，把`s[i+1]`作为下一个子串的第一个字符。注意`s[s.length − 1]`一定是最后一个字符，所以在`i=n−1`的时候一定要分割。

注意到题目说明每个子串都是回文串，所以如果最后一个字符串不是回文串，我们不会递归到`i=n`的边界，不会把不合法的分割加入答案。

时间复杂度：O(N*2^N)，其中 N 是字符串 s 的长度。最坏情况下，字符串 s 的每个字符都不同，因此每个字符都可以单独成为一个回文子串，这样就有 2^(N-1) 个分割方案。对于每个分割方案，我们需要 O(N) 的时间来检查每个子串是否是回文的。

```ts
function partition(s: string): string[][] {
    const length = s.length
    const ans: string[][] = []
    const path: string[] = []

    // 现在 s 未被分割的部分为 [start, length - 1]
    // 当前位于下标 i，讨论是否在 i 和 i+1 之间切一刀
    function dfs(index, start) {
        // s 分割完毕
        if (index === length) {
            ans.push(path.slice())  // 复制 path
            return
        }

        // 不分割
        // i=n-1 时必须分割（这是最后一段），i<n-1 时才可以不分割
        if (index < length - 1) {
            dfs(index + 1, start)
        }

        // 分割，那么得到子串 [start, i]
        // 判断子串 [start, i] 是不是回文串
        if (isPalindrome(s, start, index)) {
            path.push(s.substring(start, index + 1))
            // 现在 s 未被分割的部分为 [i+1, n-1]
            dfs(index + 1, index + 1)
            path.pop()  // 恢复现场
        }
    }

    dfs(0, 0)
    return ans
};

function isPalindrome(string, left, right) {
    while (left < right) {
        if (string.charAt(left++) !== string.charAt(right--)) {
            return false
        }
    }
    return true
}
```

### 2.对子串结束位置递归
对结束位置递归，枚举子串的结束位置`i`，判断子串`[start, i]`是否是回文串，如果是回文串，那么就把它加入当前路径`path`中，并继续递归处理剩余的字符串`[i+1, n-1]`。

时间复杂度：O(N*2^N)，其中 N 是字符串 s 的长度。最坏情况下，字符串 s 的每个字符都不同，因此每个字符都可以单独成为一个回文子串，这样就有 2^(N-1) 个分割方案。对于每个分割方案，我们需要 O(N) 的时间来检查每个子串是否是回文的。

```ts
function partition(s: string): string[][] {
    const length = s.length
    const ans: string[][] = []
    const path: string[] = []

    // 现在 s 未被分割的部分为 [i, n-1]
    // 枚举下一刀切在哪
    function dfs(start) {
        // s 分割完毕
        if (start === length) {
            ans.push([...path]) // 复制 path
            return
        }

        // 枚举子串的结束位置
        for (let i = start; i < length; i++) {
            // 判断 [i, j] 是不是回文串
            if (isPalindrome(s, start, i)) {
                path.push(s.substring(start, i + 1))  // 分割！
                // 现在 s 未被分割的部分为 [j+1, n-1]
                dfs(i + 1)
                path.pop()  // 恢复现场
            }
        }
    }

    dfs(0)
    return ans
};

function isPalindrome(string, left, right) {
    while (left < right) {
        if (string.charAt(left++) !== string.charAt(right--)) {
            return false
        }
    }
    return true
}
```

# 138.只出现一次的数字
给你一个 **非空** 整数数组`nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常量额外空间。

示例1：
- **输入**：nums = [2,2,1]
- **输出**：1

示例2：
- **输入**：nums = [4,1,2,1,2]
- **输出**：4

示例3：
- **输入**：nums = [1]
- **输出**：1

## 解题思路
### 1.位运算
根据异或位运算的特性，当进行了相同数字的异或运算后，其效果被抵消，那么用0对输入数组中的每个数字进行异或运算，最终得到的结果就是那个只出现一次的数字。

时间复杂度：O(n)。

空间复杂度：O(1)。

```ts
function singleNumber(nums: number[]): number {
    let ans = 0;
    for (const num of nums) {
        ans ^= num;
    }
    return ans;
};
```

## 总结
位运算常用技巧:
| 用途                     | 代码                                           | 说明                          |
|--------------------------|------------------------------------------------|-------------------------------|
| **判断奇偶**             | `n & 1`                                        | 1为奇数，0为偶数              |
| **取绝对值**             | `x ^ (x >> 31)` 或 `x < 0 ? ~x + 1 : x`        | 针对32位整数                  |
| **交换两个数**           | `a ^= b; b ^= a; a ^= b;`                      | 不使用临时变量                |
| **求2的n次方**           | `1 << n`                                       | 性能远高于 `Math.pow(2, n)`   |
| **判断是否是2的幂**      | `(n & (n-1)) === 0 && n > 0`                   | 非常经典的技巧                |
| **清除最后一位1**        | `n & (n-1)`                                    | 常用于位计数                  |
| **保留最低位的1**        | `n & -n`                                       | 获取最低位1                   |
| **向下取整除2**          | `n >> 1`                                       | 比 `/2` 更快                  |
| **乘以2**                | `n << 1`                                       | 比 `*2` 更快                  |

异或（^）有以下神奇特性：
- 相同的数异或结果为0：`a ^ a = 0`
- 任何数和0异或结果不变：`0 ^ a = a`
- 异或满足交换律和结合律：顺序不影响最终结果

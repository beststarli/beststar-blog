# [509.斐波那契数](https://leetcode.cn/problems/fibonacci-number/description/)
斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：
```txt
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```
给定 n ，请计算 F(n) 。

示例1：
- **输入**：n = 2
- **输出**：1
- **解释**：F(2) = F(1) + F(0) = 1 + 0 = 1

示例2：
- **输入**：n = 3
- **输出**：2
- **解释**：F(3) = F(2) + F(1) = 1 + 1 = 2

示例3：
- **输入**：4
- **输出**：3
- **解释**：F(4) = F(3) + F(2) = 2 + 1 = 3

## 解题思路
### 1.递归
斐波那契数其实就是爬楼梯的底层原理，递推公式`F(n) = F(n - 1) + F(n - 2)`，所以可以使用递归的方式来实现。

- 时间复杂度：O(2^n)，每个节点会产生两个子节点，形成二叉树。
- 空间复杂度：O(n)，递归调用栈的开销。

```ts
function fib(n: number): number {
    const memo = Array(n + 1).fill(0)
    memo[0] = 0
    memo[1] = 1

    function dfs(i) {
        if (i < 0) {
            return 0
        }
        if (memo[i] !== 0) {
            return memo[i]
        }
        return memo[i] = dfs(i - 1) + dfs(i - 2)
    }
    dfs(n)
    return memo[n]
};
```

### 2.递推
直接使用递推的方式来实现，使用一个数组来保存每个斐波那契数的值。注意数组长度为n+2，因为当n=0时，数组长度为2，避免数组越界。

- 时间复杂度：O(n)，只需要遍历一次。
- 空间复杂度：O(n)，需要一个数组来保存每个斐波那契数的值。

```ts
function fib(n: number): number {
    const memo = Array(n + 2).fill(0)
    memo[0] = 0
    memo[1] = 1

    for (let i = 2; i <= n; i++) {
        memo[i] = memo[i - 1] + memo[i - 2]
    }
    return memo[n]
};
```

### 3.空间优化
直接用两个变量在遍历时滚动更新即可，注意依旧是当n=0时，返回0。

- 时间复杂度：O(n)，只需要遍历一次。
- 空间复杂度：O(1)，只需要两个变量来保存前两个斐波那契数的值。

```ts
function fib(n: number): number {
    if (n === 0) {
        return 0
    }

    let memo0 = 0
    let memo1 = 1
    for (let i = 2; i <= n; i++) {
        let memoI = memo0 + memo1
        memo0 = memo1
        memo1 = memoI
    }
    return memo1
};
```

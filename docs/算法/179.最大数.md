# [179.最大数](https://leetcode.cn/problems/largest-number/description/)
给定一组非负整数 nums，重新排列每个数的顺序（每个数不可拆分）使之组成一个最大的整数。

**注意**：输出结果可能非常大，所以你需要返回一个字符串而不是整数。

示例1：
- **输入**：nums = [10,2]
- **输出**："210"

示例2：
- **输入**：nums = [3,30,34,5,9]
- **输出**："9534330"

## 解题思路
### 1.数组内字符串排序
主要用到了数组内置的排序方法，关键要写好排序规则，将两个数转换成字符串进行比较，比较的方式是将两个数拼接成两种不同的字符串进行比较，哪个大就说明哪个数应该排在前面，最后将排序好的数组拼接成字符串输出即可。

:::info sort()方法
sort() 方法用原地算法对数组的元素进行排序，并返回该数组。默认排序顺序是根据字符串Unicode码点进行升序排序。
```ts
arr.sort([compareFunction])
```
- compareFunction 可选。规定排序顺序的函数。如果省略，数组元素将按照转换为字符串后的Unicode码点进行排序。
- return -1: a 应该排在 b 前面
- return 1: a 应该排在 b 后面
- return 0: a 和 b 的位置不变
:::

- 时间复杂度：O(n log n)，其中 n 是数组 nums 的长度。排序算法的时间复杂度通常为 O(n log n)。
- 空间复杂度：O(n)，其中 n 是数组 nums 的长度。

```ts
function largestNumber(nums: number[]): string {
    nums.sort((a, b) => {
        const ab = a.toString() + b.toString()
        const ba = b.toString() + a.toString()
        if (ab > ba) {
            return -1
        } else if (ab < ba) {
            return 1
        } else {
            return 0
        }
    })

    const result = nums.join('')
    return (result[0] !== '0') ? result : '0'
};
```

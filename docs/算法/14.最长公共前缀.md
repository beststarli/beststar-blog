# [14.最长公共前缀](https://leetcode.cn/problems/longest-common-prefix/description/)
编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

示例1：
- **输入**：strs = ["flower","flow","flight"]
- **输出**："fl"

示例2：
- **输入**：strs = ["dog","racecar","car"]
- **输出**：""
- **解释**：输入不存在公共前缀。

## 解题思路
### 1.竖着比较每个字母
首先取出字符串数组中的第一个字符串作为基准字符串，然后逐个比较基准字符串的每个字母与其他字符串对应位置的字母是否相同，如果相同则继续比较下一个字母，如果不同则说明公共前缀到此为止，返回基准字符串从开头到当前字母位置的子串即可。需要注意的是如果当前遍历的字符串的长度和基准字符串目前遍历的下标相同了，说明遍历的字符串要比基准字符串短了，此时也说明公共前缀到此为止，返回基准字符串从开头到当前字母位置的子串即可。

- 时间复杂度：O(n*m)，其中 n 是字符串数组 strs 的长度，m 是字符串数组 strs 中字符串的平均长度。
- 空间复杂度：O(1)，只使用了常数级别的额外空间。

```ts
function longestCommonPrefix(strs: string[]): string {
    const char0 = strs[0]
    for (let i = 0; i < char0.length; i++) {
        for (const char of strs) {
            if (i === char.length || char[i] !== char0[i]) {
                return char0.slice(0, i)
            }
        }
    }
    return char0
};
```

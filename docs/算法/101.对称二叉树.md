# 101.对称二叉树
给你一个二叉树的根节点`root`，检查它是否轴对称。

示例1：
![101-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/101-1.jpg)
- **输入**：root = [1,2,2,3,4,4,3]
- **输出**：true

示例2：
![101-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/101-2.jpg)
- **输入**：root = [1,2,2,null,3,null,3]
- **输出**：false

**进阶**：
- 你可以运用递归和迭代两种方法解决这个问题吗？

## 解题思路
### 1.递归——镜像相同的树
有了[100.相同的树](/docs/算法/相同的树)的解题思路，我们就可以把对称二叉树的问题转化为类似相同的树的问题了。对于一棵二叉树，如果它是轴对称的，那么它的左子树和右子树就是相同的树了。我们可以先比较根节点的值是否相同，再递归地比较左子树的左子树和右子树的右子树是否相同，以及左子树的右子树和右子树的左子树是否相同，如果都相同，那么整棵树就轴对称了。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSymmetric(root: TreeNode | null): boolean {
    return isSameTree(root.left, root.right)
};

// 100.相同的树
function isSameTree(leftTree: TreeNode | null, rightTree: TreeNode | null): boolean {
    if (leftTree === null || rightTree === null) {
        return leftTree === rightTree
    }

    return (leftTree.val === rightTree.val)
        && isSameTree(leftTree.left, rightTree.right)
        && isSameTree(leftTree.right, rightTree.left)
}
```

### 2.迭代——队列
将跟结点的左右子树成对放入队列中，每次从队列中取出一对结点进行比较，如果值不相同则返回false，如果值相同则将这对结点的子结点成对放入队列中，直到队列为空为止，此时说明整棵树是轴对称的。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSymmetric(root: TreeNode | null): boolean {
    if (root === null) return true;

    // 使用队列，成对放入需要比较的节点
    const queue: (TreeNode | null)[] = [];
    queue.push(root.left);
    queue.push(root.right);

    while (queue.length > 0) {
        const left = queue.shift()!;
        const right = queue.shift()!;

        // 两个都为空 → 继续
        if (left === null && right === null) {
            continue;
        }

        // 一个为空或值不相等 → 不对称
        if (left === null || right === null || left.val !== right.val) {
            return false;
        }

        // 按镜像顺序放入下一层要比较的节点
        queue.push(left.left);
        queue.push(right.right);
        queue.push(left.right);
        queue.push(right.left);
    }

    return true;
};
```

### 3.迭代——栈
与队列的解法类似，但是要注意弹出节点的顺序，先弹出右子树，再弹出左子树，这样才能保证成对比较的节点是镜像位置的。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSymmetric(root: TreeNode | null): boolean {
    if (root === null) return true;

    const stack: (TreeNode | null)[] = [];
    stack.push(root.left);
    stack.push(root.right);

    while (stack.length > 0) {
        const right = stack.pop()!;   // 注意弹出顺序
        const left = stack.pop()!;

        if (left === null && right === null) continue;
        if (left === null || right === null || left.val !== right.val) {
            return false;
        }

        stack.push(left.left);
        stack.push(right.right);
        stack.push(left.right);
        stack.push(right.left);
    }

    return true;
};
```

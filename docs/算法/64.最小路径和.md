# 64.最小路径和
给定一个包含非负整数的`m x n`网格`grid`，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明**：每次只能向下或者向右移动一步。

示例1：
![64-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/64-1.jpg)
- **输入**：grid = [[1,3,1],[1,5,1],[4,2,1]]
- **输出**：7
- **解释**：因为路径 1→3→1→1→1 的总和最小。

示例2：
- **输入**：grid = [[1,2,3],[4,5,6]]
- **输出**：12

## 解题思路

### 1.递归加记忆化
定义dfs(i, j)为从(i, j)出发到右下角的最小路径和。对(i, j)分类讨论：
- 如果(i, j)是右下角，那么dfs(i, j) = grid[i][j]。
- 如果(i, j)超出了网格范围，那么dfs(i, j) = +∞。
- 否则，dfs(i, j) = grid[i][j] + min(dfs(i + 1, j), dfs(i, j + 1))。表示在(i, j)向下走和向右走两种选择中，取路径和较小的那一个。

递归入口就是答案dfs(0, 0)。

时间复杂度：$O(mn)$，其中 m 和 n 分别是网格 grid 的行数和列数。

空间复杂度：$O(mn)$，其中 m 和 n 分别是网格 grid 的行数和列数。

```ts
function minPathSum(grid: number[][]): number {
    const height = grid.length
    const width = grid[0].length
    const memo = Array.from({ length: height }, () => Array(width).fill(-1))

    function dfs(i, j) {
        if (i === height || j === width) {
            return Infinity
        }
        if (i === height - 1 && j === width - 1) {
            return grid[i][j]
        }
        if (memo[i][j] !== -1) {
            return memo[i][j]
        }

        memo[i][j] = grid[i][j] + Math.min(dfs(i + 1, j), dfs(i, j + 1))
        return memo[i][j]
    }

    return dfs(0, 0)
};
```

### 2.递推
定义memo[i][j]与dfs(i, j)相反，表示从(i, j)出发到左上角的最小路径和。相应的有：
$$
memo[i][j] = grid[i][j] + \min(memo[i - 1][j], memo[i][j - 1])
$$

注意初始化第一行和第一列的值，因为它们只能向右走或者向下走。

时间复杂度：$O(mn)$，其中 m 和 n 分别是网格 grid 的行数和列数。

空间复杂度：$O(mn)$，其中 m 和 n 分别是网格 grid 的行数和列数。

```ts
function minPathSum(grid: number[][]): number {
    const height = grid.length
    const width = grid[0].length
    const memo = Array.from({ length: height }, () => Array(width).fill(Infinity))
    memo[0][0] = grid[0][0]

    // 初始化第一列
    for (let i = 1; i < height; i++) {
        memo[i][0] = memo[i - 1][0] + grid[i][0]
    }

    // 初始化第一行
    for (let j = 1; j < width; j++) {
        memo[0][j] = memo[0][j - 1] + grid[0][j]
    }

    for (let i = 1; i < height; i++) {
        for (let j = 1; j < width; j++) {
            memo[i][j] = grid[i][j] + Math.min(memo[i - 1][j], memo[i][j - 1])
        }
    }

    return memo[height - 1][width - 1]
};
```

### 3.空间优化
当在计算memo[1][1]时，会用到memo[0][1]，之后就再用不到了，所以可以把memo[1][1]的值直接覆盖到memo[0][1]上。以此类推，空间复杂度降为$O(n)$。

时间复杂度：$O(mn)$，其中 m 和 n 分别是网格 grid 的行数和列数。

空间复杂度：$O(n)$，其中 n 是网格 grid 的列数。

```ts
function minPathSum(grid: number[][]): number {
    const height = grid.length
    const width = grid[0].length
    const memo = Array(width).fill(Infinity)    // memo[j] 表示走到当前行第 j 列的最小路径和
    memo[0] = grid[0][0]    // 初始化第一行

    for (let j = 1; j < width; j++) {
        memo[j] = memo[j - 1] + grid[0][j]
    }

    // 处理剩余的行
    for (let i = 1; i < height; i++) {
        // 更新第一列（只能从上面来）
        memo[0] += grid[i][0]  
        // 更新当前行的其他列
        for (let j = 1; j < width; j++) {
            memo[j] = grid[i][j] + Math.min(memo[j - 1], memo[j])   // memo[j] 之前存的是上一行的值
        }
    }

    return memo[width - 1]
};
```

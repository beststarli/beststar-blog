# [40.组合总和II](https://leetcode.cn/problems/combination-sum-ii/description/)
给定一个候选人编号的集合 `candidates` 和一个目标数 `target` ，找出 `candidates` 中所有可以使数字和为 `target` 的组合。

`candidates` 中的每个数字在每个组合中只能使用 **一次** 。

注意：解集不能包含重复的组合。 



示例1：
- **输入**：candidates = [10,1,2,7,6,1,5], target = 8,
- **输出**：
```
[
[1,1,6],
[1,2,5],
[1,7],
[2,6]
]
```

示例2：
- **输入**：candidates = [2,5,2,1,2], target = 5,
- **输出**：
```
[
[1,2,2],
[5]
]
```

## 解题思路
### 1.排序后回溯并去重
思路和[39.组合总和](/docs/算法/39.组合总和)类似，只是这题有重复元素，而结果数组不允许存在重复的组合，所以我们需要对输入数组进行排序，然后在递归时添加一个判断跳过重复的元素即可。跳过重复元素的判断就是利用哈希set来记录当前层已经使用过的元素，如果当前元素已经在集合中出现过，则跳过该元素。

- 时间复杂度：O(2^n)，其中n是输入数组的长度。回溯算法的时间复杂度通常是指数级的，因为在最坏情况下，算法需要探索所有可能的组合。
- 空间复杂度：O(n)。递归调用栈的深度为n，最坏情况下需要O(n)的空间来存储递归调用栈。

```ts
function combinationSum2(candidates: number[], target: number): number[][] {
    candidates.sort((a, b) => a - b)
    const ans = []
    const path = []
    function dfs(index, sum) {
        if (sum === 0) {
            ans.push([...path])
            return
        }
        const set = new Set()
        for (let i = index; i < candidates.length; i++) {
            if (sum < candidates[i]) {
                break
            }
            if (set.has(candidates[i])) {
                continue
            }
            path.push(candidates[i])
            set.add(candidates[i])
            dfs(i + 1, sum - candidates[i])
            path.pop()
        }
    }
    dfs(0, target)
    return ans
};
```

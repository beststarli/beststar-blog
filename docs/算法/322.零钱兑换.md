# 322.零钱兑换  
给你一个整数数组`coins`，表示不同面额的硬币；以及一个整数`amount`，表示总金额。

计算并返回可以凑成总金额所需的 **最少的硬币个数** 。如果没有任何一种硬币组合能组成总金额，返回`-1`。

你可以认为每种硬币的数量是无限的。

示例1：
- **输入**：coins = [1, 2, 5], amount = 11
- **输出**：3
- **解释**：11 = 5 + 5 + 1

示例2：
- **输入**：coins = [2], amount = 3
- **输出**：-1

## 解题思路
题目说明了每种硬币的数量是无限的，所以这是一个完全背包问题。

### 1.递归加记忆化
依然是递归将问题分解成子问题，当剩余总金额小于当前硬币的面额时，只能不选当前的硬币；否则，对于当前的硬币有选和不选两种情况，选了当前的硬币就将选取硬币个数加1将总金额减去当前硬币的面额；不选当前的硬币就继续选下一个硬币，选取硬币个数不变。这就是递归的入口函数，递归的出口是当总金额为0时，选取硬币个数为0；当没有硬币可选时，如果总金额不为0则返回无穷大。用一个记忆数组来存储已经计算过的子问题的结果，避免重复计算。

时间复杂度：O(n*amount)，其中n是硬币的种类数，amount是总金额。每个子问题最多被计算一次，每次计算需要O(1)的时间。

```ts
function coinChange(coins: number[], amount: number): number {
    const length = coins.length
    const memo = Array.from({ length: length }, () => Array(amount + 1).fill(-1))

    function dfs(i, j) {
        if (i < 0) {
            return (j === 0) ? 0 : Infinity
        }
        if (memo[i][j] !== -1) {
            return memo[i][j]
        }

        let res
        if (j < coins[i]) {
            res = dfs(i - 1, j)
        } else {
            res = Math.min(dfs(i - 1, j), dfs(i, j - coins[i]) + 1)
        }

        memo[i][j] = res
        return res
    }

    const ans = dfs(coins.length - 1, amount)
    return (ans === Infinity) ? -1 : ans
};
```

### 2.递推
递推的思路和递归加记忆化类似，只不过是从底向上计算。定义一个二维数组`memo`，其中`memo[i][c]`表示使用前`i`种硬币凑成总金额`c`所需的最少硬币个数。注意数组长度应为`length + 1`因为需要包含没有硬币可选的情况，由于答案是求最少硬币个数所以数组预先填入`Infinity`。

初始化第一行，当没有硬币可选时，如果总金额不为0则返回无穷大；当总金额为0时，选取硬币个数为0。对于每个硬币，如果当前总金额小于当前硬币的面额，则只能不选当前的硬币；否则，对于当前的硬币有选和不选两种情况，选了当前的硬币就将选取硬币个数加1将总金额减去当前硬币的面额；不选当前的硬币就继续选下一个硬币，选取硬币个数不变。

时间复杂度：O(n*amount)，其中n是硬币的种类数，amount是总金额。每个子问题最多被计算一次，每次计算需要O(1)的时间。

```ts
function coinChange(coins: number[], amount: number): number {
    const length = coins.length
    const memo = Array.from({ length: length + 1 }, () => Array(amount + 1).fill(Infinity))
    memo[0][0] = 0

    for (let i = 0; i < length; i++) {
        for (let c = 0; c <= amount; c++) {
            if (c < coins[i]) {
                memo[i + 1][c] = memo[i][c]
            } else {
                memo[i + 1][c] = Math.min(memo[i][c], memo[i + 1][c - coins[i]] + 1)
            }
        }
    }

    const ans = memo[length][amount]
    return (ans === Infinity) ? -1 : ans
};
```

### 3.空间优化
用一个一维数组来存储状态，从而减少空间复杂度。枚举硬币时，内层循环从当前硬币的面额开始枚举总金额，这样就可以保证在计算`memo[c]`时，`memo[c - coin]`已经是上一次迭代的结果了。

时间复杂度：O(n*amount)，其中n是硬币的种类数，amount是总金额。每个子问题最多被计算一次，每次计算需要O(1)的时间。

```ts
function coinChange(coins: number[], amount: number): number {
    const memo = Array(amount + 1).fill(Infinity)
    memo[0] = 0
    
    for (const coin of coins) {
        for (let c = coin; c <= amount; c++) {
            memo[c] = Math.min(memo[c], memo[c - coin] + 1)
        }
    }

    const ans = memo[amount]
    return (ans === Infinity) ? -1 : ans
};
```

# [LCR 172.统计目标成绩的出现次数](https://leetcode.cn/problems/zai-pai-xu-shu-zu-zhong-cha-zhao-shu-zi-lcof/description/)   
某班级考试成绩按非严格递增顺序记录于整数数组 scores，请返回目标成绩 target 的出现次数。

示例1：
- **输入**：scores = [2, 2, 3, 4, 4, 4, 5, 6, 6, 8], target = 4
- **输出**：3

示例2：
- **输入**：scores = [1, 2, 3, 5, 7, 9], target = 6
- **输出**：0

**注意**：本题与主站 34 题相同（仅返回值不同）：[https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)

## 解题思路
### 1.二分查找
原题给定的是一个非严格递增的数组，那么找到目标值的出现次数也就是找到目标值子数组的长度，需要知道首尾下标，根据二分查找右边界的缩进策略，可以很方便的知道目标值的起点索引，那么怎么得到终止索引？只需要求比目标值大1的数即可，即使数组中不存在这个数，二分查找得到的也是这个数在数组中应插入的位置，也就是目标值终止索引的后一个。做差即可得到目标值的出现次数。

- 时间复杂度：O(logn)，其中 n 是数组 scores 的长度。我们使用二分查找来找到目标值的起始和结束位置。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间来存储变量。

```ts
function findMid(num, target) {
    let left = 0
    let right = num.length - 1
    while (left <= right) {
        const mid = Math.floor((left + right) / 2)
        if (num[mid] < target) {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return left
}

function countTarget(scores: number[], target: number): number {
    const start = findMid(scores, target)
    const end = findMid(scores, target + 1)
    return end - start
};
```

# 56.合并区间
以数组`intervals`表示若干个区间的集合，其中单个区间为`intervals[i] = [starti, endi]`。请你合并所有重叠的区间，并返回*一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间*。

示例1：
- **输入**：intervals = [[1,3],[2,6],[8,10],[15,18]]
- **输出**：[[1,6],[8,10],[15,18]]
- **解释**：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

示例2：
- **输入**：intervals = [[1,4],[4,5]]
- **输出**：[[1,5]]
- **解释**：区间 [1,4] 和 [4,5] 可被视为重叠区间。

示例3：
- **输入**：intervals = [[4,7],[1,4]]
- **输出**：[[1,7]]
- **解释**：区间 [1,4] 和 [4,7] 可被视为重叠区间。

## 解题思路
### 1.暴力双循环
既然是要合并若干个区间，那思路很简单就是先对原数组用升序排序，拿住遍历的第一个区间的左边界不放，去判断右边界是否落在后续区间内，如果有，就将当前右边界与后一区间右边间做比较，取最大值更新右边界，然后重复这个过程；当右边界不落在后续区间内时，则更新左边界的值为后续区间的左边界，继续上述过程。需要注意的是最后一个区间的判定。

时间复杂度：O(n²)

```ts
function merge(intervals: number[][]): number[][] {
    intervals.sort((a,b) => a[0] - b[0])
    const ans = []

    for (let i = 0; i < intervals.length; i++) {
        let left = intervals[i][0]
        let right = intervals[i][1]

        let j = i + 1
        for (; j < intervals.length; j++) {
            
            if (right >= intervals[j][0]) {
                right = Math.max(right, intervals[j][1])
            } else {
                ans.push([left, right])
                i = j - 1
                break
            }
        }

        if (j === intervals.length) {
            ans.push([left, right])
            break
        }
    }

    return ans
};
```

### 2.单循环
对[1.暴力双循环](#1暴力双循环)解法进行优化，对于左边界的维护从循环中剥离出来，依旧动态更新右边界但是用一个变量维护来维护当前合并区间，当满足右边界在后续区间内就将维护的合并区间右边界进行更新；在不满足右边界落在后续区间内的条件时，对当前检查区间即也包括左边界进行更新即可。

时间复杂度：O(nlogn)

```ts
function merge(intervals: number[][]): number[][] {
    intervals.sort((a,b) => a[0] - b[0])
    const ans = []

    let interval = intervals[0]
    for (let i = 1; i < intervals.length; i++) {
        if (interval[1] >= intervals[i][0]) {
            interval[1] = Math.max(interval[1], intervals[i][1])
        } else {
            ans.push(interval)
            interval = intervals[i]
        }
    }

    ans.push(interval)

    return ans
};
```

# 2708.一个小组的最大实力值
给你一个下标从 **0** 开始的整数数组 `nums` ，它表示一个班级中所有学生在一次考试中的成绩。老师想选出一部分同学组成一个 **非空** 小组，且这个小组的 **实力值** 最大，如果这个小组里的学生下标为 `i0`, `i1`, `i2`, ... , `ik` ，那么这个小组的实力值定义为 `nums[i0] * nums[i1] * nums[i2] * ... * nums[ik​]` 。

请你返回老师创建的小组能得到的最大实力值为多少。

示例1：
- **输入**：nums = [3,-1,-5,2,5,-9]
- **输出**：1350
- **解释**：一种构成最大实力值小组的方案是选择下标为 [0,2,3,4,5] 的学生。实力值为 3 * (-5) * 2 * 5 * (-9) = 1350 ，这是可以得到的最大实力值。

示例2：
- **输入**：nums = [-4,-5,-4]
- **输出**：20
- **解释**：选择下标为 [0, 1] 的学生。得到的实力值为 20 。我们没法得到更大的实力值。

## 解题思路
### 1.递推
从左到右遍历nums，考虑nums[i]选或者不选：
- 如果nums[i]不选，那么当前的最大实力值就是之前nums[0]到nums[i-1]的最大实力值。
- 如果nums[i]选，那么当前的最大实力值就是之前nums[0]到nums[i-1]的最大实力值乘以nums[i]，或者之前nums[0]到nums[i-1]的最小实力值乘以nums[i]。两种情况都进行计算，就无需判断nums[i]的正负了。

对于numMin，nums[i]有四种情况：
- 不选，numMin不变。
- nums[i]单独一个数组成子序列
- numMin * nums[i]，如果numMin是正数，可能得到最小乘积
- numMax * nums[i]，如果numMax是负数，可能得到最小乘积

同理，对于numMax，nums[i]有四种情况：
- 不选，numMax不变。
- nums[i]单独一个数组成子序列
- numMin * nums[i]，如果numMin是负数，可能得到最大乘积
- numMax * nums[i]，如果numMax是正数，可能得到最大乘积

整理得：
$$
\begin{aligned}
numMin = \min(numMin, nums[i], numMin \times nums[i], numMax \times nums[i]) \\
numMax = \max(numMax, nums[i], numMin \times nums[i], numMax \times nums[i])
\end{aligned}
$$

初始值为：numMin = nums[0]，numMax = nums[0]。答案是numMax。

时间复杂度：$O(n)$，其中 n 是数组 nums 的长度。

空间复杂度：$O(1)$。

```ts
function maxStrength(nums: number[]): number {
    let numMin = nums[0]
    let numMax = nums[0]

    for (let i = 1; i < nums.length; i++) {
        const num = nums[i]
        const tempMin = numMin
        numMin = Math.min(numMin, num, numMin * num, numMax * num)
        numMax = Math.max(numMax, num, tempMin * num, numMax * num)
    }

    return numMax
};
```

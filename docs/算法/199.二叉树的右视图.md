# 199.二叉树的右视图
给定一个二叉树的根节点`root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

## 解题思路
### 1.层序遍历每层最后一个节点——BFS
观察右视图可以发现，右视图中的节点都是每层最后一个节点，那么我们可以对二叉树进行层序遍历，每层遍历时将该层的最后一个节点值添加到结果数组中。遍历过程参考[102.二叉树的层序遍历](/docs/算法/102.二叉树的层序遍历)。

时间复杂度：O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function rightSideView(root: TreeNode | null): number[] {
    if (root === null) {
        return []
    }

    const ans = []
    let curr = [root]
    while (curr.length) {
        const next = []
        ans.push(curr[curr.length - 1].val)
        for (const node of curr) {
            if (node.left) {
                next.push(node.left)
            }
            if (node.right) {
                next.push(node.right)
            }
        }
        curr = next
    }
    return ans
};
```

### 2.首次到达深度——DFS
先对右树进行递归，再对左树进行递归，那么对于每个深度来说，第一次到达该深度的节点就是右视图中的节点。我们可以维护一个结果数组，当递归到达一个深度时，如果该深度等于结果数组的长度，那么说明这是第一次到达该深度，将该节点值添加到结果数组中。

时间复杂度：O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function rightSideView(root: TreeNode | null): number[] {
    const ans = []

    function dfs(node, depth) {
        if (node === null) {
            return
        }
        if (depth === ans.length) {
            ans.push(node.val)
        }
        dfs(node.right, depth + 1)
        dfs(node.left, depth + 1)
    }
    dfs(root, 0)
    return ans
};
```

## 二刷
### 1.层序遍历每层最后一个节点——BFS
- 时间复杂度：O(n)，其中 n 是二叉树的节点数。
- 空间复杂度：O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function rightSideView(root: TreeNode | null): number[] {
    if (root === null) {
        return []
    }
    const ans = []
    let curr = [root]
    while (curr.length !== 0) {
        const next = []
        const vals = []
        for (const node of curr) {
            vals.push(node.val)
            if (node.left !== null) {
                next.push(node.left)
            }
            if (node.right !== null) {
                next.push(node.right)
            }
        }
        ans.push(vals[vals.length - 1])
        curr = next
    }
    return ans
};
```

## 2.首次到达深度——DFS
- 时间复杂度：O(n)，其中 n 是二叉树的节点数。
- 空间复杂度：O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function rightSideView(root: TreeNode | null): number[] {
    const ans = []
    function dfs(node, depth) {
        if (node === null) {
            return
        }
        if (depth === ans.length) {
            ans.push(node.val)
        }
        dfs(node.right, depth + 1)
        dfs(node.left, depth + 1)
    }
    dfs(root, 0)
    return ans
};
```

# 394.字符串解码
给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。

测试用例保证输出的长度不会超过 $10^5$。

示例1：
- **输入**：s = "3[a]2[bc]"
- **输出**："aaabcbc"

示例2：
- **输入**：s = "3[a2[c]]"
- **输出**："accaccacc"

示例3：
- **输入**：s = "2[abc]3[cd]ef"
- **输出**："abcabccdcdcdef"

示例4：
- **输入**：s = "abc3[cd]xyz"
- **输出**："abccdcdcdxyz"

## 解题思路
### 1.对字符串每个字符递归
对字符串每个字符分类讨论：
- 空串，返回空串
- 首项为字母，向后递归：
    - 子串均为字母，返回子串
    - 子串为数字、字母、括号混合类型，继续递归
- 首项为数字，子串一定包含一对括号：
    - 寻找左括号的下标`i`
    - 寻找与之匹配的右括号下标`j`，注意应忽略内部嵌套括号的右括号
    - 将原字符串分为三部分分析：
        - 由首数字到左括号前，为重复次数
        - 由左括号到右括号，为待重复的子串，继续递归分析
        - 由右括号到字符串末尾，为剩余字符串，继续递归分析

时间复杂度：$O(U^m)$，其中 $U \le 300$ 为 $k$ 的最大值，$m = O\!\left(\dfrac{n}{\log U}\right)$，$n$ 是 $s$ 的长度。

```ts
function decodeString(s: string): string {
    if (s.length === 0) {
        return s
    }

    if (s[0] >= 'a' && s[0] <= 'z') {
        return s[0] + decodeString(s.slice(1))
    }

    const i = s.indexOf('[')
    let balance = 1
    for (let j = i + 1; ; j++) {
        if (s[j] === '[') {
            balance++
        } else if (s[j] === ']') {
            balance--
            if (balance === 0) {
                const k = parseInt(s.slice(0, i))
                return decodeString(s.slice(i + 1, j)).repeat(k) + decodeString(s.slice(j + 1))
            }
        }
    }
};
```

### 2.将字符串拆分
把字符串看作是若干基础字符串和倍数串的组合，基础字符串由字母组成，倍数字符串由数字和括号组成。当遇到数字时更新倍数，遇到左括号向下递归，遇到右括号递归返回。用一个变量`i`来记录当前递归过程遍历到的字符下标。

时间复杂度：$O(U^m)$，其中 $U \le 300$ 为 $k$ 的最大值，$m = O\!\left(\dfrac{n}{\log U}\right)$，$n$ 是 $s$ 的长度。

```ts
function decodeString(s: string): string {
    let i = 0

    function decode() {
        let res = ''
        let k = 0
        while (i < s.length) {
            const c = s[i]
            i++
            if (c >= 'a' && c <= 'z') {
                res += c
            } else if ('0' <= c && c <= '9') {
                k = k * 10 + parseInt(c)
            } else if (c === '[') {
                res += decode().repeat(k)
                k = 0
            } else {
                break
            }
        }
        return res
    }
    return decode()
};
```

### 3.栈模拟递归
将[解法2](/docs/算法/字符串解码#2将字符串拆分)的递归过程用栈来模拟。

在往下「递」的时候，计算机会把当前函数中的局部变量保存到栈中；下层递归函数「归」之后，再从栈中把保存的变量取出。我们可以手动模拟这个过程，具体实现如下：

时间复杂度：$O(U^m)$，其中 $U \le 300$ 为 $k$ 的最大值，$m = O\!\left(\dfrac{n}{\log U}\right)$，$n$ 是 $s$ 的长度。

```ts
function decodeString(s: string): string {
    const stack = []
    let res = ''
    let k = 0

    for (const c of s) {
        if (c >= 'a' && c <= 'z') {
            res += c
        } else if (c >= '0' && c <= '9') {
            k = k * 10 + parseInt(c)
        } else if (c === '[') {
            // 模拟递归
            // 在递归之前，把当前递归函数中的局部变量 res 和 k 保存到栈中
            stack.push([res, k])
            // 递归，初始化 res 和 k
            res = ''
            k = 0
        // c === ']'
        } else {
            // 递归结束，从栈中恢复递归之前保存的局部变量
            const [pre_res, pre_k] = stack.pop()
            // 此时 res 是下层递归的返回值，将其重复 pre_k 次，拼接到递归前的 pre_res 之后
            res = pre_res + res.repeat(pre_k)
        }
    }
    return res
};
```

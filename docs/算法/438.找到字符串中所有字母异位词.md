# 438.找到字符串中所有字母异位词
给定两个字符串`s`和`p`，找到`s`中所有`p`的**异位词**的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

:::info 字母异位词
字母异位词是通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次。
:::

示例1：
- **输入**：s = "cbaebabacd", p = "abc"
- **输出**：[0,6]
- **解释**：
    - 起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
    - 起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。

示例2：
- **输入**：s = "abab", p = "ab"
- **输出**：[0,1,2]
- **解释**：
    - 起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
    - 起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
    - 起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。

## 解题思路
### 1.先剪枝再统计子串字符出现频率
通过理解题干不难发现，这样符合`p`异位词条件的子串有一个特点，就是和`p`拥有相同的字符和对应字符的出现频率而顺序无需考虑，首先想到先用一个`pMap`来统计`p`中拥有的字符和出现频率，对于输出字符串`s`，按照一定的条件将其切分成子串进行剪枝，不难发现切割条件是当遍历到`s`中出现`pMap`不存在的字符则作为边界将前部字符串切割为字串，保留当前切割位置为新子串左边界继续遍历`s`，得到预处理后的子串数组`segments`，数组中存储的每个元素都是一个对象，包含子串本身和其在`s`中的起始索引。对于`segments`再次遍历每个元素，用一个和`p`长度相同的固定大小的窗口去遍历记录每个子串中在当前窗口下出现的字符和其频率存入一个map，当这个map与`pMap`相同，则窗口移动至的索引位置即是目标值，将其填入`result`数组最终返回。

:::warning **耗时极长**
本方法涉及三层for循环，虽可以通过全部测试用例完成解题，但实际耗时达到了**2359ms**，超级慢！！！
![438](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/438.png)
:::

时间复杂度：O(n * m)，m为`p`长度
```ts
function findAnagrams(s: string, p: string): number[] {
    if (s.length < p.length) return []

    const result: number[] = []

    const pMap = new Map<string, number>()
    for (const char of p) {
        pMap.set(char, (pMap.get(char) || 0) + 1)
    }

    const segments: { start: number; substr: string }[] = []
    let currentStart = 0
    let i = 0

    for (i = 0; i < s.length; i++) {
        if (!pMap.has(s[i])) {
            if (i > currentStart) {
                segments.push({
                    start: currentStart,
                    substr: s.substring(currentStart, i)
                });
            }
            currentStart = i + 1
        }
    }

    if (i > currentStart) {
        segments.push({
            start: currentStart,
            substr: s.substring(currentStart, i)
        });
    }
    
    for (const segment of segments) {
        const { start, substr } = segment
        const segLen = substr.length
        if (segLen < p.length) continue

        for (let j = 0; j <= segLen - p.length; j++) {
            const windowStr = substr.substring(j, j + p.length)
            const windowMap = new Map<string, number>()
            for (const char of windowStr) {
                windowMap.set(char, (windowMap.get(char) || 0) + 1)
            }

            let isMatch = true
            if (windowMap.size !== pMap.size) {
                isMatch = false
            } else {
                for (const [char, count] of pMap) {
                    if (windowMap.get(char) !== count) {
                        isMatch = false
                        break
                    }
                }
                for (const [char] of windowMap) {
                    if (!pMap.has(char)) {
                        isMatch = false
                        break
                    }
                }
            }

            if (isMatch) {
                result.push(start + j)
            }
        }
    }

    return result
}
```

## 二刷
### 1.定长滑动窗口
首先用一个长度为26的数组`countArr`来统计`p`中每个字符出现的频率，接着用一个长度同样为26的数组`countStr`来统计`s`中当前窗口下每个字符出现的频率，窗口大小固定为`p`的长度，窗口右边界从0开始向右移动，每次移动时更新`countStr`中对应字符的频率，当窗口大小达到`p`长度时比较两个数组是否相等，如果相等则将当前窗口左边界索引加入结果数组中，最后返回结果数组。

时间复杂度：O(n)，其中n为`s`的长度，比较两个数组是否相等的时间复杂度为O(1)因为数组长度固定为26。

空间复杂度：O(1)，因为使用了两个长度为26的数组来统计字符频率，空间复杂度为常数级别。

```ts
function findAnagrams(s: string, p: string): number[] {
    const countArr = new Array(26).fill(0)
    for (const char of p) {
        countArr[char.charCodeAt(0) - 'a'.charCodeAt(0)]++
    }

    const ans = []
    const countStr = new Array(26).fill(0)
    for (let right = 0; right < s.length; right++) {
        countStr[s[right].charCodeAt(0) - 'a'.charCodeAt(0)]++
        const left = right - p.length + 1
        if (left < 0) {
            continue
        }
        if (isArraysEqual(countStr, countArr)) {
            ans.push(left)
        }
        countStr[s[left].charCodeAt(0) - 'a'.charCodeAt(0)]--
    }
    return ans
};

function isArraysEqual(a: number[], b: number[]): boolean {
    if (a.length !== b.length) return false;
    
    for (let i = 0; i < a.length; i++) {
        if (a[i] !== b[i]) {
            return false;
        }
    }
    return true;
}
```

### 2.不定长滑动窗口
枚举子串的右端点，如果发现子串中一种字母的出现频率超过了`p`中该字母的出现频率，则移动左端点缩小窗口，直到子串中该字母的出现频率不超过`p`中该字母的出现频率为止，如果此时子串的长度等于`p`的长度，则说明找到了一个异位词，将左端点索引加入结果数组中。

内层循环结束后，字符串`s`中每种字母的出现次数都不超过`p`中该字母的出现次数，如果一种字母的出现次数小于`p`中该字母的出现次数，则子串长度必然小于`p`的长度，因此当子串长度等于`p`的长度时，说明找到了一个异位词。

可以把[定长滑动窗口解法](/docs/算法/找到字符串中所有字母异位词#1定长滑动窗口)的`countStr`和`countArr`合并成一个数组`count`：
- 对于`p`中的每个字符，`count`中对应位置的值加1；
- 对于`s`中每个字符，`count`中对应位置的值减1；
- 当`count[char] < 0`，说明窗口中的字母`char`个数超过了`p`中该字母的个数，此时需要移动左端点缩小窗口。

时间复杂度：O(n)，其中n为`s`的长度。

空间复杂度：O(1)，因为使用了一个长度为26的数组来统计字符频率，空间复杂度为常数级别。

```ts
function findAnagrams(s: string, p: string): number[] {
    const count = new Array(26).fill(0)
    for (const char of p) {
        count[char.charCodeAt(0) - 'a'.charCodeAt(0)]++
    }

    const ans = []
    let left = 0
    for (let right = 0; right < s.length; right++) {
        const char = s[right].charCodeAt(0) - 'a'.charCodeAt(0)
        count[char]--
        while (count[char] < 0) {
            count[s[left].charCodeAt(0) - 'a'.charCodeAt(0)]++
            left++
        }
        if (right - left + 1 === p.length) {
            ans.push(left)
        }
    }
    return ans
};
```

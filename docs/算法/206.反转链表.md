# 206.反转链表
给你单链表的头节点`head`，请你反转链表，并返回反转后的链表。

示例1：
![206-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/206-1.jpg)
- **输入**：head = [1,2,3,4,5]
- **输出**：[5,4,3,2,1]

示例2：
![206-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/206-2.jpg)
- **输入**：head = [1,2]
- **输出**：[2,1]

示例3：
- **输入**：head = []
- **输出**：[]

**进阶**：
- 链表可以选用迭代或递归方式完成反转。你能否用两种方法解决这道题？

## 解题思路
### 1.迭代——双指针（头插）
迭代法的核心思路就是用三个指针 prev、cur、temp，每次把 cur.next 指向前一个节点，从而反转方向。

逐步演示：
```md
链表 1 → 2 → 3 → null，目标是 null ← 1 ← 2 ← 3

┌───────┬──────┬──────┬─────────────────┬──────────────────┐
│ 迭代   │ prev │ cur  │  cur.next 指向  │     链表状态       │
├───────┼──────┼──────┼─────────────────┼──────────────────┤
│ 初始   │ null │ 1    │ —               │ 1→2→3→null       │
├───────┼──────┼──────┼─────────────────┼──────────────────┤
│ 第1轮  │ null │ 1    │ cur.next = null │ 1→null，2→3→null │
├───────┼──────┼──────┼─────────────────┼──────────────────┤
│ 第2轮  │ 1    │ 2    │ cur.next = 1    │ 2→1→null，3→null │
├───────┼──────┼──────┼─────────────────┼──────────────────┤
│ 第3轮  │ 2    │ 3    │ cur.next = 2    │ 3→2→1→null       │
├───────┼──────┼──────┼─────────────────┼──────────────────┤
│ 结束   │ 3    │ null │ —               │ 3→2→1→null ↑     │
└───────┴──────┴──────┴─────────────────┴──────────────────┘

为什么返回 prev？循环结束时：
- cur === null（遍历完了）
- prev 指向的是原链表的最后一个节点，也就是反转后的头节点

也可以说：prev 永远是上一个处理过的节点，迭代结束时它就是新的头节点。

另一种角度: 循环结束条件是 cur === null，此时 cur 是 null，而 prev 正好是最后一个被处理的节点，也就是反转后的第一个节点。
```

- 时间复杂度：O(n)，其中 n 是链表的节点数。我们需要遍历链表一次来反转它。
- 空间复杂度：O(1)，我们只使用了常数级别的额外空间来存储指针。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseList(head: ListNode | null): ListNode | null {
    let prev: ListNode | null = null
    let cur: ListNode | null = head

    while (cur !== null) {
        const nextTemp: ListNode | null = cur.next
        cur.next = prev
        prev = cur
        cur = nextTemp
    }

    return prev
};
```

### 2.递归（尾插）
递归的思想就是重复调用函数本身来反转链表。递归法的核心思想是将链表分成两部分：第一部分是当前节点`head`，第二部分是从`head.next`开始的子链表。我们先递归地反转子链表，然后将当前节点的下一个节点指向当前节点本身，最后将当前节点的下一个节点设置为`null`。

- 时间复杂度：O(n)，其中 n 是链表的节点数。我们需要遍历链表一次来反转它。
- 空间复杂度：O(n)，递归调用栈的深度为 n，当链表长度较大时可能会导致栈溢出。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function reverseList(head: ListNode | null): ListNode | null {
    // 递归终止条件：空链表或只有一个节点
    if (head === null || head.next === null) {
        return head;
    }

    // 递归反转后面的部分
    const newHead = reverseList(head.next);

    head.next.next = head; // 把当前节点挂到新链表的末尾
    head.next = null; // 如果不写这行，新链表的末尾两个节点成环，这俩节点互相指向对方

    return newHead;
};
```

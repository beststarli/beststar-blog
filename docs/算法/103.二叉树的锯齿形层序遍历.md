# [103.二叉树的锯齿形层序遍历](https://leetcode.cn/problems/binary-tree-zigzag-level-order-traversal/description/)
给你二叉树的根节点 root ，返回其节点值的 锯齿形层序遍历 。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。


示例1：
![103](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/103.jpg)
- **输入**：root = [3,9,20,null,null,15,7]
- **输出**：[[3],[20,9],[15,7]]

示例2：
- **输入**：root = [1]
- **输出**：[[1]]

示例3：
- **输入**：root = []
- **输出**：[]

## 解题思路
### 1.布尔计数反转
还是用一个当前遍历节点数组curr维护正在遍历的层，以及一个next数组维护下一层的节点，遍历完当前层后，将next赋值给curr，继续遍历下一层。每次遍历完一层后，使用一个布尔值needToReverse来判断当前层的遍历方向，如果是从右往左，则将当前层的节点值反转后再加入结果数组中。注意next数组的维护顺序是从左到右维护下一层的节点，反转只针对当前层的节点值进行反转。

- 时间复杂度：O(n)，其中 n 是二叉树的节点数。每个节点恰好被访问一次。
- 空间复杂度：O(n)，为存储答案所使用的空间。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function zigzagLevelOrder(root: TreeNode | null): number[][] {
    if (root === null) {
        return []
    }
    const ans = []
    let curr = [root]
    let needToReverse = false 

    while (curr.length !== 0) {
        const next = []
        const vals = []
        for (const node of curr) {
            vals.push(node.val)
            if (node.left) {
                next.push(node.left)
            }
            if (node.right) {
                next.push(node.right)
            }
        }
        if (needToReverse) {
            ans.push(vals.reverse())
        } else {
            ans.push(vals)
        }
        curr = next
        needToReverse = !needToReverse
    }
    return ans
};
```

## 总结
本题的主体思路与[102.二叉树的层序遍历](/docs/算法/102.二叉树的层序遍历.md)类似，只是需要在每一层的遍历中，使用一个布尔值来判断当前层的遍历方向，如果是从右往左，则将当前层的节点值反转后再加入结果数组中。

# 102.二叉树的层序遍历
给你二叉树的根节点`root`，返回其节点值的 **层序遍历**。（即逐层地，从左到右访问所有节点）。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的 **长度** 。这条路径可能经过也可能不经过根节点`root`。

两节点之间路径的 **长度** 由它们之间边数表示。

示例1：
![102-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/102-1.jpg)
- **输入**：root = [3,9,20,null,null,15,7]
- **输出**：[[3],[9,20],[15,7]]

示例2：
- **输入**：root = [1]
- **输出**：[[1]]

示例3：
- **输入**：root = []
- **输出**：[]


## 解题思路
### 1.广度优先搜索BFS
注意到题目最后返回的数组内部是扁平化的每层节点值的子数组，这个解法是在对每层节点进行遍历时，将该层的节点存入一个数组，这个数组的长度就是当前层循环添加节点值的次数，每次循环会将当前层节点的子节点添加入一个临时遍历数组维护，当该层遍历结束，将该层节点值数组推入结果数组，并将维护的临时遍历数组更新为下一层遍历数组。

时间复杂度: O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function levelOrder(root: TreeNode | null): number[][] {
    if (root === null) {
        return []
    }

    const ans = []
    let curr = [root]
    while (curr.length) {
        const next = []
        const vals = []
        for (const node of curr) {
            vals.push(node.val)
            if (node.left) {
                next.push(node.left)
            }
            if (node.right) {
                next.push(node.right)
            }
        }
        curr = next
        ans.push(vals)
    }
    return ans
};
```

### 2.队列
力扣支持队列数据结构，用队列来实现广度优先搜索。每次从队列中取出当前层的节点，访问它们的值，并将它们的子节点加入队列。直到队列为空。

时间复杂度: O(n)，其中 n 是二叉树的节点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function levelOrder(root: TreeNode | null): number[][] {
    if (root === null) {
        return []
    }

    const ans = []
    const queue = new Queue()

    queue.enqueue(root)
    while (!queue.isEmpty()) {
        const vals = []
        let length = queue.size()
        while (length--) {
            const node = queue.dequeue() as TreeNode
            vals.push(node.val)
            if (node.left) {
                queue.enqueue(node.left);
            }
            if (node.right) {
                queue.enqueue(node.right);
            }
        }
        ans.push(vals)
    }
    return ans
};
```

# [36.有效的数独](https://leetcode.cn/problems/valid-sudoku/description/)   
请你判断一个 `9 x 9` 的数独是否有效。只需要 **根据以下规则** ，验证已经填入的数字是否有效即可。

数字 `1-9` 在每一行只能出现一次。
数字 `1-9` 在每一列只能出现一次。
数字 `1-9` 在每一个以粗实线分隔的 `3x3` 宫内只能出现一次。（请参考示例图）

**注意**：
- 一个有效的数独（部分已被填充）不一定是可解的。
- 只需要根据以上规则，验证已经填入的数字是否有效即可。
- 空白格用 `'.'` 表示。

示例1：
![36](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/36.png)
- **输入**：
```txt
board = 
[["5","3",".",".","7",".",".",".","."]
,["6",".",".","1","9","5",".",".","."]
,[".","9","8",".",".",".",".","6","."]
,["8",".",".",".","6",".",".",".","3"]
,["4",".",".","8",".","3",".",".","1"]
,["7",".",".",".","2",".",".",".","6"]
,[".","6",".",".",".",".","2","8","."]
,[".",".",".","4","1","9",".",".","5"]
,[".",".",".",".","8",".",".","7","9"]]
```
- **输出**：true

- **输入**：

```txt
board =

```
- **输出**：false
- **解释**：除了第一行的第一个数字从 5 改为 8 以外，空格内其他数字均与 示例1 相同。 但由于位于左上角的 3x3 宫内有两个 8 存在, 因此这个数独是无效的。

## 解题思路
### 1.状态矩阵
针对数独的行列上数字唯一的判断要求很好实现，只需要建立对应的二维数组，rowHas[i][num]表示第i行是否出现过数字num，colHas[j][num]表示第j列是否出现过数字num，即可。

那么3x3宫格的判断要怎么做，其实就是每三行每三列归为一个大格子，可以有i‘ = Math.floor(i / 3)，j‘ = Math.floor(j / 3)，建立三维数组subBoxHas[i'][j'][num]，表示在第i'行第j'列的3x3宫格中是否出现过数字num。

这三个条件只要有一个不符合就返回false。

- 时间复杂度：O(1)，因为数独的大小是固定的9x9，所以时间复杂度是常数级别。
- 空间复杂度：O(1)，同样因为数独的大小是固定的9x9，所以空间复杂度也是常数级别。

```ts
function isValidSudoku(board: string[][]): boolean {
    const rowHas = Array.from({ length: 9 }, () => Array(9).fill(false))
    const colHas = Array.from({ length: 9 }, () => Array(9).fill(false))
    const subBoxHas = Array.from({ length: 3 }, () => Array.from({ length: 3 }, () => Array(9).fill(false)))
    for (let i = 0; i < 9; i++) {
        for (let j = 0; j < 9; j++) {
            const str = board[i][j]
            if (str === '.') {
                continue
            }
            const num = str.charCodeAt(0) - '1'.charCodeAt(0)
            if (rowHas[i][num] || colHas[j][num] || subBoxHas[Math.floor(i / 3)][Math.floor(j / 3)][num]) {
                return false
            }
            rowHas[i][num] = colHas[j][num] = subBoxHas[Math.floor(i / 3)][Math.floor(j / 3)][num] = true
        }
    }
    return true
};
```

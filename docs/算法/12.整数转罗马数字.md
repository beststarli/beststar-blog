# [12.整数转罗马数字](https://leetcode.cn/problems/integer-to-roman/description/)
七个不同的符号代表罗马数字，其值如下：

| 符号 | 值 |
|------|---:|
| I | 1 |
| V | 5 |
| X | 10 |
| L | 50 |
| C | 100 |
| D | 500 |
| M | 1000 |

罗马数字是通过添加从最高到最低的小数位值的转换而形成的。将小数位值转换为罗马数字有以下规则：
- 如果该值不是以 4 或 9 开头，请选择可以从输入中减去的最大值的符号，将该符号附加到结果，减去其值，然后将其余部分转换为罗马数字。
- 如果该值以 4 或 9 开头，使用 减法形式，表示从以下符号中减去一个符号，例如 4 是 5 (`V`) 减 1 (`I`): IV ，9 是 10 (`X`) 减 1 (`I`)：`IX`。-仅使用以下减法形式：4 (`IV`)，9 (`IX`)，40 (`XL`)，90 (`XC`)，400 (`CD`) 和 900 (`CM`)。
- 只有 10 的次方（`I`, `X`, `C`, `M`）最多可以连续附加 3 次以代表 10 的倍数。你不能多次附加 5 (`V`)，50 (`L`) 或 500 (`D`)。如果需要将符号附加4次，请使用 **减法形式**。

给定一个整数，将其转换为罗马数字。

示例1：
- **输入**：num = 3749
- **输出**："MMMDCCXLIX"
- **解释**：
```txt
3000 = MMM 由于 1000 (M) + 1000 (M) + 1000 (M)
 700 = DCC 由于 500 (D) + 100 (C) + 100 (C)
  40 = XL 由于 50 (L) 减 10 (X)
   9 = IX 由于 10 (X) 减 1 (I)
注意：49 不是 50 (L) 减 1 (I) 因为转换是基于小数位
```

示例2：
- **输入**：num = 58
- **输出**："LVIII"

示例3：
- **输入**：num = 1994
- **输出**："MCMXCIV"
- **解释**：
```txt
1000 = M
 900 = CM
  90 = XC
   4 = IV
```

## 解题思路
### 1.哈希表
建立数字与符号的映射关系，遍历数字数组，当输入数字大于等于当前数字时，将对应的符号添加到结果中，并将输入数字减去当前数字，直到输入数字为0。

- 时间复杂度：O(n)，其中 n 是输入数字的大小。我们只需要遍历数字数组。
- 空间复杂度：O(1)，我们只需要常数空间来存储数字与符号的映射关系。

```ts
function intToRoman(num: number): string {
    const vals = [1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1]
    const symbols = ['M', 'CM', 'D', 'CD', 'C', 'XC', 'L', 'XL', 'X', 'IX', 'V', 'IV', 'I']
    let ans = ''
    for (let i = 0; i < vals.length; i++) {
        while (num >= vals[i]) {
            ans += symbols[i]
            num -= vals[i]
        }
    }
    return ans
};
```

# 78.子集
给你一个整数数组`nums`，数组中的元素 **互不相同** 。返回该数组所有可能的子集（幂集）。
:::info **子集**
数组的 **子集** 是从数组中选择一些元素（可能为空）。
:::

解集 **不能** 包含重复的子集。你可以按 **任意顺序** 返回解集。

示例1：
- **输入**：nums = [1,2,3]
- **输出**：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

示例2：
- **输入**：nums = [0]
- **输出**：[[],[0]]

## 解题思路
### 1.记录路径并恢复——DFS
对输入数组中数字的索引做DFS递归，每次递归都将当前路径添加至结果中。递归结束后恢复现场，回退可选项。

时间复杂度：O(n * 2^n)，其中n是输入数组的长度。生成子集的时间复杂度是O(2^n)，每个子集需要O(n)的时间来构建。

```ts
function subsets(nums: number[]): number[][] {
    const ans = []

    function dfs(startIndex: number, path: number[]) {
        // 当startIndex为0，将[]添加至ans中
        ans.push([...path])

        for (let i = startIndex; i < nums.length; i++) {
            path.push(nums[i])
            dfs(i + 1, path)
            path.pop()  // 恢复现场，回退可选项
        }

    }

    dfs(0, [])
    return ans
};
```

## 二刷
### 1.记录路径并恢复——DFS
针对[之前解法](/docs/算法/子集#1记录路径并恢复dfs)优化递归函数传参。

- 时间复杂度：O(n * 2^n)，其中n是输入数组的长度。生成子集的时间复杂度是O(2^n)，每个子集需要O(n)的时间来构建。
- 空间复杂度：O(n)。

```ts
function subsets(nums: number[]): number[][] {
    const ans = []
    const path = []

    function dfs(startIndex) {
        ans.push([...path])
        for (let i = startIndex; i < nums.length; i++) {
            path.push(nums[i])
            dfs(i + 1)
            path.pop()
        }
    }
    dfs(0)
    return ans
};
```

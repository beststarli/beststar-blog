# 2.两数相加
给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例1：
![2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/2.jpg)
- **输入**：l1 = [2,4,3], l2 = [5,6,4]
- **输出**：[7,0,8]
- **解释**：342 + 465 = 807.

示例2：
- **输入**：l1 = [0], l2 = [0]
- **输出**：[]

示例3：
- **输入**：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
- **输出**：[8,9,9,9,0,0,0,1]

## 解题思路
### 1.迭代
要返回相加后的链表，自然想到新建一个哨兵节点，迭代的关键是注意到求和后节点的值是相加两节点的值加上进位的值对10取模，进位的值是相加两节点的值加上进位的值除以10的商。最重要的一点是迭代结束后还要检查一下最后是否还有进位，如果有的话需要再添加一个节点。

时间复杂度：O(max(m, n))，其中 m 和 n 分别是两个链表的长度。我们需要遍历两个链表一次。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function addTwoNumbers(l1: ListNode | null, l2: ListNode | null): ListNode | null {
    let newNode = new ListNode(0)
    let temp = newNode
    let sum = 0

    while (l1 || l2 || sum) {
        if (l1) {
            sum += l1.val
            l1 = l1.next
        }

        if (l2) {
            sum += l2.val
            l2 = l2.next
        }

        temp = temp.next = new ListNode(sum % 10)
        sum = Math.floor(sum / 10)
    }

    return newNode.next    
};
```

### 2.递归
递归的整体思路与迭代类似，关键在于如何处理终止条件和递归关系。递归的终止条件是当两个链表都为空且没有进位时，返回 null。递归关系是当前节点的值等于相加两节点的值加上进位的值对10取模，下一节点等于递归调用相加两节点的下一节点和进位的值除以10的商。

时间复杂度：O(max(m, n))，其中 m 和 n 分别是两个链表的长度。我们需要遍历两个链表一次。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function addTwoNumbers(l1: ListNode | null, l2: ListNode | null): ListNode | null {
    return addTwoNumber(l1, l2, 0)
};

const addTwoNumber = (l1: ListNode | null, l2: ListNode | null, carry: number) => {
    if (!l1 && !l2 && carry === 0) {
        return null
    }

    let sum = carry
    if (l1) {
        sum += l1.val
        l1 = l1.next
    }

    if (l2) {
        sum += l2.val
        l2 = l2.next
    }

    return new ListNode(sum % 10, addTwoNumber(l1, l2, Math.floor(sum / 10)))
}
```

## 总结
递归法的关键是找到递归的边界条件和递归关系。

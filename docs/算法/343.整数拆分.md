# [343.整数拆分](https://leetcode.cn/problems/integer-break/description/)   
给定一个正整数 `n` ，将其拆分为 `k` 个 **正整数** 的和（ `k >= 2` ），并使这些整数的乘积最大化。

返回 *你可以获得的最大乘积* 。

示例1：
- **输入**：n = 2
- **输出**：1
- **解释**：2 = 1 + 1, 1 × 1 = 1。

示例2：
- **输入**：n = 10
- **输出**：36
- **解释**：10 = 3 + 3 + 4, 3 × 3 × 4 = 36。

## 解题思路
### 1.递归加记忆化
既然是对一个数字拆分成多个数字的和，那么可以建立一个长度为 n + 1 的数组 memo，memo[num] 表示将数字 num 拆分成多个数字的和所能得到的最大乘积。对于每个数字 num，我们可以尝试将其拆分成两个数字 i 和 num - i，其中 i 的范围是从 1 到 num - 1。对于每个拆分，我们有两种选择：
1. 不再拆分 num - i，直接计算 i * (num - i)
2. 继续拆分 num - i，计算 i * dfs(num - i)

我们取这两种选择以及当前的最大值做比较，并更新 memo[num]。最终返回 memo[n] 即为所求的最大乘积。

- 时间复杂度：O(n^2)，其中 n 是输入的整数。我们需要计算每个数字的最大乘积，每个数字需要尝试拆分成两个数字的所有可能组合。
- 空间复杂度：O(n)，其中 n 是输入的整数。我们使用了一个大小为 n + 1 的数组来存储每个数字的最大乘积。

```ts
function integerBreak(n: number): number {
    const memo = Array(n + 1).fill(-1)
    function dfs(num) {
        if (num === 1) {
            return 1
        }
        if (memo[num] !== -1) {
            return memo[num]
        }
        let max = 0
        for (let i = 1; i < num; i++) {
            max = Math.max(max, i * (num - i), i * dfs(num - i))
        }
        memo[num] = max
        return memo[num]
    }
    return dfs(n)
};
```

### 2.递推
将递归翻译为递推，注意到题中所给条件n是一个大于等于2的数，为了方便取值在初始化时我们将memo长度设置为n+1，这样的memo[i]代表的就是将数字i拆分成多个数字的和所能得到的最大乘积。对于每个数字i，我们可以尝试将其拆分成两个数字j和i-j，其中j的范围是从1到i-1。对于每个拆分，我们有两种选择：
1. 不再拆分 i - j，直接计算 j * (i - j)
2. 继续拆分 i - j，计算 j * memo[i - j]

我们取这两种选择以及当前的最大值做比较，并更新 memo[i]。注意在初始化时将memo[2]设置为1，因为2只能拆分成1+1，乘积为1。然后外层循环从3开始一直到n，代表的是我们要计算的数字，内层循环从1到i-1，代表的是拆分的第一个数字j。最终返回 memo[n] 即为所求的最大乘积。

- 时间复杂度：O(n^2)，其中 n 是输入的整数。我们需要计算每个数字的最大乘积，每个数字需要尝试拆分成两个数字的所有可能组合。
- 空间复杂度：O(n)，其中 n 是输入的整数。我们使用了一个大小为 n + 1 的数组来存储每个数字的最大乘积。

```ts
function integerBreak(n: number): number {
    const memo = Array(n + 1).fill(-1)
    memo[2] = 1
    for (let i = 3; i <= n; i++) {
        for (let j = 1; j <= i - 1; j++) {
            memo[i] = Math.max(memo[i], j * (i - j), j * memo[i - j])
        }
    }
    return memo[n]
};
```

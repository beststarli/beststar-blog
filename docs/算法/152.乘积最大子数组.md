# 152.乘积最大子数组
给你一个整数数组`nums`，请你找出数组中乘积最大的非空连续 子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

:::info **子数组**
**子数组** 是数组中连续的 **非空** 元素序列。
:::

测试用例的答案是一个 **32-位** 整数。

**请注意**，一个只包含一个元素的数组的乘积是这个元素的值。

示例1：
- **输入**：nums = [2,3,-2,4]
- **输出**：6
- **解释**：子数组 [2,3] 有最大乘积 6。

示例2：
- **输入**：nums = [-2,0,-1]
- **输出**：0
- **解释**：结果不能为 2, 因为 [-2,-1] 不是子数组。

## 解题思路
### 1.端点递推
在遍历nums时，维护两个信息：
- 右端点下标为i的子数组的最大乘积，记为$f_{\max}[i]$。
- 右端点下标为i的子数组的最小乘积，记为$f_{\min}[i]$。

对nums[i]分类讨论：
- nums[i]单独组成子数组，那么$f_{\max}[i] = nums[i]$。
- nums[i]和前面的子数组拼起来，也就是在右端点下标为i-1的乘积最大子数组之后添加nums[i]，那么$f_{\max}[i] = f_{\max}[i-1] \times nums[i]$。也可以在右端点下标为i-1的乘积最小子数组之后添加nums[i]，那么$f_{\max}[i] = f_{\min}[i-1] \times nums[i]$。两种情况都进行计算，就无需判断nums[i]的正负了。

三种情况取最大值，得
$$
f_{\max}[i] = \max(nums[i], f_{\max}[i-1] \times nums[i], f_{\min}[i-1] \times nums[i])
$$

同理，有
$$
f_{\min}[i] = \min(nums[i], f_{\max}[i-1] \times nums[i], f_{\min}[i-1] \times nums[i])
$$

由于以nums[0]为右端点的子数组乘积只能是nums[0]，所以初始值为$f_{\max}[0] = nums[0]$，$f_{\min}[0] = nums[0]$。答案是所有$f_{\max}[i]$的最大值。

时间复杂度：$O(n)$，其中 n 是数组 nums 的长度。

空间复杂度：$O(n)$，其中 n 是数组 nums 的长度。

```ts
function maxProduct(nums: number[]): number {
    const length = nums.length
    const fMax = new Array(length)
    const fMin = new Array(length)

    fMax[0] = fMin[0] = nums[0]
    for (let i = 1; i < length; i++) {
        const x = nums[i]
        fMax[i] = Math.max(fMax[i - 1] * x, fMin[i - 1] * x, x)
        fMin[i] = Math.min(fMax[i - 1] * x, fMin[i - 1] * x, x)
    }

    return Math.max(...fMax)
};
```

### 2.空间优化
由于$f_{\max}[i]$和$f_{\min}[i]$的计算只依赖于$f_{\max}[i-1]$和$f_{\min}[i-1]$，所以可以用两个变量来维护当前的最大乘积和最小乘积，空间复杂度降为$O(1)$。

状态转移方程简化为：
$$
\begin{aligned}
f_{\max} &= \max(f_{\max} \cdot x,\, f_{\min} \cdot x,\, x) \\
f_{\min} &= \min(f_{\max} \cdot x,\, f_{\min} \cdot x,\, x)
\end{aligned}
$$

在实现时，可以初始化$f_{\max}$和$f_{\min}$为1，因为1乘以nums[0]就是以nums[0]为右端点的子数组的乘积。这样可以从下标0开始遍历nums，避免了对下标0的特殊处理。

时间复杂度：$O(n)$，其中 n 是数组 nums 的长度。

空间复杂度：$O(1)$，其中 n 是数组 nums 的长度。


#### 写法一
```ts
function maxProduct(nums: number[]): number {
    let ans = -Infinity // 注意答案可能是负数
    let fMax = 1
    let fMin = 1

    for (const num of nums) {
        const maxX = fMax
        fMax = Math.max(fMax * num, fMin * num, num)
        fMin = Math.min(maxX * num, fMin * num, num)
        ans = Math.max(ans, fMax)
    }
    
    return ans
};
```

#### 写法二
```ts
function maxProduct(nums: number[]): number {
    let ans = -Infinity
    let fMax = 1
    let fMin = 1

    for (const num of nums) {
        if (num < 0) {
            // 下面与 x 相乘后，最大的正数变成最小的负数，最小的负数变成最大的正数
            // 提前交换，这样可以把 x < 0 和 x >= 0 的情况合并，合并后，计算 max 和 min 可以少一项
            [fMax, fMin] = [fMin, fMax]
        }
        fMax = Math.max(fMax * num, num)
        fMin = Math.min(fMin * num, num)
        ans = Math.max(ans, fMax)
    }
    
    return ans
};
```

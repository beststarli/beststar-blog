# 145.二叉树的后序遍历
给你一棵二叉树的根节点 `root` ，返回其节点值的 **后序遍历** 。

示例1：
- **输入**：root = [1,null,2,3]
- **输出**：[3,2,1]
- **解释**：![145-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/145-1.png)

示例2：
- **输入**：root = [1,2,3,4,5,null,8,null,null,6,7,9]
- **输出**：[4,6,7,5,2,9,8,3,1]
- **解释**：![145-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/145-2.png)

示例3：
- **输入**：root = []
- **输出**：[]

示例4：
- **输入**：root = [1]
- **输出**：[1]

## 解题思路
### 1.递归
二叉树后序遍历的顺序是先左子树，再右子树然后再根节点，那么在递归方城内就是先将当前节点的值推入结果数组，然后递归左子树再递归右子树，边界条件是节点为空则返回。

时间复杂度：O(n)，其中n是二叉树的结点数。

空间复杂度：O(n)，其中n是二叉树的结点数，最坏情况下二叉树退化成链表，递归调用栈的深度为n。平均情况下，二叉树的高度为log(n)，递归调用栈的深度为log(n)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function postorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    function dfs(node: TreeNode | null, result: number[]) {
        if (node === null) {
            return
        }

        dfs(node.left, result)
        dfs(node.right, result)
        result.push(node.val)‰
    }
    dfs(root, ans)
    return ans
};
```
### 2.Morris遍历——模拟逆前序+反转
按照[前序Morris遍历](/docs/算法/二叉树的前序遍历#2morris遍历)的思路，镜像前序遍历二叉树，最后将结果数组反转即可。

时间复杂度：O(n)。

空间复杂度：O(1)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function postorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    let cur = root
    while (cur !== null) {
        if (cur.right === null) {
            ans.push(cur.val)
            cur = cur.left
        } else {
            let prev = cur.right
            while (prev.left !== null && prev.left !== cur) {
                prev = prev.left
            }

            if (prev.left === null) {
                prev.left = cur
                ans.push(cur.val)
                cur = cur.right
            } else {
                prev.left = null
                cur = cur.left
            }
        }
    }
    return ans.reverse()
};
```

### 3.迭代——单栈
**面试更推荐使用这个写法**。

时间复杂度：O(n)。

空间复杂度：O(n)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function postorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    const stack = []
    let cur = root
    let lastVisit = null

    while (cur || stack.length > 0) {
        while (cur) {
            stack.push(cur)
            cur = cur.left
        }
        const peek = stack[stack.length - 1]
        if (peek.right === null || peek.right === lastVisit) {
            ans.push(peek.val)
            lastVisit = stack.pop()
        } else {
            cur = peek.right
        }
    }
    return ans
};
```

### 4.迭代——双栈
**双栈写法更容易理解**。

时间复杂度：O(n)。

空间复杂度：O(n)。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function postorderTraversal(root: TreeNode | null): number[] {
    const ans = []
    const stack1 = []
    const stack2 = []
    if (!root) {
        return ans
    }

    stack1.push(root)
    while (stack1.length > 0) {
        const node = stack1.pop()
        stack2.push(node)

        if (node.left) {
            stack1.push(node.left)
        } 
        if (node.right) {
            stack1.push(node.right)
        }
    }

    while (stack2.length > 0) {
        ans.push(stack2.pop().val)
    }

    return ans
};
```

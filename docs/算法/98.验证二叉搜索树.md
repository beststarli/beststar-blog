# 98.验证二叉搜索树
给你一个二叉树的根节点`root`，判断其是否是一个有效的二叉搜索树。

**有效** 二叉搜索树定义如下：
- 节点的左子树只包含 **严格小于** 当前节点的数。
- 节点的右子树只包含 **严格大于** 当前节点的数。
- 所有左子树和右子树自身必须也是二叉搜索树。
:::info **子树**
`treeName`树中的一个节点及其所有子孙节点所构成的树称为`treeName`的 **子树**。
:::

示例1：
![98-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/98-1.jpg)
- **输入**：root = [2,1,3]
- **输出**：true

示例2：
![98-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/98-2.jpg)
- **输入**：root = [5,1,4,null,null,3,6]
- **输出**：false
- **解释**：根节点的值是 5 ，但是右子节点的值是 4 。

## 解题思路
### 1.中序遍历
中序遍历后得到的数组内元素自然呈升序，那么通过对二叉树进行中序遍历，得到一个数组，判断该数组是否为升序即可。对于一个结点来说，左子树的结点值都应该小于该结点的值，右子树的结点值都应该大于该结点的值，那么中序遍历得到的数组就应该是升序的。而对于根节点没有上一个结点，就可以将上一个结点的值初始化为负无穷，保证根节点的值一定大于上一个结点的值。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
function isValidBST(root: TreeNode | null): boolean {
    let pre = -Infinity

    function dfs(node: TreeNode | null) {
        if (node == null) {
            return true
        }
        if (!dfs(node.left)) {
            return false
        }
        if (node.val <= pre) {
            return false
        }
        pre = node.val
        return dfs(node.right)
    }
    return dfs(root)
};
```

### 2.前序遍历
前序遍历时维护一个区间，对于一个结点来说，左子树的结点值都应该小于该结点的值，右子树的结点值都应该大于该结点的值，那么对于左子树来说，区间为`(min, node.val)`，对于右子树来说，区间为`(node.val, max)`。初始时根节点的区间为`(-Infinity, Infinity)`。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isValidBST(root: TreeNode | null, left = -Infinity, right=Infinity): boolean {
    if (root === null) {
        return true 
    }

    const nodeValue = root.val
    return left < nodeValue && nodeValue < right 
            && isValidBST(root.left, left, nodeValue) 
            && isValidBST(root.right, nodeValue, right)
};
```

### 3.后序遍历
与前序遍历类似但方向相反，后序遍历是把区间的判断放在最后进行的，将区间从二叉树底部向上进行维护。

时间复杂度：O(n)，其中n是二叉树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isValidBST(root: TreeNode | null): boolean {
    function dfs(node: TreeNode | null) {
        if (node === null) {
            return [Infinity, -Infinity]
        }
        const [lMin, lMax] = dfs(node.left)
        const [rMin, rMax] = dfs(node.right)
        const nodeValue = node.val

        if (nodeValue <= lMax || nodeValue >= rMin) {
            return [-Infinity, Infinity]
        }
        return [Math.min(lMin, nodeValue), Math.max(rMax, nodeValue)]
    }
    return dfs(root)[1] !== Infinity
};
```

## 二刷
### 1.中序遍历
中序遍历得到的二叉树节点值的数组完全符合二叉搜索树的升序特征，因此我们可以通过中序遍历来判断二叉树是否为二叉搜索树。符合要求的二叉树满足当前节点的值大于前一个遍历节点的值。按照中序遍历的顺序，先对左树进行递归，如果递归结果是false则代表左树已经不满足结果，直接返回false；然后判断当前节点的值，如果比前一个遍历节点要小，则不满足要求也返回false；更新前一节点的值为现节点并递归右子树。因为根节点没有前节点，所以初始化前节点的值pre为-Infinity

- 时间复杂度：O(n)，其中n是二叉树的节点数。
- 空间复杂度：O(n)，其中n是二叉树的节点数。最坏情况下二叉树退化成链表，递归调用栈的深度为n。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isValidBST(root: TreeNode | null): boolean {
    let pre = -Infinity
    function dfs(node) {
        if (node === null) {
            return true
        }
        if (dfs(node.left) === false) {
            return false
        }
        if (node.val <= pre) {
            return false
        }
        pre = node.val
        return dfs(node.right)
    }
    return dfs(root)
};
```

# 74.搜索二维矩阵
给你一个满足下述两条属性的`m x n`整数矩阵： 
- 每行中的整数从左到右按非严格递增顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。

给你一个整数`target`，如果`target`在矩阵中，返回`true`；否则，返回`false`。

示例1：
![74-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/74-1.jpg)
- **输入**：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
- **输出**：true

示例2：
![74-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/74-2.jpg)
- **输入**：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
- **输出**：false

## 解题思路
### 1.二分查找
观察这个二维矩阵可以发现，每行数组元素自然呈递增序列，且每列数据首尾单调递增。如果对其中每行数组首位相接，实际上就是对一个有序数组进行二分查找。问题就变得很简单了。

时间复杂度：O(log(m*n))。

```ts
function searchMatrix(matrix: number[][], target: number): boolean {
    const wholeArr = []
    matrix.forEach(arr => arr.forEach(item => wholeArr.push(item)))

    let left = 0
    let right = wholeArr.length - 1

    while (left <= right) {
        const mid = Math.floor((right - left) / 2 + left)
        if (target < wholeArr[mid]) {
            right = mid - 1
        } else if (target > wholeArr[mid]) {
            left = mid + 1
        } else {
            return true
        }
    }

    return false
};
```

### 2.角点搜索
根据[240.搜索二维矩阵II](/docs/算法/240.搜索二维矩阵II)的解题思路，我们也可以从矩阵的右上角开始搜索，如果当前元素等于目标值，则直接返回 true；如果当前元素大于目标值，则说明目标值不可能在当前列中，应该向左移动；如果当前元素小于目标值，则说明目标值不可能在当前行中，应该向下移动。这样不断地进行比较和移动，直到找到目标值或者超出矩阵的边界。当然从左下角点开始搜索也是可以的。

时间复杂度：O(m+n)，其中 m 和 n 分别是 matrix 的行数和列数。

```ts
function searchMatrix(matrix: number[][], target: number): boolean {
    const height = matrix.length
    const width = matrix[0].length

    let row = 0
    let col = width - 1

    while (row < height && col >= 0) {
        if (matrix[row][col] === target) {
            return true
        }
        if (matrix[row][col] < target) {
            row++
        } else {
            col--
        }
    }
    return false
};
```

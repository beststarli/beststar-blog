# [216.组合总和III](https://leetcode.cn/problems/combination-sum-iii/description/)
找出所有相加之和为 `n` 的 `k` 个数的组合，且满足下列条件：
- 只使用数字1到9
- 每个数字 **最多使用一次**
返回 *所有可能的有效组合的列表* 。该列表不能包含相同的组合两次，组合可以以任何顺序返回。


示例1：
- **输入**：k = 3, n = 7
- **输出**：[[1,2,4]]
- **解释**：
```txt
1 + 2 + 4 = 7   
没有其他符合的组合了。
```

示例2：
- **输入**：k = 3, n = 9
- **输出**：[[1,2,6],[1,3,5],[2,3,4]]
- **解释**：
```txt
1 + 2 + 6 = 9
1 + 3 + 5 = 9
2 + 3 + 4 = 9
没有其他符合的组合了。
```

示例3：
- **输入**：k = 4, n = 1
- **输出**：[]
- **解释**：
```txt
不存在有效的组合。
在[1,9]范围内使用4个不同的数字，我们可以得到的最小和是1+2+3+4 = 10，因为10 > 1，没有有效的组合。
```

## 解题思路
### 1.递归
参考[77.组合](/docs/算法/77.组合)的思路，我们可以使用回溯法来解决这个问题。我们从数字1开始，尝试将每个数字加入当前组合中，并递归地继续寻找下一个数字，直到组合的长度达到k并且组合的和等于n。

```ts
function combinationSum3(k: number, n: number): number[][] {
    const path = []
    const ans = []
    function dfs(index, leftSum) {
        const need = k - path.length
        // 算超或总和小于目标
        if (leftSum < 0 || leftSum > Math.floor((index + (index - need + 1)) * need / 2)) {
            return
        }
        if (need === 0) {
            ans.push([...path])
            return 
        }
        for (let i = index; i >= need; i--) {
            path.push(i)
            dfs(i - 1, leftSum - i) 
            path.pop()
        }
    }
    dfs(9, n)
    return ans
};
```

# 139.单词拆分  
给你一个字符串`s`和一个字符串列表`wordDict`作为字典。如果可以利用字典中出现的一个或多个单词拼接出`s`则返回`true`。

**注意**：不要求字典中出现的单词全部都使用，并且字典中的单词可以重复使用。

示例1：
- **输入**：s = "leetcode", wordDict = ["leet", "code"]
- **输出**：true
- **解释**：返回 true 因为 "leetcode" 可以由 "leet" 和 "code" 拼接成。

示例2：
- **输入**：s = "applepenapple", wordDict = ["apple", "pen"]
- **输出**：true
- **解释**：
```txt
返回 true 因为 "applepenapple" 可以由 "apple" "pen" "apple" 拼接成。
注意，你可以重复使用字典中的单词。
```

示例3：
- **输入**：s = "catsandog", wordDict = ["cats", "dog", "sand", "and", "cat"]
- **输出**：false

## 解题思路
与其去将字典中的单词去拼接字符串`s`，不如将字符串`s`去拆分成字典中的单词。对于字符串`s`的每一个前缀，如果这个前缀在字典中，那么就继续判断剩余的字符串是否也能被字典中的单词拼接成。对于剩余的字符串，我们可以继续使用同样的方法进行判断，直到字符串被完全拆分或者无法拆分为止。

### 1.递归加记忆化
从字符串`s`尾部开始递归判断，如果`s`的前缀在字典中，那么就继续判断剩余的字符串是否也能被字典中的单词拼接成。对于剩余的字符串，我们可以继续使用同样的方法进行判断，直到字符串被完全拆分或者无法拆分为止。可以将字符串字典转成哈希表以加快查询速度。

定义`dfs(length)`，表示能否把前缀`s[:length]`（表示`s[0]`到`s[length−1]`这段子串）划分成若干段，使得每段都在`wordDict`中。

设`wordDict`中字符串的最长长度为`maxLen`，枚举的上限不超过`maxLen`，因为更长的子串必然不在`wordDict`中。枚举`i=length−1,length−2,length−3,…,max(length−maxLen,0)`，只要其中一个`i`满足`s[i:length]`在`wordDict`中且`dfs(i)=true`，那么`dfs(length)`就是`true`.

递归的入口为`dfs(s.length)`，递归边界是`dfs(0)=true`，即递归到空串，`s`被完全拆分了。

时间复杂度：$O(mL + nL^2)$，其中 m 是 wordDict 中单词的数量，L 是 wordDict 中单词的平均长度，n 是字符串 s 的长度。将 wordDict 转成哈希表需要 O(mL) 的时间，递归的时间复杂度是 $O(nL^2)$，因为每个 dfs(length) 最多会枚举 L 个 i，每次枚举需要 O(L) 的时间来判断 s[i:length] 是否在 wordDict 中。

### 2.递推
修改memo[i]的定义与dfs(i)相同，表示能否把前缀s[:i]划分成若干段，使得每段都在wordDict中。递推的入口是memo[0]=true，递推的边界是memo[i] = memo[j] && s[j:i] in wordDict，其中j=0,1,...,i-1。

时间复杂度：$O(mL + nL^2)$，其中 m 是 wordDict 中单词的数量，L 是 wordDict 中单词的平均长度，n 是字符串 s 的长度。将 wordDict 转成哈希表需要 O(mL) 的时间，递推的时间复杂度是 $O(nL^2)$，因为每个 memo[i] 最多会枚举 L 个 j，每次枚举需要 O(L) 的时间来判断 s[j:i] 是否在 wordDict 中。

```ts
function wordBreak(s: string, wordDict: string[]): boolean {
    const memo = Array(s.length + 1).fill(false)
    const maxLength = Math.max(...wordDict.map(word => word.length))
    const words = new Set(wordDict) 
    const length = s.length
    memo[0] = true

    for (let i = 1; i <= length; i++) {
        for (let j = i - 1; j >= Math.max(i - maxLength, 0); j--) {
            if (memo[j] && words.has(s.slice(j, i))) {
                memo[i] = true
                break
            }
        }
    }
    return memo[length]
};
```

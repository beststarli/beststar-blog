# [705.设计哈希集合](https://leetcode.cn/problems/design-hashset/description/)   
不使用任何内建的哈希表库设计一个哈希集合（HashSet）。

实现 `MyHashSet` 类：
- `void add(key)` 向哈希集合中插入值 `key` 。
- `bool contains(key)` 返回哈希集合中是否存在这个值 `key` 。
- `void remove(key)` 将给定值 `key` 从哈希集合中删除。如果哈希集合中没有这个值，什么也不做。

示例1：
- **输入**：
```txt
["MyHashSet", "add", "add", "contains", "contains", "add", "contains", "remove", "contains"]
[[], [1], [2], [1], [3], [2], [2], [2], [2]]
```
- **输出**：[null, null, null, true, false, null, true, null, false]
- **解释**：
```txt
MyHashSet myHashSet = new MyHashSet();
myHashSet.add(1);      // set = [1]
myHashSet.add(2);      // set = [1, 2]
myHashSet.contains(1); // 返回 True
myHashSet.contains(3); // 返回 False ，（未找到）
myHashSet.add(2);      // set = [1, 2]
myHashSet.contains(2); // 返回 True
myHashSet.remove(2);   // set = [1]
myHashSet.contains(2); // 返回 False ，（已移除）
```

## 解题思路
### 1.数组实现
通过数组存放key，用indexOf()不为-1来判断key在数组中是否存在，删除key则使用数组的splice()方法。

- 时间复杂度：O(n)，其中 n 是哈希集合中元素的数量。对于每个操作，最坏情况下需要遍历整个数组来查找元素。
- 空间复杂度：O(n)，其中 n 是哈希集合中元素的数量。

```ts
class MyHashSet {
    private keySet

    constructor() {
        this.keySet = []
    }

    add(key: number): void {
        const has = this.keySet.indexOf(key)
        if (has === -1) {
            this.keySet.push(key)
        }
    }

    remove(key: number): void {
        const has = this.keySet.indexOf(key)
        if (has !== -1) {
            this.keySet.splice(has, 1)
        }
    }

    contains(key: number): boolean {
        const has = this.keySet.indexOf(key)
        if (has !== -1) {
            return true
        } else {
            return false
        }
    }
}

/**
 * Your MyHashSet object will be instantiated and called as such:
 * var obj = new MyHashSet()
 * obj.add(key)
 * obj.remove(key)
 * var param_3 = obj.contains(key)
 */
```

### 2.对象实现
将keySet定义为对象，key作为对象的属性名，value为true或false来表示是否存在该key。当查询是否包含时，存在key没有在keySet中定义的情况，那么此时的this.keySet[key]为undefined，返回false即可。

```ts
class MyHashSet {
    private keySet

    constructor() {
        this.keySet = {}
    }

    add(key: number): void {
        this.keySet[key] = true
    }

    remove(key: number): void {
        this.keySet[key] = false
    }

    contains(key: number): boolean {
        if (this.keySet[key] !== undefined) {
            return this.keySet[key]
        }
        return false
    }
}

/**
 * Your MyHashSet object will be instantiated and called as such:
 * var obj = new MyHashSet()
 * obj.add(key)
 * obj.remove(key)
 * var param_3 = obj.contains(key)
 */
```

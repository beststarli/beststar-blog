# [70.爬楼梯](https://leetcode.cn/problems/climbing-stairs/description/)
假设你正在爬楼梯。需要`n`阶你才能到达楼顶。

每次你可以爬`1`或`2`个台阶。你有多少种不同的方法可以爬到楼顶呢？

示例1：
- **输入**：n = 2
- **输出**：2
- **解释**：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶

示例2：
- **输入**：n = 3
- **输出**：3
- **解释**：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶

## 解题思路
### 1.动态规划
对于求解爬`n`阶楼梯`dp(n)`，可以看成是两部分之和：
- 爬`n-1`阶楼梯后再爬1阶，即`dp(n-1)`。
- 爬`n-2`阶楼梯后再爬2阶，即`dp(n-2)`。

因此可以得到状态转移方程：$dp(n) = dp(n-1) + dp(n-2)$。

在算法中我们创建一个dp数组，dp[i]表示爬i阶楼梯的方法数，初始状态为dp[0]=1（不爬楼梯也是一种方法），dp[1]=1（爬1阶楼梯只有一种方法）。然后根据状态转移方程依次计算出dp[2]到dp[n]的值，最后返回dp[n]。

时间复杂度：O(n)。

```ts
function climbStairs(n: number): number {
    const dp = []
    dp[0] = 1
    dp[1] = 1

    for (let i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2]
    }

    return dp[n]
};
```

## 二刷
### 1.递归
#### 写法一（超时）
根据加法原理，dfs(n) = dfs(n-1) + dfs(n-2)，即爬n阶楼梯的方法数等于爬n-1阶楼梯的方法数加上爬n-2阶楼梯的方法数。递归边界是当楼梯数小于等于1时，只有一种方法。

:::danger **超时！**
无法通过全部测试用例
:::

- 时间复杂度：O(2^n)
- 空间复杂度：O(n)（递归调用栈的深度）

```ts
function climbStairs(n: number): number {
    function dfs(i) {
        if (i <= 1) {
            return 1
        }
        return dfs(i - 1) + dfs(i - 2)
    }
    return dfs(n)
};
```
#### 写法二（递归加记忆化）
用一个memo数组来存储已经计算过的结果，避免重复计算，从而提高效率。注意memo数组的长度要比n大1，因为我们需要存储从0到n的结果。

- 时间复杂度：O(n)
- 空间复杂度：O(n)（递归调用栈的深度 + memo数组的空间）

```ts
function climbStairs(n: number): number {
    const memo = Array(n + 1).fill(0)

    function dfs(i) {
        if (i <= 1) {
            return 1
        }
        if (memo[i] !== 0) {
            return memo[i]
        }
        return memo[i] = dfs(i - 1) + dfs(i - 2)
    }
    return dfs(n)
};
```

### 2.递推
将递归解法二一比一翻译为递推，注意条件初始化，即`memo[0] = memo[1] = 1`，然后从2开始循环计算每一阶楼梯的方法数，直到n阶。

- 时间复杂度：O(n)
- 空间复杂度：O(n)（memo数组的空间）

```ts
function climbStairs(n: number): number {
    const memo = Array(n + 1).fill(0)
    memo[0] = memo[1] = 1

    for (let i = 2; i <= n; i++) {
        memo[i] = memo[i - 1] + memo[i - 2]
    }

    return memo[n]
};
```

### 3.空间优化
事实上如果考虑0级台阶的情况，并将0级台阶的走法视为1的话，那么爬楼梯的递推公式就是斐波那契数列，于是只需两个变量的滚动更新，就可以得到最终答案。

- 时间复杂度：O(n)
- 空间复杂度：O(1)（只使用了两个变量）

```ts
function climbStairs(n: number): number {
    let s0 = 1
    let s1 = 1

    for (let i = 2; i <= n; i++) {
        [s0, s1] = [s1, s0 + s1]
    }
    return s1
};
```

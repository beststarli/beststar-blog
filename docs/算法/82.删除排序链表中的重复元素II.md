# [82.删除排序链表中的重复元素II](https://leetcode.cn/problems/remove-duplicates-from-sorted-list-ii/description/)   
给定一个已排序的链表的头 `head` ， *删除原始链表中所有重复数字的节点，只留下不同的数字* 。返回 *已排序的链表* 。

示例1：
![82-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/82-1.jpg)
- **输入**：head = [1,2,3,3,4,4,5]
- **输出**：[1,2,5]

示例2：
![82-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/82-2.jpg)
- **输入**：head = [1,1,1,2,3]
- **输出**：[2,3]

## 解题思路
### 1.指针
与[83.删除排序链表中的重复元素](/docs/算法/83.删除排序链表中的重复元素.md)相比，此时我们要删除一批重复值的节点，此时是站在p0，去管处p0.next和p0.next.next的值是否相同，如果相同则说明p0.next是重复值的节点，继续循环体删除p0.next，直到p0.next和p0.next.next的值不相同为止，此时说明p0.next是不同的节点，指针后移。所以循环的条件是p0.next和p0.next.next都不为null。

- 时间复杂度：O(n)，其中 n 是链表的节点数。我们只需要遍历链表一次。
- 空间复杂度：O(1)。我们只使用了常数级别的额外空间。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function deleteDuplicates(head: ListNode | null): ListNode | null {
    const dummy = new ListNode(0, head)
    let p0 = dummy
    while (p0.next && p0.next.next) {
        const val = p0.next.val
        if (p0.next.next.val === val) {
            while (p0.next && p0.next.val === val) {
                p0.next = p0.next.next
            }
        } else {
            p0 = p0.next
        }
    }
    return dummy.next
};
```

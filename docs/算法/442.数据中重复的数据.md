# [442.数据中重复的数据](https://leetcode.cn/problems/find-all-duplicates-in-an-array/description/)
给你一个长度为n的整数数组nums，其中nums的所有整数都在范围[1, n]内，且每个整数出现最多两次。请你找出所有出现两次的整数，并以数组形式返回。

示例1：
- **输入**：nums = [4,3,2,7,8,2,3,1]
- **输出**：[2,3]

示例2：
- **输入**：nums = [1,1,2]
- **输出**：[1]

示例3：
- **输入**：nums = [1]
- **输出**：[]

## 解题思路
### 1.
注意到题中条件：数组长度是n,整数范围在[1, n]内，且每个整数出现最多两次。我们可以利用数组下标来标记数字是否出现过。具体做法是遍历数组，对于每个数字num，计算其对应的下标index = abs(num) - 1，然后将nums[index]取负数来标记num已经出现过。如果nums[index]已经是负数，说明num已经出现过一次，那么我们就将num加入结果数组中。

- 时间复杂度：O(n)，其中n是数组的长度。我们只需要遍历数组一次。
- 空间复杂度：O(1)，不考虑返回结果的空间开销。

```ts
function findDuplicates(nums: number[]): number[] {
    const ans = []
    for (const num of nums) {
        const index = Math.abs(num) - 1
        if (nums[index] < 0) {
            ans.push(Math.abs(num))
        } else {
            nums[index] = -nums[index]
        }
    }
    return ans
};
```

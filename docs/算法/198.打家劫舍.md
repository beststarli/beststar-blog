# 198.打家劫舍
你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警**。

给定一个代表每个房屋存放金额的非负整数数组，计算你 **不触动警报装置的情况下** ，一夜之内能够偷窃到的最高金额。

示例1：
- **输入**：[1,2,3,1]
- **输出**：4
- **解释**：
```txt
偷窃 1 号房屋 (金额 = 1) ，然后偷窃 3 号房屋 (金额 = 3)。
偷窃到的最高金额 = 1 + 3 = 4 。
```

示例2：
- **输入**：[2,7,9,3,1]
- **输出**：12
- **解释**：
```txt
偷窃 1 号房屋 (金额 = 2), 偷窃 3 号房屋 (金额 = 9)，接着偷窃 5 号房屋 (金额 = 1)。
偷窃到的最高金额 = 2 + 9 + 1 = 12 。
```

## 解题思路
### 1.递归并记忆化
递归就是在考虑选择当前的情况下，去继续寻求下一步的最优解，一步一步将问题变小。那么对于当前的房屋，我们有两种选择：
- 选择偷窃当前的房屋，那么就不能偷窃相邻的房屋了，下一步就要考虑偷窃当前房屋的下两间房屋了。即`dfs(i - 2) + nums[i]`。
- 选择不偷窃当前的房屋，那么就可以偷窃相邻的房屋了，下一步就要考虑偷窃当前房屋的下一间房屋了。即`dfs(i - 1)`。

时间复杂度：O(n)。

```ts
function rob(nums: number[]): number {
    const length = nums.length
    const stolen = Array(length).fill(-1)   // -1代表未计算过

    function dfs(i) {
        // 没有房屋了
        if (i < 0) {
            return 0
        }
        // 偷窃过，直接返回结果
        if (stolen[i] !== -1) {
            return stolen[i]
        }
        // 偷窃当前房屋，或者不偷窃当前房屋，取两者的最大值
        const res = Math.max(dfs(i - 1), dfs(i - 2) + nums[i])
        stolen[i] = res // 记忆化
        return res
    }

    return dfs(length - 1)  // 从最后一个房屋开始考虑
};
```

### 2.递推
直接将[解法1](/docs/算法/打家劫舍#1递归并记忆化)的`dfs(i)`翻译成递推`stolen[i]`，但是`dfs()`会访问到`-1`和`-2`的下标，所以stolen数组需要比nums数组多两个元素。

时间复杂度：O(n)。

```ts
function rob(nums: number[]): number {
    const length = nums.length
    const stolen = Array(length + 2).fill(0)

    for (let i = 0; i < length; i++) {
        stolen[i + 2] = Math.max(stolen[i + 1], stolen[i] + nums[i])
    }

    return stolen[length + 1]
};
```

### 3.滚动数组
[解法2](/docs/算法/打家劫舍#2递推)中`stolen[i]`只和`stolen[i - 1]`和`stolen[i - 2]`有关，所以可以用两个变量来代替整个数组。

时间复杂度：O(n)。

空间复杂度：O(1)。

```ts
function rob(nums: number[]): number {
    let stolen0 = 0
    let stolen1 = 0

    for (const num of nums) {
        [stolen0, stolen1] = [stolen1, Math.max(stolen1, stolen0 + num)]
    }
    return stolen1
};
```

## 二刷
### 1.递归加记忆化
时间复杂度：O(n)。

空间复杂度：O(n)。

```ts
function rob(nums: number[]): number {
    const length = nums.length
    const stolen = Array(length).fill(-1)

    function dfs(i) {
        if (i < 0) {
            return 0
        }
        if (stolen[i] !== -1) {
            return stolen[i]
        }
        stolen[i] = Math.max(dfs(i - 1), dfs(i - 2) + nums[i])
        return stolen[i]
    }
    return dfs(length - 1)
};
```

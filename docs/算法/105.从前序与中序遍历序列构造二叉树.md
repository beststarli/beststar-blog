# 105.从前序与中序遍历序列构造二叉树
给定两个整数数组`preorder`和`inorder`，其中`preorder`是二叉树的先序遍历，`inorder`是同一棵树的中序遍历，请构造二叉树并返回其根节点。

示例1：
![105-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/105-1.jpg)
- **输入**：preorder = [3,9,20,15,7], inorder = [9,3,15,20,7]
- **输出**：[3,9,20,null,null,15,7]

示例2：
- **输入**：preorder = [-1], inorder = [-1]
- **输出**：[-1]

## 解题思路
### 1.递归建树
根据前序遍历和中序遍历的特点可以知道，前序遍历的第一个元素就是树的根节点，然后是左子树的节点，最后是右子树的节点。而中序遍历则是先左子树的节点，再根节点，最后是右子树的节点。也就是：
- 前序遍历：根节点 -> 左子树 -> 右子树
- 中序遍历：左子树 -> 根节点 -> 右子树

因此，我们可以通过前序遍历的第一个元素找到并建立根节点，然后在中序遍历中找到根节点的索引，这个索引向前就是左子树的节点，向后就是右子树的节点。接着，根据得到的左右子树在原前序遍历和中序遍历中的范围作为左右子树的前序遍历和中序遍历数组，递归构建左右子树即可。

时间复杂度：O(n²)，其中n是树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function buildTree(preorder: number[], inorder: number[]): TreeNode | null {
    if (preorder.length === 0 || inorder.length === 0) {
        return null
    }
    
    const rootVal = preorder[0]
    const root = new TreeNode(rootVal)

    const rootIndex = inorder.indexOf(rootVal)

    root.left = buildTree(
        preorder.slice(1, rootIndex + 1),
        inorder.slice(0, rootIndex)
    )

    root.right = buildTree(
        preorder.slice(rootIndex + 1),
        inorder.slice(rootIndex + 1)
    )

    return root
};
```

### 2.哈希表优化
用哈希map来优化节点索引的查找，避免每次都调用`indexOf`方法，从而将时间复杂度优化到O(n)。

时间复杂度：O(n)，其中n是树的结点数。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function buildTree(preorder: number[], inorder: number[]): TreeNode | null {
    const map = new Map()
    inorder.forEach((val, index) => map.set(val, index))

    function dfs(preStart, preEnd, inStart, inEnd) {
        if (preStart > preEnd || inStart > inEnd) {
            return null
        }

        const rootVal = preorder[preStart]
        const root = new TreeNode(rootVal)
        const rootIndex = map.get(rootVal)

        const leftSize = rootIndex - inStart

        root.left = dfs(preStart + 1, preStart + leftSize, inStart, rootIndex - 1)
        root.right = dfs(preStart + leftSize + 1, preEnd, rootIndex + 1, inEnd)

        return root
    }
    return dfs(0, preorder.length - 1, 0, inorder.length - 1)
};
```

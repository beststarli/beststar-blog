# [188.买卖股票的最佳时机IV](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/description/)   
给你一个整数数组 `prices` 和一个整数 `k` ，其中 `prices[i]` 是某支给定的股票在第 `i` 天的价格。

设计一个算法来计算你所能获取的最大利润。你最多可以完成 `k` 笔交易。也就是说，你最多可以买 `k` 次，卖 `k` 次。

**注意**：你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。


示例1：
- **输入**：k = 2, prices = [2,4,1]
- **输出**：2
- **解释**：在第 1 天 (股票价格 = 2) 的时候买入，在第 2 天 (股票价格 = 4) 的时候卖出，这笔交易所能获得利润 = 4-2 = 2 。

示例2：
- **输入**：k = 2, prices = [3,2,6,5,0,3]
- **输出**：7
- **解释**：
```txt
在第 2 天 (股票价格 = 2) 的时候买入，在第 3 天 (股票价格 = 6) 的时候卖出, 这笔交易所能获得利润 = 6-2 = 4 。
随后，在第 5 天 (股票价格 = 0) 的时候买入，在第 6 天 (股票价格 = 3) 的时候卖出, 这笔交易所能获得利润 = 3-0 = 3 。
```

## 解题思路
### 1.递归加记忆化
j表示当前至多可以交易的次数。由于最后未持有股票，手上的股票一定会卖掉，所以代码中的j-1可以写在买股票的时候，也可以写在卖股票的时候，这两种写法都是可以的。

**注意**：买入加卖出才算一笔交易，不是两笔交易。

建立三维数组memo[i][j][hold]，表示在第i天，至多可以交易j次，当前是否持有股票hold的情况下的最大利润。递归函数的边界是：
- 当j<0时，返回负无穷，表示交易次数不够。
- 当i<0时，如果当前持有股票，返回负无穷，表示不合法；如果当前不持有股票，返回0，表示没有利润。
- 如果memo[i][j][hold]不为Infinity，说明已经计算过，直接返回memo[i][j][hold]。

递归函数的状态转移方程是：
- 当hold为true时，表示当前持有股票，有两种选择：
  1. 不操作，继续持有股票，利润为dfs(i - 1, j, 1)
  2. 卖出股票，利润为dfs(i - 1, j - 1, 0) + prices[i]
- 当hold为false时，表示当前不持有股票，有两种选择：
  1. 不操作，继续不持有股票，利润为dfs(i - 1, j, 0)
  2. 买入股票，利润为dfs(i - 1, j, 1) - prices[i]

- 时间复杂度：O(n*k)
- 空间复杂度：O(n*k)

```ts
function maxProfit(k: number, prices: number[]): number {
    const memo = Array.from({ length: prices.length }, () => Array(k + 1).fill(null).map(() => [Infinity, Infinity]))
    function dfs(i, j, hold) {
        if (j < 0) {
            return -Infinity
        }
        if (i < 0) {
            return hold ? -Infinity : 0
        }
        if (memo[i][j][hold] !== Infinity) {
            return memo[i][j][hold]
        }
        if (hold) {
            memo[i][j][hold] = Math.max(dfs(i - 1, j, 1), dfs(i - 1, j - 1, 0) - prices[i])
        } else {
            memo[i][j][hold] = Math.max(dfs(i - 1, j, 0), dfs(i - 1, j, 1) + prices[i])
        }
        return memo[i][j][hold]
    }
    dfs(prices.length - 1, k, 0)
    return memo[prices.length - 1][k][0]
};
```

### 2.递推
将递归翻译为递推，建立三维数组memo[i][j][hold]，表示在第i天，至多可以交易j次，当前是否持有股票hold的情况下的最大利润。相比递归，递推数组要多一行，表示第0天的情况。多一列，表示交易次数为0的情况。初始化memo[0][j][0] = 0，表示第0天不持有股票的利润为0。

将递归函数的状态转移方程翻译为递推公式：
- 当hold为true时，表示当前持有股票，有两种选择：
  1. 不操作，继续持有股票，利润为memo[i - 1][j][1]
  2. 前一天买入股票，利润为memo[i - 1][j - 1][0] - prices[i]
- 当hold为false时，表示当前不持有股票，有两种选择：
  1. 不操作，继续不持有股票，利润为memo[i - 1][j][0]
  2. 前一天卖出股票，利润为memo[i - 1][j][1] + prices[i]

- 时间复杂度：O(n*k)
- 空间复杂度：O(n*k)

```ts
function maxProfit(k: number, prices: number[]): number {
    const memo = Array.from({ length: prices.length + 1}, () => Array(k + 2).fill(null).map(() => [-Infinity, -Infinity]))
    for (let j = 1; j < k + 2; j++) {
        memo[0][j][0] = 0
    }
    for (let i = 0; i < prices.length; i++) {
        for (let j = 1; j < k + 2; j++) {
            memo[i + 1][j][0] = Math.max(memo[i][j][0], memo[i][j][1] + prices[i])
            memo[i + 1][j][1] = Math.max(memo[i][j][1], memo[i][j - 1][0] - prices[i])
        }
    }
    return memo[prices.length][k + 1][0]
};
```

### 3.空间优化
当天的收益情况只与前一天的收益情况有关，所以可以将三维数组优化为二维数组，减少空间复杂度。

- 时间复杂度：O(n*k)
- 空间复杂度：O(k)

```ts
function maxProfit(k: number, prices: number[]): number {
    const memo = Array(k + 2).fill(null).map(() => [-Infinity, -Infinity])
    for (let j = 1; j < k + 2; j++) {
        memo[j][0] = 0
    }
    for (const price of prices) {
        for (let j = k + 1; j > 0; j--) {
            memo[j][0] = Math.max(memo[j][0], memo[j][1] + price)
            memo[j][1] = Math.max(memo[j][1], memo[j - 1][0] - price)
        }
    }
    return memo[k + 1][0]
};
```

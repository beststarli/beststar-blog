# 33.搜索旋转排序数组
整数数组 `nums` 按升序排列，数组中的值 **互不相同** 。

在传递给函数之前，`nums` 在预先未知的某个下标 `k`（`0 <= k < nums.length`）上进行了 **向左旋转**，使数组变为`[nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]`（下标 **从 0 开始** 计数）。例如，`[0,1,2,4,5,6,7]`下标`3`上向左旋转后可能变为`[4,5,6,7,0,1,2]`。

给你 **旋转后** 的数组`nums`和一个整数`target`，如果`nums`中存在这个目标值`target`，则返回它的下标，否则返回`-1`。

你必须设计一个时间复杂度为`O(log n)`的算法解决此问题。

示例1：
- **输入**：nums = [4,5,6,7,0,1,2], target = 0
- **输出**：4

示例2：
- **输入**：nums = [4,5,6,7,0,1,2], target = 3
- **输出**：-1

示例3：
- **输入**：nums = [1], target = 0
- **输出**：-1

## 解题思路
### 1.分界后二分查找
本题所谓的升序数组左旋，其实就可以看作是对两个升序数组查找唯一`target`值，而两个子数组的分界线正好是原数组的最大值与最小值，那么我只需要构造一个辅助函数`findMin`来寻找分解线，判断目标值落在哪一个子数组内，再使用二分查找即可。

时间复杂度：O(log n)。

```ts
function search(nums: number[], target: number): number {
    const min = findMin(nums)
    if (target > nums[nums.length - 1]) {
        return lowerBound(nums, target, 0, min - 1)
    } else {
        return lowerBound(nums, target, min, nums.length - 1)
    }
};

function lowerBound(nums, target, left, right) {
    while (left <= right) {
        const mid = Math.floor((right + left) / 2)
        if (nums[mid] === target) {
            return mid
        } else if (nums[mid] < target) {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}

function findMin(nums) {
    let left = 0
    let right = nums.length - 1

    while (left < right) {
        const mid = Math.floor((left + right) / 2)
        if (nums[mid] <= nums[nums.length - 1]) {
            right = mid
        } else {
            left = mid + 1
        }
    }
    return left
}
```

### 2.一次二分查找完成
对于旋转排序数组，从中抽取一个数`x`与`nums[n - 1]`比较大小，有两种情况：
- 如果`x > nums[n - 1]`，说明`x`在左侧子数组中，左侧段所有元素都大于`nums[n - 1]`，右侧段所有元素都小于`nums[n - 1]`。
- 如果`x <= nums[n - 1]`，说明`x`在右侧子数组中，左侧段所有元素都大于`nums[n - 1]`，右侧段所有元素都小于`nums[n - 1]`。

时间复杂度：O(log n)。

#### 2.1.写法一
取`x = nums[mid]`与`target`的位置关系进行判断：
- 如果`x`和`target`在同一侧，则直接比较`x`和`target`的大小关系进行二分查找。
- 如果`x`和`target`不在同一侧：
    - 如果`target`在第一段，`x`在第二段，则`target`在`x`的左侧
    - 如果`x`在第一段，`target`在第二段，则`target`在`x`的右侧

```ts
function search(nums: number[], target: number): number {
    const last = nums[nums.length - 1]
    let left = -1
    let right = nums.length - 1
    while (left + 1 < right) {
        const mid = Math.floor((left + right) / 2)
        const x = nums[mid]
        // target 在第一段，x 在第二段
        if (target > last && x <= last) {
            right = mid     // 下轮循环去左边找
        // x 在第一段，target 在第二段
        } else if (x > last && target <= last) {
            left = mid      // 下轮循环去右边找
        // 否则，x 和 target 在同一段，这就和方法一的 lowerBound 一样了
        } else if (x >= target) {
            right = mid
        } else {
            left = mid
        }
    }
    return (nums[right] === target) ? right : -1
};
```

#### 2.2.写法二
只讨论target在x左边，或是target = x的情况，其他情况下target在x右边：
- 如果`x > nums[n−1]`，说明`x`在第一段中，那么`target`也必须在第一段中（否则`target`一定在`x`的右边）且`x`必须大于等于`target`。
- 如果`x≤nums[n−1]`，说明`x`在第二段中（或者`nums`只有一段），那么`target`可以在第一段，也可以在第二段：
    - 如果`target`在第一段，那么`target`一定在`x`的左边。
    - 如果`target`在第二段，那么`x`必须大于等于`target`。

```ts
function search(nums: number[], target: number): number {
    const last = nums[nums.length - 1]
    function check(index) {
        const x = nums[index]
        if (x > last) {
            return target > last && x >= target
        }
        return target > last || x >= target
    }

    // 开区间 (-1, n-1)
    let left = -1
    let right = nums.length - 1
    
    // 开区间不为空
    while (left + 1 < right) {
        const mid = Math.floor((left + right) / 2)
        if (check(mid)) {
            right = mid
        } else {
            left = mid
        }
    }

    return nums[right] === target ? right : -1
};
```

## 二刷
### 1.分界再二分查找
对于target可与数组最后一个值比大小，如果大则位于数组分界前半部分，反之位于数组分界后半部分。先用辅助函数findMin()找到数组中最小值的下标，再根据target与数组最后一个值的大小关系，确定target所在的区间，最后在该区间内使用二分查找。

- 时间复杂度：O(log n)。
- 空间复杂度：O(1)。

```ts
function findMin(arr: number[]) {
    let minIndex = 0
    for (let i = 0; i < arr.length; i++) {
        minIndex = (arr[minIndex] < arr[i]) ? minIndex : i
    }
    return minIndex
}

function lowerBounds(arr: number[], target: number, left, right) {
    while (left <= right) {
        const mid = Math.floor((left + right) / 2)
        if (arr[mid] < target) {
            left = mid + 1
        } else if (arr[mid] > target) {
            right = mid - 1
        } else {
            return mid
        }
    }
    return -1
}

function search(nums: number[], target: number): number {
    const minIndex = findMin(nums)
    if (target > nums[nums.length - 1]) {
        return lowerBounds(nums, target, 0, minIndex - 1)
    } else {
        return lowerBounds(nums, target, minIndex, nums.length - 1)
    }
};
```

### 2.一次二分查找完成
循环主体分三种情况讨论：
- 如果中间值等于目标值，直接返回中间值下标。
- 如果中间值大于等于数组最左端的值，说明中间值在第一段中：
    - 如果目标值在数组最左端和中间值之间，说明目标值在第一段中，右边界移动到中间值下标。
    - 否则，说明目标值在第二段中，左边界移动到中间值下标。
- 如果中间值小于数组最左端的值，说明中间值在第二段中：
    - 如果目标值在中间值和数组最右端之间，说明目标值在第二段中，左边界移动到中间值下标。
    - 否则，说明目标值在第一段中，右边界移动到中间值下标。

关于判据的等号什么时候加，最外层判据保证左闭右开，内部判据与外部判据保持一致即可：即走前半部分则左闭右开，走后半部分则左开右闭。

- 时间复杂度：O(log n)。
- 空间复杂度：O(1)。

```ts
function search(nums: number[], target: number): number {
    let left = 0
    let right = nums.length - 1
    while (left <= right) {
        const mid = Math.floor((left + right) / 2)
        if (nums[mid] === target) {
            return mid
        } 
        if (nums[mid] >= nums[left]) {
            if (target >= nums[left] && target < nums[mid]) {
                right = mid - 1
            } else {
                left = mid + 1
            }
        } else {
            if (target > nums[mid] && target <= nums[right]) {
                left = mid + 1
            } else {
                right = mid - 1
            }
        }
    }
    return -1
}
```

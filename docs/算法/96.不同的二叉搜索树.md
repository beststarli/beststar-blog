# [96.不同的二叉搜索树](https://leetcode.cn/problems/unique-binary-search-trees/description/)   
给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

示例1：
![96](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/96.jpg)
- **输入**：n = 3
- **输出**：5

示例2：
- **输入**：n = 1
- **输出**：1

## 解题思路
### 1.递推
本题的关键是要知道二叉树的总数 = 左子树的总数 * 右子树的总数，这个公式。建立递推数组memo，memo[i]表示由i个节点组成的二叉搜索树的总数。对于每个节点i，我们可以将其作为根节点，然后左子树的节点数为j，右子树的节点数为i - 1 - j，其中j的范围是从0到i - 1。对于每个拆分，我们有两种选择：
1. 左子树的节点数为j，右子树的节点数为i - 1 - j，计算memo[j] * memo[i - 1 - j]
2. 若有一支子树节点个树为0，则另一支子树的节点个数为i - 1，计算memo[i - 1]即可。
3. 将所有可能的拆分的结果相加，得到memo[i]。

- 时间复杂度：O(n^2)，其中 n 是输入的整数。我们需要计算每个数字的二叉搜索树的总数，每个数字需要尝试拆分成两个数字的所有可能组合。
- 空间复杂度：O(n)，其中 n 是输入的整数。我们使用了一个大小为 n + 1 的数组来存储每个数字的二叉搜索树的总数。

```ts
function numTrees(n: number): number {
    const memo = Array(n + 1).fill(0)
    memo[1] = 1
    for (let i = 2; i <= n; i++) {
        for (let j = 0; j <= i - 1; j++) {
            const leftNum = j
            const rightNum = i - 1 - j
            if (leftNum === 0 || rightNum === 0) {
                memo[i] += memo[leftNum]
                memo[i] += memo[rightNum]
            } else {
                memo[i] += memo[leftNum] * memo[rightNum]
            }
        }
    }
    return memo[n]
};
```

## 总结
根据卡特兰数递推公式：
```txt
二叉树的总种类数字 = 左子树种类树 * 右子树种类数
```
左子树的种类数和右子树的种类数都是由卡特兰数递推公式计算出来的。

# [66.加一](https://leetcode.cn/problems/plus-one/description/)
给定一个表示 **大整数** 的整数数组 `digits`，其中 `digits[i]` 是整数的第 `i` 位数字。这些数字按从左到右，从最高位到最低位排列。这个大整数不包含任何前导 `0`。

将大整数加 1，并返回结果的数字数组。

示例1：
- **输入**：digits = [1,2,3]
- **输出**：[1,2,4]
- **解释**：
```txt
输入数组表示数字 123。
加 1 后得到 123 + 1 = 124。
因此，结果应该是 [1,2,4]。
```

示例2：
- **输入**：digits = [4,3,2,1]
- **输出**：[4,3,2,2]
- **解释**：
```txt
输入数组表示数字 4321。
加 1 后得到 4321 + 1 = 4322。
因此，结果应该是 [4,3,2,2]。
```

示例3：
- **输入**：digits = [9]
- **输出**：[1,0]
- **解释**：
```txt
输入数组表示数字 9。
加 1 得到了 9 + 1 = 10。
因此，结果应该是 [1,0]。
```

## 解题思路
### 1.进位数携带
理解题意发现其实就是当加1后数字大于9时需要进位，进位后当前位变为0，继续向前一位加1，直到没有进位或者到达数组首位。那么就可以用一个变量carry来表示进位，初始值为1，表示加1。然后从数组末尾开始遍历，每一位加上carry，如果大于9则当前位变为0，carry继续为1，否则当前位为当前位加上carry，carry变为0。最后如果carry不为0，则需要在数组首位插入1。

- 时间复杂度：O(n)，其中 n 是数组的长度。每个元素恰好被访问一次。
- 空间复杂度：O(n)，其中 n 是数组的长度。空间主要为结果数组的开销。

```ts
function plusOne(digits: number[]): number[] {
    const ans = []
    let carry = 1
    for (let i = digits.length - 1; i >= 0; i--) {
        const num = digits[i] + carry
        const val = num % 10
        carry = Math.floor(num / 10)
        ans.push(val)
    }
    (carry !== 0) && ans.push(carry)
    return ans.reverse()
};
```

# 24.两两交换链表中的节点
给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

示例1：
![24](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/24.jpg)
- **输入**：head = [1,2,3,4]
- **输出**：[2,1,4,3]

示例2：
- **输入**：head = []
- **输出**：[]

示例3：
- **输入**：head = [1]
- **输出**：[1]

## 解题思路
### 1.迭代
解法参考自[灵茶山艾府](https://leetcode.cn/problems/swap-nodes-in-pairs/solutions/2374872/tu-jie-die-dai-di-gui-yi-zhang-tu-miao-d-51ap/?envType=study-plan-v2&envId=top-100-liked)。
![24-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/24-1.jpg)

迭代还是建立守卫结点，用三个结点来维护当前结点的顺序，分别是要交换的两个结点以及两个结点后的一个结点。每次交换完后，将守卫结点的头指向交换后两个结点的后一个结点，继续交换下一对结点。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function swapPairs(head: ListNode | null): ListNode | null {
    const newNode = new ListNode(0, head)
    let node0 = newNode
    let node1 = head

    while (node1 && node1.next) {
        const node2 = node1.next
        const node3 = node2.next

        node0.next = node2
        node2.next = node1
        node1.next = node3

        node0 = node1
        node1 = node3
    }
    return newNode.next
};
```

### 2.递归
其实在写迭代解法时就可以发现，while循环部分就是可以用递归来实现的。递归的思路是交换当前结点和下一个结点，然后把下一个结点的下一个结点作为参数传入递归函数，继续交换下一对结点。

时间复杂度：O(n)，其中 n 是链表的节点数。

```ts
/**
 * Definition for singly-linked list.
 * class ListNode {
 *     val: number
 *     next: ListNode | null
 *     constructor(val?: number, next?: ListNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.next = (next===undefined ? null : next)
 *     }
 * }
 */

function swapPairs(head: ListNode | null): ListNode | null {
    if (head === null || head.next === null) {
        return head;
    }

    const node1 = head;
    const node2 = head.next;
    const node3 = node2.next;

    node1.next = swapPairs(node3); // 1 指向递归返回的链表头
    node2.next = node1; // 2 指向 1

    return node2; // 返回交换后的链表头节点
};
```

## 总结
递归题目的解法无论是使用迭代还是递归求解，关键还是要把题目最小化，以小见大，找到递归关系。

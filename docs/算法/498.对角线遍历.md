# [498.对角线遍历](https://leetcode.cn/problems/diagonal-traverse/description/)
给你一个大小为`m x n`的矩阵`mat`，请以对角线遍历的顺序，用一个数组返回这个矩阵中的所有元素。

示例1：
![498](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/498.png)
- **输入**：mat = [[1,2,3],[4,5,6],[7,8,9]]
- **输出**：[1,2,4,7,5,3,6,8,9]

示例2：
- **输入**：mat = [[1,2],[3,4]]
- **输出**：[1,2,3,4]

## 解题思路
### 1.奇偶分情况处理
这题的技巧和[51.N皇后](/docs/算法/51.N皇后.md)相似，遍历的对角线上的元素，他们的行列下标之和是相同的，因此我们可以通过`k = i + j`来确定当前遍历的是哪一条对角线。
- 当`k`为偶数时，说明当前对角线是从右上到左下的方向遍历的，因此我们需要从`j = min(k, n - 1)`开始，向左下遍历，直到`i = min(k, m - 1)`。
- 当`k`为奇数时，说明当前对角线是从左下到右上的方向遍历的，因此我们需要从`i = min(k, m - 1)`开始，向右上遍历，直到`j = min(k, n - 1)`。

- 时间复杂度：O(m * n)，其中 m 和 n 分别是矩阵的行数和列数。我们需要遍历矩阵中的每个元素一次。
- 空间复杂度：O(1)，不考虑返回结果所使用的空间。

```ts
function findDiagonalOrder(mat: number[][]): number[] {
    const m = mat.length
    const n = mat[0].length
    const ans = []

    for (let k = 0; k < m + n - 1; k++) {
        const minJ = Math.max(k - (m - 1), 0)
        const maxJ = Math.min(k, n - 1)
        if (k % 2 === 0) {
            for (let j = minJ; j <= maxJ; j++) {
                ans.push(mat[k - j][j])
            }
        } else {
            for (let j = maxJ; j >= minJ; j--) {
                ans.push(mat[k - j][j])
            }
        }
    }
    return ans
};
```

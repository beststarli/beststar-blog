# [572.另一棵树的子树](https://leetcode.cn/problems/subtree-of-another-tree/description/)
给你两棵二叉树 `root` 和 `subRoot` ，检验 `root` 中是否包含和 `subRoot` 具有相同结构和节点值的子树。如果存在，返回 `true` ；否则，返回 `false` 。

二叉树 `tree` 的一棵子树包括 `tree` 的某个节点和这个节点的所有后代节点。`tree` 也可以看做它自身的一棵子树。

示例1：
![572-1](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/572-1.jpg)
- **输入**：root = [3,4,5,1,2], subRoot = [4,1,2]
- **输出**：true

示例2：
![572-2](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/572-2.jpg)
- **输入**：root = [3,4,5,1,2,null,null,null,null,0], subRoot = [4,1,2]
- **输出**：false

## 解题思路
### 1.全部递归
借助[100.相同的树](/docs/算法/100.相同的树)的解法，来判断两棵树是否相同。只要遇到 `root` 中的某个节点和 `subRoot` 相同，就返回 `true`，否则继续递归判断 `root` 的左子树和右子树。

- 时间复杂度：O(n * m)，其中n是root的节点数，m是subRoot的节点数。
- 空间复杂度：O(n + m)，递归需要O(n + m)的栈开销。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSameTree(p: TreeNode | null, q: TreeNode | null) {
    if (p === null || q === null) {
        return p === q
    }
    const left = isSameTree(p.left, q.left)
    const right = isSameTree(p.right, q.right)
    return p.val === q.val && left && right
}

function isSubtree(root: TreeNode | null, subRoot: TreeNode | null): boolean {
    if (root === null) {
        return false
    }
    return isSameTree(root, subRoot) || isSubtree(root.left, subRoot) || isSubtree(root.right, subRoot) 
};
```

### 2.递归优化——深度相同再判断
借助[104.二叉树的最大深度](/docs/算法/104.二叉树的最大深度)的解法，先计算 `subRoot` 的深度，然后在 `root` 中递归查找深度相同的节点，再判断是否是相同的树。

- 时间复杂度：O(n * m)，其中n是root的节点数，m是subRoot的节点数。
- 空间复杂度：O(n + m)，递归需要O(n + m)的栈开销。

```ts
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     val: number
 *     left: TreeNode | null
 *     right: TreeNode | null
 *     constructor(val?: number, left?: TreeNode | null, right?: TreeNode | null) {
 *         this.val = (val===undefined ? 0 : val)
 *         this.left = (left===undefined ? null : left)
 *         this.right = (right===undefined ? null : right)
 *     }
 * }
 */

function isSameTree(p: TreeNode | null, q: TreeNode | null) {
    if (p === null || q === null) {
        return p === q
    }
    const left = isSameTree(p.left, q.left)
    const right = isSameTree(p.right, q.right)
    return p.val === q.val && left && right
}

function maxDepth(root: TreeNode | null): number {
    let ans = 0
    function dfs(node, depth) {
        if (node === null) {
            return
        }
        depth++
        ans = Math.max(ans, depth)
        dfs(node.left, depth)
        dfs(node.right, depth)
    }
    dfs(root, 0)
    return ans
};

function isSubtree(root: TreeNode | null, subRoot: TreeNode | null): boolean {

    const subDepth = maxDepth(subRoot)
    let found = false;

    function dfs(node: TreeNode | null): number {
        if (node === null || found) return 0;

        const leftDepth = dfs(node.left);
        const rightDepth = dfs(node.right);
        const depth = Math.max(leftDepth, rightDepth) + 1;

        // 只有当 node 的深度和 subRoot 深度相等时才比较
        if (depth === subDepth && isSameTree(node, subRoot)) {
            found = true;
        }

        return depth;
    }
    dfs(root)
    return found
};
```

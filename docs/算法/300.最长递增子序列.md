# 300.最长递增子序列  
给你一个整数数组 `nums` ，找到其中最长严格递增子序列的长度。

**子序列** 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，`[3,6,2,7]` 是数组 `[0,3,1,6,2,2,7]` 的**子序列**。

:::info **子序列**
**子序列** 是可以通过从另一个数组删除或不删除某些元素，但不更改其余元素的顺序得到的数组。
:::

示例1：
- **输入**：nums = [10,9,2,5,3,7,101,18]
- **输出**：4
- **解释**：最长递增子序列是 [2,3,7,101]，因此长度为 4 。

示例2：
- **输入**：nums = [0,1,0,3,2,3]
- **输出**：4

示例3：
- **输入**：nums = [7,7,7,7,7,7,7]
- **输出**：1

**进阶**：
- 你能将算法的时间复杂度降低到 `O(n log(n))` 吗?

## 解题思路
### 1.递归加记忆化
定义`dfs(i)`，表示以`nums[i]`结尾的最长递增子序列的长度。枚举`j=0,1,...,i-1`，如果`nums[j] < nums[i]`，那么`dfs(i)`可以由`dfs(j) + 1`转移而来。对原数组中的每个元素都调用一次`dfs(i)`，最终的答案是所有`dfs(i)`的最大值。

时间复杂度：$O(n^2)$，其中 n 是数组 nums 的长度。

```ts
function lengthOfLIS(nums: number[]): number {
    let ans = 0
    const length = nums.length
    const memo = Array(length).fill(0)
    
    for (let i = 0; i < length; i++) {
        ans = Math.max(ans, dfs(i, nums, memo))
    }
    return ans
};

function dfs(i, nums, memo) {
    if (memo[i] > 0) {
        return memo[i]
    }

    let res = 0
    for (let j = 0; j < i; j++) {
        if (nums[j] < nums[i]) {
            res = Math.max(res, dfs(j, nums, memo))
        }
    } 
    res++
    return memo[i] = res
}
```

### 2.递推
还是一比一翻译，用`memo[i]`表示以`nums[i]`结尾的最长递增子序列的长度。递推的入口是`memo[0]=1`，递推关系是`memo[i] = Math.max(memo[i], memo[j])`，其中`j=0,1,...,i-1`且`nums[j] < nums[i]`。`ans`是所有`memo[i] + 1`的最大值。

时间复杂度：$O(n^2)$，其中 n 是数组 nums 的长度。

```ts
function lengthOfLIS(nums: number[]): number {
    let ans = 0
    const length = nums.length
    const memo = Array(length).fill(0)

    for (let i = 0; i < length; i++) {
        for (let j = 0; j < i; j++) {
            if (nums[j] < nums[i]) {
                memo[i] = Math.max(memo[i], memo[j])
            }
        }
        ans = Math.max(ans, ++memo[i])
    }
    return ans
};
```

### 3.贪心+二分查找
这个方法是对[解法2](/docs/算法/最长递增子序列#2递推)的优化。

举例说明：如果`memo[3]`和`memo[4]`的值都是`2`，但是nums[3] = 9999，nums[4] = 9。现在要计算memo[5]，要满足`nums[j] < nums[i]`，使用nums[4]更合适，nums[3]虽然也满足条件，但是memo[3]和memo[4]相等，也就是在更新memo[5]时效果相同，那么nums[3]就是无用数据，总体来说，对于相同的memo值，只需保留更小的nums[i]即可。

定义`g[i]`表示`memo[j] = i + 1`，`nums[j]`的最小值就是长为`i + 1`的上升子序列的末尾元素的最小值。`+ 1`是因为`memo[j]`是从1开始的，而`g[0]`对应的不是`memo[0]`，而是`memo[j] = 1`。

时间复杂度：$O(n \log n)$，其中 n 是数组 nums 的长度。

#### 额外空间
```ts
function lengthOfLIS(nums: number[]): number {
    const g = []
    for (const num of nums) {
        const j = lowerBound(g, num) 
        if (j == g.length) {
            g.push(num)
        } else {
            g[j] = num
        }
    }
    return g.length
};

// 闭区间写法
function lowerBound(g, target) {
    let left = 0
    let right = g.length - 1

    while (left <= right) {
        const mid = left + Math.floor((right - left) / 2)
        if (g[mid] < target) {
            left = mid + 1  // 缩小范围[mid + 1, right]
        } else {
            right = mid - 1  // 缩小范围[left, mid - 1]
        }
    }
    return left
}
```

#### 原地修改
```ts
function lengthOfLIS(nums: number[]): number {
    let gLength = 0
    for (const num of nums) {
        const j = lowerBound(nums, gLength - 1, num) 
        nums[j] = num
        if (j == gLength) {
            gLength++
        }
    }
    return gLength
};

function lowerBound(nums, right, target) {
    let left = 0

    while (left <= right) {
        const mid = left + Math.floor((right - left) / 2)
        if (nums[mid] < target) {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return left
}
```

# [125. 验证回文串](https://leetcode.cn/problems/valid-palindrome/description/)
如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个 **回文串** 。

字母和数字都属于字母数字字符。

给你一个字符串 `s`，如果它是 **回文串** ，返回 `true` ；否则，返回 `false` 。

示例1：
- **输入**：s = "A man, a plan, a canal: Panama"
- **输出**：true
- **解释**："amanaplanacanalpanama" 是回文串。

示例2：
- **输入**：s = "race a car"
- **输出**：false
- **解释**："raceacar" 不是回文串

示例3：
- **输入**：s = " "
- **输出**：true
- **解释**：在移除非字母数字字符之后，s 是一个空字符串 "" 。由于空字符串正着反着读都一样，所以是回文串。

## 解题思路
### 1.双指针——奇偶分开处理
根据题目要求先对字符串做处理，去掉非字母数字字符并将大写字母转换为小写字母。然后根据字符串长度的奇偶性分开处理，奇数长度的回文串中间的字符不需要比较，偶数长度的回文串则需要比较中间两个字符。

- 时间复杂度：O(n)，其中 n 是字符串数组的长度。
- 空间复杂度：O(n)，需要额外的空间来存储处理后的字符串。

```ts
function isPalindrome(s: string): boolean {
    const strArr = []
    for (let i = 0; i < s.length; i++) {
        const char = s[i]
        if (char >= 'A' && char <= 'Z') {
            strArr.push(char)
        }
        if (char >= 'a' && char <= 'z') {
            strArr.push(char)
        }
        if (char >= '0' && char <= '9') {
            strArr.push(char)
        }
    }

    const str = strArr.join('').toLowerCase()
    const strLength = str.length

    if (str === '') {
        return true
    }

    // 处理长度为偶数
    if (strLength % 2 === 0) {
        let left = Math.floor(strLength / 2) - 1
        let right = Math.floor(strLength / 2)
        while (str[left] === str[right]) {
            if (left === 0) {
                return true
            }
            left--
            right++
        }
    // 处理长度为奇数
    } else {
        let left = Math.floor(strLength / 2)
        let right = Math.floor(strLength / 2)
        while (str[left] === str[right]) {
            if (left === 0) {
                return true
            }
            left--
            right++
        }
    }
    return false
};
```

### 2.双指针——直接处理
对于字符串的处理可以直接在双指针的过程中进行，左指针从字符串的开头开始，右指针从字符串的结尾开始，两个指针向中间移动，直到两个指针相遇。每次移动时，如果左指针指向的字符不是字母数字字符，则左指针向右移动；如果右指针指向的字符不是字母数字字符，则右指针向左移动；如果两个指针都指向字母数字字符，则比较它们是否相等，如果不相等则返回 false。

- 时间复杂度：O(n)，其中 n 是字符串数组的长度。
- 空间复杂度：O(1)，只需要常数空间来存储指针和临时变量。

```ts
function isPalindrome(s: string): boolean {
    let left = 0
    let right = s.length - 1
    
    function isValid(char) {
        return (char >= 'A' && char <= 'Z') ||
            (char >= 'a' && char <= 'z') ||
            (char >= '0' && char <= '9')
    }

    while (left < right) {
        if (!isValid(s[left])) {
            left++
        } else if (!isValid(s[right])) {
            right--
        } else if (s[left].toLowerCase() === s[right].toLowerCase()) {
            left++
            right--
        } else {
            return false
        }
    }
    return true
};
```

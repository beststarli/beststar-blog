# 9.回文数
给你一个整数`x`，如果`x`是一个回文整数，返回`true`；否则，返回`false`。

**回文数** 是指正序（从左向右）和倒序（从右向左）读都是一样的整数。
:::info **回文数**
如果一个整数向前和向后读都相同，则它是一个 回文数。 例如，`121`是回文数，而`123`不是。
:::

- 例如，`121`是回文，而`123`不是。

示例1：
- **输入**：x = 121
- **输出**：true

示例2：
- **输入**：x = -121
- **输出**：false
- **解释**：从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

示例3：
- **输入**：x = 10
- **输出**：false
- **解释**：从右向左读, 为 01 。因此它不是一个回文数。

**进阶**：
- 你能不将整数转为字符串来解决这个问题吗？

## 解题思路
### 1.双指针
根据回文数的定义，想到使用双指针向中间靠拢的方式来判断是否为回文数。首先需要将整数转为字符串，然后使用两个指针分别指向字符串的首尾，比较指针所指向的字符是否相同，如果不同则返回`false`，如果相同则继续向中间靠拢，直到两个指针相遇或者交错，此时说明该整数是回文数，返回`true`。

时间复杂度：O(n)，其中 n 是整数 x 的位数。

```ts
function isPalindrome(x: number): boolean {
    if (x < 0) {
        return false 
    }

    const xString = x.toString()
    const xArr = xString.split('')

    let left = 0
    let right = xArr.length - 1

    while (left < right) {
        if (xArr[left] !== xArr[right]) {
            return false
        } else {
            left++
            right--
            continue
        }
    }
    return true
};
```

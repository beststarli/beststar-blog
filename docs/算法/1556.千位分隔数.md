# [1556.千位分隔数](https://leetcode.cn/problems/thousand-separator/description/)
给你一个整数n，请你每隔三位添加点（即 "." 符号）作为千位分隔符，并将结果以字符串格式返回。


示例1：
![103](https://blog-1385521233.cos.ap-guangzhou.myqcloud.com/docs/algorithm/103.jpg)
- **输入**：root = [3,9,20,null,null,15,7]
- **输出**：[[3],[20,9],[15,7]]

示例2：
- **输入**：root = [1]
- **输出**：[[1]]

示例3：
- **输入**：root = []
- **输出**：[]

## 解题思路
### 1.计数插值
将输入数字转换为字符串，然后从右往左遍历字符串，用一个量来计数，利用一个数组来维护结果，每隔三位插入一个点（"."）作为千位分隔符并重置计数。最后将结果数组反转合并后返回。注意处理长度刚好为3的情况，防止在头部插入"."。

- 时间复杂度：O(n)，其中 n 是数字 n 的位数。我们需要遍历数字的每一位。
- 空间复杂度：O(n)，为存储结果字符串所使用的空间。

```ts
function thousandSeparator(n: number): string {
    const nStr = n.toString()
    const ans = []
    let count = 0
    for (let i = nStr.length - 1; i >= 0; i--) {
        ans.push(nStr[i])
        count++
        if (count % 3 === 0 && nStr.length > 2 && i !== 0) {
            count = 0
            ans.push('.')
        }
    }
    return ans.reverse().join('')
};
```

### 2.空间优化
对于解法一，直接操作字符串本身，不使用额外数组空间来维护结果。对字符串每间隔三个字符进行更新，同时字符串长度加一。

- 时间复杂度：O(n)，其中 n 是数字 n 的位数。我们需要遍历数字的每一位。
- 空间复杂度：O(1)，不使用额外的空间来存储结果字符串。

```ts
function thousandSeparator(n: number): string {
    let str = n.toString()
    let length = str.length
    for (let i = length - 3; i > 0; i -= 3, length++) {
        str = str.substring(0, i) + '.' + str.substring(i, length)
    }
    return str
};
```

### 3.正则表达式
直接使用正则表达式来匹配每隔三位的数字，并在其前面插入点（"."）作为千位分隔符。

- 时间复杂度：O(n)，其中 n 是数字 n 的位数。我们需要遍历数字的每一位。
- 空间复杂度：O(1)，不使用额外的空间来存储结果字符串。

```ts
function thousandSeparator(n: number): string {
    // 正向零宽断言: (?=p)
    // 反向零宽断言: (?!p)
    // 可同时支持多个断言，"且" 关系
    // 如: (?=p1)(?!p2) 是既满足 p1 模式，也要不满足 p2 模式
    return String(n).replace(/(?!^)(?=(\d{3})+$)/g, '.');
};
```

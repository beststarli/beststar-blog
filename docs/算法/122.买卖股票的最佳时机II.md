# [122.买卖股票的最佳时机II](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-ii/description/)   
给你一个整数数组 `prices` ，其中 `prices[i]` 表示某支股票第 `i` 天的价格。

在每一天，你可以决定是否购买和/或出售股票。你在任何时候 **最多** 只能持有 一股 股票。然而，你可以在 **同一天** 多次买卖该股票，但要确保你持有的股票不超过一股。

返回 *你能获得的 **最大** 利润* 。

示例1：
- **输入**：prices = [7,1,5,3,6,4]
- **输出**：7
- **解释**：
```txt
在第 2 天（股票价格 = 1）的时候买入，在第 3 天（股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4。
随后，在第 4 天（股票价格 = 3）的时候买入，在第 5 天（股票价格 = 6）的时候卖出, 这笔交易所能获得利润 = 6 - 3 = 3。
最大总利润为 4 + 3 = 7 。
```

示例2：
- **输入**：prices = [1,2,3,4,5]
- **输出**：4
- **解释**：
```txt
在第 1 天（股票价格 = 1）的时候买入，在第 5 天 （股票价格 = 5）的时候卖出, 这笔交易所能获得利润 = 5 - 1 = 4。
最大总利润为 4 。
```

示例3：
- **输入**：prices = [7,6,4,3,1]
- **输出**：0
- **解释**：在这种情况下, 交易无法获得正利润，所以不参与交易可以获得最大利润，最大利润为 0。

## 解题思路
### 1.递归加记忆化
对于一天来说，我们有两种状态，持有股票和不持有股票。我们可以使用一个二维数组 `memo` 来存储每一天的最大利润，其中 `memo[i][0]` 表示第 `i` 天不持有股票的最大利润，`memo[i][1]` 表示第 `i` 天持有股票的最大利润。
我们可以使用递归来计算每一天的最大利润：
- 如果我们不持有股票，我们可以选择在第 `i` 天卖出股票，或者不进行任何操作。
- 如果我们持有股票，我们可以选择在第 `i` 天买入股票，或者不进行任何操作。

然后分别计算这两种状态的最大利润，作为第 `i` 天的持有或不持有的最大利润。因为最后一天无论是否持有股票，我们都希望不持有股票，所以返回 `memo[prices.length - 1][0]` 即可。

- 当第`i`天持有时，它这一天可能的情况是：
    1. 第`i-1`天持有股票，今天不操作，利润不变，状态不变，`memo[i][1] = memo[i-1][1]`
    2. 第`i-1`天不持有股票，今天买入股票，利润减少，状态改变，`memo[i][1] = memo[i-1][0] - prices[i]`
- 当第`i`天不持有时，它这一天可能的情况是：
    1. 第`i-1`天不持有股票，今天不操作，利润不变，状态不变，`memo[i][0] = memo[i-1][0]`
    2. 第`i-1`天持有股票，今天卖出股票，利润增加，状态改变，`memo[i][0] = memo[i-1][1] + prices[i]`

- 时间复杂度：O(n)，其中 n 是数组 prices 的长度。我们需要遍历每一天的价格来计算最大利润。
- 空间复杂度：O(n)，其中 n 是数组 prices 的长度。我们使用了一个大小为 n x 2 的数组来存储每一天的最大利润。


```ts
function maxProfit(prices: number[]): number {
    const memo = Array.from({length: prices.length}, () => Array(2).fill(Infinity))
    function dfs(i, hold) {
        if (i < 0) {
            return hold ? -Infinity : 0
        }
        if (memo[i][hold] !== Infinity) {
            return memo[i][hold]
        }
        let res 
        if (hold) {
            res = Math.max(dfs(i - 1, 1), dfs(i - 1, 0) - prices[i])
        } else {
            res = Math.max(dfs(i - 1, 0), dfs(i - 1, 1) + prices[i])
        }
        return memo[i][hold] = res
    }
    return dfs(prices.length - 1, 0)
};
```

### 2.递推
翻译为递推，memo[i][0]表示第i天不持有股票的最大利润，memo[i][1]表示第i天持有股票的最大利润。初始化memo[0][0]为0，表示第一天不持有股票的最大利润为0，memo[0][1]为-Infinity，表示第一天持有股票的最大利润为负无穷，因为我们不能在第一天买入股票。

从第一天开始遍历，每天有两种情况：
1. 如果我们不持有股票，我们可以选择在第i天卖出股票，或者不进行任何操作。我们取这两种情况的最大值，作为第i天不持有股票的最大利润。
2. 如果我们持有股票，我们可以选择在第i天买入股票，或者不进行任何操作。我们取这两种情况的最大值，作为第i天持有股票的最大利润。

- 时间复杂度：O(n)，其中 n 是数组 prices 的长度。我们需要遍历每一天的价格来计算最大利润。
- 空间复杂度：O(n)，其中 n 是数组 prices 的长度。我们使用了一个大小为 n x 2 的数组来存储每一天的最大利润。

```ts
function maxProfit(prices: number[]): number {
    const memo = Array.from({ length: prices.length + 1 }, () => [0, 0])
    memo[0][1] = -Infinity

    for (let i = 0; i < prices.length; i++) {
        memo[i + 1][0] = Math.max(memo[i][0], memo[i][1] + prices[i])
        memo[i + 1][1] = Math.max(memo[i][1], memo[i][0] - prices[i])
    }
    return memo[prices.length][0]
};
```

### 3.空间优化
因为对于一天来说就两个状态，持有或不持有，而这两个状态都可以只靠前一天就得到，无非就是：
- 今天开始持有：
    - 昨天开始就持有，然后昨天没卖
    - 昨天开始不持有，但是从昨天买入
- 今天开始不持有：
    - 昨天开始就不持有，然后没买
    - 昨天开始持有，但是昨天卖出了

也就是滚动更新公式：`[p0, p1] = [Math.max(p0, p1 + price), Math.max(p1, p0 - price)]`

- 时间复杂度：O(n)，其中 n 是数组 prices 的长度。我们需要遍历每一天的价格来计算最大利润。
- 空间复杂度：O(1)，我们只使用了两个变量来存储每一天的最大利润，而不需要额外的数组来存储每一天的状态。

```ts
function maxProfit(prices: number[]): number {
    let p0 = 0
    let p1 = -Infinity

    for (const price of prices) {
        [p0, p1] = [Math.max(p0, p1 + price), Math.max(p1, p0 - price)]
    }
    return p0
};
```

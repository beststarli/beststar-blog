# 72.编辑距离
给你两个单词 `word1` 和 `word2`， *请返回将 `word1` 转换成 `word2` 所使用的最少操作数*。

你可以对一个单词进行如下三种操作：
- 插入一个字符
- 删除一个字符
- 替换一个字符

示例1：
- **输入**：word1 = "horse", word2 = "ros"
- **输出**：3
- **解释**：
```txt
horse -> rorse (将 'h' 替换为 'r')
rorse -> rose (删除 'r')
rose -> ros (删除 'e')
```

示例2：
- **输入**：word1 = "intention", word2 = "execution"
- **输出**：5
- **解释**：
```txt
intention -> inention (删除 't')
inention -> enention (将 'i' 替换为 'e')
enention -> exention (将 'n' 替换为 'x')
exention -> exection (将 'n' 替换为 'c')
exection -> execution (插入 'u')
```

## 解题思路
### 1.递归加记忆化
构造`dfs(i, j)`表示`word1[0...i]`和`word2[0...j]`的编辑距离。对`word1[i]`和`word2[j]`分类讨论：
- 如果`word1[i]`和`word2[j]`相同，那么编辑距离为`word1[0...i-1]`和`word2[0...j-1]`的编辑距离，即`dfs(i, j) = dfs(i - 1, j - 1)`。
- 如果`word1[i]`和`word2[j]`不同，那么编辑距离为以下三种情况的较小值加1：
  - 删除`word1[i]`，即`dfs(i, j) = dfs(i - 1, j) + 1`。
  - 插入`word2[j]`，即`dfs(i, j) = dfs(i, j - 1) + 1`。
  - 替换`word1[i]`为`word2[j]`，即`dfs(i, j) = dfs(i - 1, j - 1) + 1`。

时间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 word1 和 word2 的长度。

空间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 word1 和 word2 的长度。

```ts
function minDistance(word1: string, word2: string): number {
    const length1 = word1.length
    const length2 = word2.length
    const memo = Array.from({ length: length1 }, () => Array(length2).fill(-1))

    function dfs(i, j) {
        if (i < 0) {
            return j + 1    // 插入剩余的j + 1个字符
        }
        if (j < 0) {
            return i + 1    // 插入剩余的i + 1个字符
        }
        if (memo[i][j] !== -1) {
            return memo[i][j]
        }
        if (word1[i] === word2[j]) {
            return memo[i][j] = dfs(i - 1, j - 1)   // 不做操作，同时向下进行
        }
        return memo[i][j] = Math.min(
            dfs(i - 1, j),      // 插入
            dfs(i, j - 1),      // 删除
            dfs(i - 1, j - 1)   // 替换
        ) + 1
    }

    return dfs(length1 - 1, length2 - 1)
};
```

### 2.递推
定义`memo[i][j]`表示`word1[0...i]`和`word2[0...j]`的编辑距离。对`word1[i]`和`word2[j]`分类讨论：
- 如果`word1[i]`和`word2[j]`相同，那么`memo[i][j] = memo[i - 1][j - 1]`。
- 如果`word1[i]`和`word2[j]`不同，那么`memo[i][j] = Math.min(memo[i - 1][j], memo[i][j - 1], memo[i - 1][j - 1]) + 1`。

构造长度为`(m + 1) × (n + 1)`的二维数组，多一行一列用于存储边界条件，即`memo[0][0] = 0`无需操作，预先填入初始值0，然后做预处理，对于`word1`非空而`word2`空的情况，编辑距离为`word1`的长度；对于`word1`空而`word2`非空的情况，编辑距离为`word2`的长度。

递推入口是`memo[length1][length2]`，其中`length1`和`length2`分别是`word1`和`word2`的长度。

时间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 word1 和 word2 的长度。

空间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 word1 和 word2 的长度。

```ts
function minDistance(word1: string, word2: string): number {
    const length1 = word1.length
    const length2 = word2.length
    const memo = Array.from({ length: length1 + 1 }, () => Array(length2 + 1).fill(0))
    
    for (let i = 0; i <= length1; i++) {
        memo[i][0] = i  // 删除i次
    }
    for (let j = 0; j <= length2; j++) {
        memo[0][j] = j  // 插入j次
    }

    for (let i = 1; i <= length1; i++) {
        for (let j = 1; j <= length2; j++) {
            if (word1[i - 1] === word2[j - 1]) {
                memo[i][j] = memo[i - 1][j - 1]
            } else{
                memo[i][j] = Math.min(
                    memo[i - 1][j],     // 删除
                    memo[i][j - 1],     // 插入
                    memo[i - 1][j - 1]  // 替换
                ) + 1
            }
        }
    }

    return memo[length1][length2]
};
```

### 3.空间优化
由于[解法2](/docs/算法/编辑距离#2递推)中`memo[i][j]`的计算只依赖于`memo[i - 1][j]`、`memo[i][j - 1]`和`memo[i - 1][j - 1]`，所以可以用两个长度为`n + 1`的一维数组来维护当前行和上一行的编辑距离，空间复杂度降为$O(n)$。

时间复杂度：$O(mn)$，其中 m 和 n 分别是字符串 word1 和 word2 的长度。

空间复杂度：$O(n)$，其中 n 是字符串 word2 的长度。

```ts
function minDistance(word1: string, word2: string): number {
    const length1 = word1.length
    const length2 = word2.length
    const memo = Array(length2 + 1).fill(0)

    // 初始化，memo[j] = j
    for (let j = 0; j <= length2; j++) {
        memo[j + 1] = j + 1
    }

    for (const char of word1) {
        let pre = memo[0]
        memo[0]++   // 等价于memo[i][0] = i

        for (let j = 0; j < length2; j++) {
            const temp = memo[j + 1]
            if (char === word2[j]) {
                memo[j + 1] = pre
            } else {
                memo[j + 1] = Math.min(
                    memo[j + 1],    // 删除
                    memo[j],        // 插入
                    pre             // 替换
                ) + 1
            }
            pre = temp
        }
    }

    return memo[length2]
};
```

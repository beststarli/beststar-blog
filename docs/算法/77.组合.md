# 77.组合
给定两个整数`n`和`k`，返回范围`[1, n]`中所有可能的`k`个数的组合。

你可以按 **任何顺序** 返回答案。

示例1：
- **输入**：n = 4, k = 2
- **输出**：
```txt
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

示例2：
- **输入**：n = 1, k = 1
- **输出**：[[1]]

## 解题思路
### 1.递归可选项后恢复现场
我们可以对输入范围内的数字进行递归，每次递归都将当前路径添加至结果中。递归结束后恢复现场，回退可选项。这题可以注意到从后往前处理更方便一些。因为如果从前往后处理，最初选择的数字较大时，后续的可选项就会较少，容易出现越界的情况。

时间复杂度：O(k * C(n, k))，其中C(n, k)是组合数，表示从n个元素中选取k个元素的组合数。生成每个组合需要O(k)的时间。

```ts
function combine(n: number, k: number): number[][] {
    const ans = []
    let path = []

    function dfs(index) {
        if (path.length === k) {
            // 用展开语法来复制新数组，否则后续path的修改会影响到ans中的元素
            ans.push([...path])
            return 
        }

        for (let i = index; i >= k - path.length; i--) {
            path.push(i)
            dfs(i - 1)
            path.pop()
        }
    }

    dfs(n)
    return ans
};
```

### 2.递归是否要选后恢复现场
我们也可以对输入范围内的数字进行递归，每次递归都分为两种情况：选当前数字和不选当前数字。每次递归结束后恢复现场，回退可选项。

时间复杂度：O(k * C(n, k))，其中C(n, k)是组合数，表示从n个元素中选取k个元素的组合数。生成每个组合需要O(k)的时间。

```ts
function combine(n: number, k: number): number[][] {
    const ans = []
    let path = []

    function dfs(index) {
        const need = k - path.length

        if (need === 0) {
            ans.push([...path])
            return
        }

        // 不选当前数字
        if (index > need) {
            dfs(index - 1)
        }

        // 选择当前数字
        path.push(index)
        dfs(index - 1)
        path.pop()  // 回溯，恢复现场
    }

    dfs(n)
    return ans
};
```

## 总结
分析回溯问题的时间复杂度，有一个简易公式：**路径长度**×**搜索树的叶子数**。

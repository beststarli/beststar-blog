# 45.跳跃游戏II
给你一个字符串`s`。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。例如，字符串`"ababcc"`能够被分为`["abab", "cc"]`，但类似`["aba", "bcc"]`或`["ab", "ab", "cc"]`的划分是非法的。

注意，划分结果需要满足：将所有划分结果按顺序连接，得到的字符串仍然是`s`。

返回一个表示每个字符串片段的长度的列表。

示例1：
- **输入**：s = "ababcbacadefegdehijhklij"
- **输出**：[9,7,8]
- **解释**：
```txt
划分结果为 "ababcbaca"、"defegde"、"hijhklij" 。
每个字母最多出现在一个片段中。
像 "ababcbacadefegde", "hijhklij" 这样的划分是错误的，因为划分的片段数较少。 
```

示例2：
- **输入**：s = "eccbbbbdec"
- **输出**：[10]

## 解题思路
### 1.区间合并
实时上是在做字母首尾下标区间的合并。

对字符串进行遍历，记录每个字母最后一次出现的下标位置。用`start`和`end`分别记录当前片段的起始和结束位置。再次遍历`s`，由于当前区间必须包含所有`s[i]`，所以用`last[s[i]]`更新区间右端点`end`的最大值。如果发现`end=i`，那么当前区间合并完毕，把区间长度`end − start + 1`加入答案。然后更新`start = end + 1`作为下一个区间的左端点。遍历完毕，返回答案。

时间复杂度：O(n)，其中 n 是字符串 s 的长度。

```ts
function partitionLabels(s: string): number[] {
    const length = s.length
    const last = Array(26)
    for (let i = 0; i < length; i++) {
        last[s.charCodeAt(i) - 'a'.charCodeAt(0)] = i   // 不断更新每个字母最后出现的下标
    }

    const ans = []
    let start = 0
    let end = 0
    for (let i = 0; i < length; i++) {
        end = Math.max(end, last[s.charCodeAt(i) - 'a'.charCodeAt(0)])  // 更新当前区间右端
        // 当前区间合并完毕
        if (end === i) {
            ans.push(end - start + 1)   // 区间长度加入答案
            start = end + 1     // 下一个区间的左端点
        }
    }
    return ans
};
```

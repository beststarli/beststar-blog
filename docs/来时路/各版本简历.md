  \item \textbf{API拦截与防腐：}基于Axios实现统一的JWT令牌注入和错误机制处理；设计API防腐层，避免视图数据结构与接口数据结构耦合，显著降低前后端联调成本。
  \item \textbf{可复用逻辑和组件封装：}封装定制化的水利蓝风格通用基础组件，图表组件和地图组件，组件复用率80\%以上；基于Vue3组合式API实现业务逻辑拆分和复用，提升迭代更新效率。
\item \textbf{页面模块开发：}负责实时监测预警模块、崩岸综合风险研判模块以及模型库和知识库模块的开发。
\end{itemize}
\content{\textbf{难点与优化：}}
\begin{itemize}
  \item 基于 SSE 方案替代短轮询实现现场设备数据实时接入，通过发布订阅机制为消费组件分发 数据，并基于 Echarts 实现 10w 级数据的高效可视化。  
  \item 基于 Promise 封装模型并发执行器，支撑同岸段多断面风险分析模型的并发调用和错误重 试；基于策略模式实现不同条件下灵活组合的模型自动化链式调用。  
  \item 通过第三方库按需导入、精灵图技术结合 Vite 插件进行资源压缩，减少打包体积 35\%。通 过路由、组件懒加载技术和图片懒加载组件降低首屏时间。  
\end{itemize}
\content{\textbf{项目成果：}项目在上线期间成功预警过两次崩岸险情，成功支撑发表软件著作权3项，学术论文1篇。}

  \content{\textbf{主要工作：}}
  \begin{itemize}                                                                                                       
    \item \textbf{前端架构设计与开发：}独立完成系统前端业务模块开发，基于 Template-Registry 模式设计资源管理框架（Schema/Patch/Grid/Vector模板），通过 register 插件机制动态注册视图组件、图标、资源类型与上下文菜单，实现高内聚低耦合的模块化架构。                                  
    \item \textbf{类型安全的API抽象层：}后端基于FastAPI按业务域拆分为 11 个路由模块，Pydantic                           
  模型校验；前端封装泛型API调用层，支持 JSON/二进制数据传输，通过动态路由机制实现本地开发与远程部署环境切换。           
    \item \textbf{前端路由与状态管理架构：}基于 React Router 实现 SPA
  路由体系，拆分登录鉴权、编辑视图、全局设置等视图层路由守卫；采用 Zustand 分层管理全局状态，结合 persist
  中间件实现跨会话配置持久化。
    \item \textbf{基于 Actor 模型的并发计算框架：}自研 Actor-Dispatcher 消息传递架构，配合 Web Worker 线程池与 Scheduler
   优先级调度器，将网格细分/合并等高密度计算任务卸载至后台线程，通过 Transferable 零拷贝传输避免主线程阻塞。

  式实现跨组件数据联动；通过父子关系映射维护完整场景资源树结构，配合懒加载与按需展开策略优化大规模树结构渲染性能。
